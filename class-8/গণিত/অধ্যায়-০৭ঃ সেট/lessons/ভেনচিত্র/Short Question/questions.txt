Question:
[Image]ভেনচিত্র হতে A ∩ B ∩ C নির্ণয় কর।

Options:


Correct Answer: 

Solution:
[math]\begin{aligned}&amp; \text { ভেনচিত্র হতে পাই, } A=\{1,2,4,5\}, \\&amp; B=\{2,3,4,6\} \\&amp; \text { এবং } C=\{4,5,6,7\} \\&amp; \therefore A \cap B=\{1,2,4,5\} \cap\{2,3,4,6\}=\{2,4\} \\&amp; \therefore A \cap B \cap C=\{2,4\} \cap\{4,5,6,7\}=\{4\}\end{aligned}[/math]নির্ণেয় সেট [math]\mathrm{A} \cap \mathrm{B} \cap \mathrm{C}=\{4\}[/math]

Question:
ভেনচিত্র থেকে (A ∩ B)' নির্ণয় কর।

[Image]

Options:


Correct Answer: 

Solution:
ভেনচিত্র থেকে পাই,[math]\begin{aligned}&amp; \mathrm U=\{1,2,3,4\}, \\&amp; \mathrm A=\{1,2\} \text { এবং } \\&amp; \mathrm B=\{2,3\} \\&amp; \therefore \mathrm A \cap \mathrm B=\{1,2\} \cap\{2,3\}=\{2\} \\&amp; \therefore(\mathrm A \cap \mathrm B)^{\prime}=\mathrm U-(\mathrm A \cap \mathrm B)\\&amp;=\{1,2,3,4\}-\{2\}=\{1,3,4\}\end{aligned}[/math]নির্ণেয় সেট [math](\mathrm{A} \cap \mathrm{B})^{\prime}=\{1,3,4\}[/math]
