Options:


Correct Answer: 

Solution:

Question:
বার্ষিক শতকরা ১০ টাকা মুনাফায় ৫০০০ টাকার ৩ বছরের চক্রবৃদ্ধি মূলধন নির্ণয় কর।

Options:


Correct Answer: 

Solution:
এখানে,প্রারম্ভিক মূলধন, P = ৫০০০ টাকামুনাফার হার, r = ১০% =[math]\frac{\text{১০}}{\text{১০০}}[/math]সময়, n = ৩ বছরআমরা জানি,চক্রবৃদ্ধি মূলধন, C = P(১ + r)nবা, C = ৫০০০[math]\left(\text১+\frac{\text ১\text০}{\text ১০০}\right)[/math]টাকা    = ৫০০০ (১ + ০.১)৩ টাকা    = ৫০০০ (১.১)৩ = (৫০০০ &times;১.৩৩১) টাকা = ৬৬৫৫ টাকানির্ণেয় চক্রবৃদ্ধি মূলধন ৬৬৫৫ টাকা।
