Options:


Correct Answer: 

Solution:

Question:
কোনো বৃত্তের পরিধি [math]\pi[/math] হলে, এর ক্ষেত্রফল কত?

Options:


Correct Answer: 

Solution:
ধরি, বৃত্তের ব্যাসার্ধ [math] = r[/math] একক[math]\therefore \quad[/math] বৃত্তের পরিধি [math]=2 \pi r[/math] এককপ্রশ্নমতে, [math]2 \pi r=\pi[/math]বা, [math]r=\frac{\pi}{2 \pi}=\frac{1}{2}[/math][math]\therefore[/math] বৃত্তের ক্ষেত্রফল [math]=\pi r^2=\pi\left(\frac{1}{2}\right)^2[/math] বর্গ একক [math]=\frac{\pi}{4}[/math] বর্গ এককনির্ণেয় বৃত্তের ক্ষেত্রফল [math]\frac{\pi}{4}[/math] বর্গ একক।

Options:


Correct Answer: 

Solution:
মনে করি, বৃত্তাকার বাগানের ব্যাসার্ধ [math]= r[/math] ফুট[math]\therefore \quad \text { ব্যাস }=2 \mathrm{r} \text { ফুট }[/math]প্রশ্নমতে, [math]2 \mathrm{r}=30[/math][math]\therefore \quad r=\frac{30}{2}=15 \text { ফুট }[/math]রাস্তাসহ বাগানের ব্যাসার্ধ, [math]R=(15+5)[/math] ফুট = 20 ফুট[math]\therefore[/math] রাস্তার ক্ষেত্রফল[math]\begin{aligned}\text { } &amp; =\pi \mathrm{R}^2-\pi \mathrm{r}^2 \\&amp; =\left(\mathrm{R}^2-\mathrm{r}^2\right) \pi=\left(20^2-15^2\right) \pi \text { বর্গফুট } \\&amp; =(400-22.4) \pi \text { বর্গফুট }=175 \pi \text { বর্গফুট }\end{aligned}[/math]নির্ণেয় বাগানের ক্ষেত্রফল [math]175 \pi[/math] বর্গফুট।

Question:
1962.5 বর্গ সে.মি. ক্ষেত্রফলবিশিষ্ট একটি বৃত্তের ব্যাস কত সে.মি.?

Options:


Correct Answer: 

Solution:
ধরি, বৃত্তের ব্যাসার্ধ [math] = r[/math] সে.মি.[math]\therefore[/math] বৃত্তের ক্ষেত্রফল [math]=\pi r^2[/math] বর্গ সে.মি.প্রশ্নমতে, [math]\pi r^2=1962.5[/math]বা, [math]r^2=\frac{1962.5}{\pi}=\frac{1962.5}{3.14}=625[math]\therefore r=\sqrt{625}=25[/math]\therefore[/math] বৃত্তের ব্যাস [math]=2 r=2 \times 25=50[/math] সে.মি.নির্ণেয় বৃত্তের ব্যাস [math] 50[/math] সে.মি.।
