Question:
পরস্পর [math]\rm{d}[/math] দূরত্বে থাকা সমান্তরাল দুটি দীর্ঘ তারের মধ্য দিয়ে সমমানের বিপরীতমুখী প্রবাহ ( [math]1[/math]) প্রবাহিত হলে, এদের ঠিক মধ্যবর্তী কোন বিন্দুতে চৌম্বক আবেশের মান কত হবে?

Options:
ক. শূন্য
খ. [math]\dfrac{\mu_{0} \mathrm{l}}{\mathrm{ml}}[/math]
গ. [math]\dfrac{2\mu_{0} \mathrm{l}}{\mathrm{ml}}[/math]
ঘ. [math]\dfrac{2 \pi \rm{I}}{\mu_0 \rm{d}}[/math]


Correct Answer: খ

Solution:
সমান্তরাল তারের জন্য, একটি তারের জন্য চৌম্বক ক্ষেত্র হল [math]B = \frac{\mu_0 I}{2 \pi r}[/math]
, যেখানে [math]r[/math]
 হল উৎস থেকে দূরত্ব। মধ্যবর্তী বিন্দুতে, প্রতিটি তার থেকে দূরত্ব হল [math]\frac{d}{2}[/math]
। যেহেতু প্রবাহ বিপরীতমুখী, তাই চৌম্বক ক্ষেত্র দুটি একই দিকে যোগ হবে। সুতরাং, মোট চৌম্বক আবেশ হল [math]B_{total} = B_1 + B_2 = \frac{\mu_0 I}{2 \pi (d/2)} + \frac{\mu_0 I}{2 \pi (d/2)} = \frac{\mu_0 I}{\pi d} + \frac{\mu_0 I}{\pi d} = \frac{2 \mu_0 I}{\pi d}[/math]
। এখানে প্রশ্নে প্রবাহের মান 'I' কে 'I' ধরা হয়েছে এবং তারের মধ্যবর্তী দূরত্ব 'd' কে 'd' ধরা হয়েছে।এই সমস্যাটি অ্যাম্পিয়ারের সূত্র এবং চৌম্বকত্বের নীতি ব্যবহার করে সমাধান করা হয়। অ্যাম্পিয়ারের সূত্র একটি বদ্ধ লুপ বরাবর চৌম্বক ক্ষেত্র এবং সেই লুপ দ্বারা আবদ্ধ তড়িৎপ্রবাহের মধ্যে সম্পর্ক স্থাপন করে। দুটি তারের মধ্যবর্তী বিন্দুতে চৌম্বক ক্ষেত্র প্রতিটি তারের দ্বারা সৃষ্ট চৌম্বক ক্ষেত্রের ভেক্টর যোগফল দ্বারা নির্ধারিত হয়। বিপরীতমুখী প্রবাহের কারণে, প্রতিটি তারের দ্বারা সৃষ্ট চৌম্বক ক্ষেত্র একই দিকে কাজ করে, তাই তাদের মান যোগ হয়।

Question:
নিচের উদ্দীপকটি পড়ো এবং নং প্রশ্নের উত্তর দাও।

[math]1.5 \ m[/math] লম্বা দুটি সমান্তরাল তারের মধ্যবর্তী দূরত্ব [math]0.03 m[/math] এবং প্রতিটি তারে [math]120 \ amp[/math] প্রবাহমাত্রা চলছে।

যেকোনো একটি তারের [math]1 \ m[/math] দৈর্ঘ্যের ওপর ক্রিয়ারত বলের মান প্ৰায় কত?

Options:
ক. [math]3.2 \times 10^{-2} N[/math]
খ. [math]6.4 \times 10^{-2} N[/math]
গ. [math]\text { 7. } 2 \times 10^{-2} N[/math]
ঘ. [math]9.6 \times 10^{-2} \mathrm{~N}[/math]


Correct Answer: ঘ

Solution:
দুটি সমান্তরাল পরিবাহীর প্রতি একক দৈর্ঘ্যের উপর প্রযুক্ত বলের সূত্র হলো [math]F/L = (\mu_0 I_1 I_2) / (2\pi d)[/math]
. এখানে, [math]\mu_0[/math]
 হলো শূন্যস্থানের প্রবেশ্যতা যার মান [math]4\pi \times 10^{-7} \ Tm/A[/math]
. প্রশ্নানুসারে, [math]I_1 = I_2 = 120 \ A[/math]
 এবং [math]d = 0.03 \ m[/math]
. যেকোনো একটি তারের [math]1 \ m[/math]
 দৈর্ঘ্যের ওপর ক্রিয়ারত বলের মান বের করতে হবে, অর্থাৎ [math]F/L[/math]
 এর মান নির্ণয় করতে হবে।গণনা:[math]F/L = (4\pi \times 10^{-7} \ Tm/A \times 120 \ A \times 120 \ A) / (2\pi \times 0.03 \ m)[/math]
 [math]F/L = (2 \times 10^{-7} \times 14400) / 0.03 \ N/m[/math]
 [math]F/L = 28800 \times 10^{-7} / 0.03 \ N/m[/math]
 [math]F/L = 0.00288 / 0.03 \ N/m[/math]
 [math]F/L = 0.096 \ N/m[/math]
সুতরাং, [math]1 \ m[/math]
 দৈর্ঘ্যের ওপর ক্রিয়ারত বলের মান প্রায় [math]0.096 \ N[/math]
 বা [math]9.6 \times 10^{-2} \ N[/math]
.এই সূত্রটি ব্যবহার করা হয় যখন দুটি সমান্তরাল পরিবাহীর মধ্যে চৌম্বকীয় বল গণনা করা হয়। পরিবাহীগুলির মধ্যে প্রবাহের দিক একই হলে আকর্ষণ বল এবং বিপরীত দিকে হলে বিকর্ষণ বল প্রযুক্ত হয়।

Question:
নিচের উদ্দীপকটি পড়ো এবং প্রশ্নের উত্তর দাও।

[math]1.5 \ m[/math] লম্বা দুটি সমান্তরাল তারের মধ্যবর্তী দূরত্ব [math]0.03.m[/math] এবং প্রতিটি তারে [math]120 \ amp[/math] প্রবাহমাত্রা চলছে।

তারগুলো মুক্তভাবে চলতে সক্ষম হলে, এবং প্রতিটি তারের ভর [math]150 \ gm[/math] হলে প্রতিটি তারের আদি ত্বরণ কত হবে?

Options:
ক. [math]0.32 \ m.s^{-2}[/math]
খ. [math]0.64 \ m.s^{-2}[/math]
গ. [math]0.72 \ m.s^{-2}[/math]
ঘ. [math]0.96 \ m.s^{-2}[/math]


Correct Answer: ঘ

Solution:
দুটি সমান্তরাল তারের মধ্যে আকর্ষণ বা বিকর্ষণ বলের সূত্র হল [math]F/L = (\mu_0 I^2)/(2\pi d)[/math]
, যেখানে [math]F[/math]
 হল বল, [math]L[/math]
 হল তারের দৈর্ঘ্য, [math]\mu_0[/math]
 হল শূন্যস্থানের প্রবেশ্যতা ( [math]4\pi \times 10^{-7} \ T.m/A[/math]
), [math]I[/math]
 হল প্রবাহমাত্রা এবং [math]d[/math]
 হল তার দুটির মধ্যবর্তী দূরত্ব।এখানে দেওয়া আছে:[math]L = 1.5 \ m[/math]
 [math]d = 0.03 \ m[/math]
 [math]I = 120 \ amp[/math]
তারের ভর [math]m = 150 \ gm = 0.150 \ kg[/math]
প্রথমে, প্রতি একক দৈর্ঘ্যে বল বের করা যাক:[math]F/L = (4\pi \times 10^{-7} \ T.m/A \times (120 \ amp)^2) / (2\pi \times 0.03 \ m)[/math]
 [math]F/L = (2 \times 10^{-7} \times 14400) / 0.03[/math]
 [math]F/L = (28800 \times 10^{-7}) / 0.03[/math]
 [math]F/L = 0.00288 / 0.03[/math]
 [math]F/L = 0.096 \ N/m[/math]
এবার, তারের উপর মোট বল হবে [math]F = (F/L) \times L[/math]
 [math]F = 0.096 \ N/m \times 1.5 \ m[/math]
 [math]F = 0.144 \ N[/math]
নিউটনের দ্বিতীয় গতিসূত্র অনুযায়ী, ত্বরণ ( [math]a[/math]
) = বল ( [math]F[/math]
) / ভর ( [math]m[/math]
)[math]a = 0.144 \ N / 0.150 \ kg[/math]
 [math]a = 0.96 \ m/s^2[/math]
সুতরাং, প্রতিটি তারের আদি ত্বরণ হবে [math]0.96 \ m/s^2[/math]
. তার দুটি একই দিকে প্রবাহিত হওয়ায় একে অপরকে আকর্ষণ করবে এবং এই আকর্ষণের ফলেই ত্বরণ সৃষ্টি হবে।
