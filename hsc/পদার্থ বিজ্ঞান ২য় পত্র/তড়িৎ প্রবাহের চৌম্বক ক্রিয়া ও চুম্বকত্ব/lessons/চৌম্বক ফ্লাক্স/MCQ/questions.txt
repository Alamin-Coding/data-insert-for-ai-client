খ. [math]\text { Wbm}^{-2}[/math]
গ. [math]\text { H}[/math]
ঘ. [math]\text { WbA}^{-1}[/math]


Correct Answer: খ

Solution:
চৌম্বক ফ্লাক্স ঘনত্বের একক হল টেসলা (Tesla), যা [math]\text { Wb m}^{-2}[/math]
 দ্বারাও প্রকাশ করা হয়। চৌম্বক ফ্লাক্স ( [math]\Phi_B[/math]
 ) হলো একটি তলের মধ্য দিয়ে অতিক্রান্ত চৌম্বক বলরেখার মোট সংখ্যা, যার একক ওয়েবার (Wb)। চৌম্বক ফ্লাক্স ঘনত্ব ( [math]B[/math]
) হলো প্রতি একক ক্ষেত্রফলে চৌম্বক ফ্লাক্সের পরিমাপ, অর্থাৎ [math]B = \frac{\Phi_B}{A}[/math]
, যেখানে [math]A[/math]
 হলো ক্ষেত্রফল। তাই এর একক হয় [math]\frac{\text{Wb}}{\text{m}^2}[/math]
 বা [math]\text { Wb m}^{-2}[/math]
। বিকল্প A (Wb) চৌম্বক ফ্লাক্সের একক। বিকল্প C (H) হেনরি (Henry) হলো ইন্ডাকট্যান্সের (inductance) একক। বিকল্প D (WbA [math]^{-1}[/math]
) হলো হেনরি (H) এর সমতুল্য, যা ইন্ডাকট্যান্সের একক।

Question:
চৌম্বক প্রবেশ্যতার একক কোনটি?

Options:
ক. Wb/a
খ. Wb/m
গ. H/m
ঘ. Hm/a


Correct Answer: গ

Solution:
চৌম্বক প্রবেশ্যতার একক হলো হেনরি প্রতি মিটার (H/m)। এটি একটি পদার্থের মধ্য দিয়ে চৌম্বক ক্ষেত্র কতটা সহজে প্রবেশ করতে পারে তা পরিমাপ করে। চৌম্বক প্রবেশ্যতা (permeability) চৌম্বক ফ্লাক্স ঘনত্ব (magnetic flux density) এবং চৌম্বক ক্ষেত্র তীব্রতার (magnetic field strength) অনুপাত হিসেবেও সংজ্ঞায়িত করা যেতে পারে। অন্যান্য বিকল্পগুলো চৌম্বক প্রবেশ্যতার সঠিক একক নয়। Wb/a চৌম্বক ফ্লাক্সের পরিবর্তন হার নির্দেশ করে, Wb/m চৌম্বক ফ্লাক্স ঘনত্বের সাথে সম্পর্কিত হতে পারে কিন্তু এটি একক নয়, এবং Hm/a সঠিক একক নয়।
