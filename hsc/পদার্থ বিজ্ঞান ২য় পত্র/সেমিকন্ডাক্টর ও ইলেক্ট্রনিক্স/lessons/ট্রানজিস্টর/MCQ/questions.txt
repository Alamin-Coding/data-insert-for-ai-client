Question:
An npn transistor in CE configuration has [math]a = 0.995[/math]. What is the base current if the collector current is 4 mA?

Options:
A. [math]18 \mu \mathrm{A}[/math]
B. [math]23 \mu \mathrm{A}[/math]
C. [math]28 \mu \mathrm{A}[/math]
D. [math]20 \mu \mathrm{A}[/math]


Correct Answer: D.[math]20 \mu \mathrm{A}[/math]

Solution:
[math]\alpha=\frac{\mathrm{I}_{\mathrm{C}}}{\mathrm{I}_{\mathrm{E}}} \Rightarrow \mathrm{I}_{\mathrm{E}}=\frac{\mathrm{I}_{\mathrm{C}}}{\alpha}=\frac{4}{0.995} \mathrm{~mA}=4.0201 \mathrm{~mA}[/math]
 [math]\therefore I_{B}=I_{E}-I_{C}=0.02 \mathrm{~mA}=20 \mu \mathrm{A}[/math]

Question:
ট্রানজিস্টরের ক্ষেত্রে [math]\alpha[/math] ও [math]\boldsymbol{\beta}[/math] এর মধ্যে সম্পর্ক নিম্নরূপ—

Options:
ক. [math]\beta=\frac{\alpha}{1+\alpha}[/math]
খ. [math]\beta=\frac{1+\alpha}{1-\alpha}[/math]
গ. [math]\alpha=\frac{\beta}{1+\beta}[/math]
ঘ. [math]\alpha=\frac{1+\beta}{\beta}[/math]


Correct Answer: C.[math]\alpha=\frac{\beta}{1+\beta}[/math]

Solution:
ট্রানজিস্টরের ক্ষেত্রে আমরা জানি,নিঃসারক প্রবাহ = পীঠ প্রবাহ + সংগ্রাহক প্রবাহঅর্থাৎ, [math]I_{E}=I_{B}+I_{C}[/math]
 [math]\begin{array}{l}\Rightarrow\text { } \Delta \mathrm{l}_{E}=\Delta\mathrm{I}_{B}+\Delta \mathrm{l}_{C} \\\Rightarrow\text { } \Delta \mathrm{I}_{B}=\Delta \mathrm{I}_{E}-\Delta \mathrm{l}_{C}\end{array}[/math]
এখন, [math]\beta=\frac{\Delta \mathrm{I}_{\mathrm{C}}}{\Delta\mathrm{I}_{\mathrm{B}}}=\frac{\Delta \mathrm{I}_{\mathrm{C}}}{\Delta\mathrm{I}_{\mathrm{E}}-\Delta \mathrm{I}_{\mathrm{C}}}[/math]
 [math]\begin{array}{l}\beta=\frac{\frac{\Delta \mathrm{l}_{\mathrm{C}}}{\Delta\mathrm{l}_{\mathrm{E}}}}{1-\frac{\Delta \mathrm{l}_{\mathrm{C}}}{\Delta\mathrm{I}_{\mathrm{E}}}} \\\therefore \beta=\frac{\alpha}{1-\alpha}\end{array}[/math]
 [math]\Rightarrow \beta(1 - \alpha) = \alpha\\\Rightarrow \beta - \beta \alpha = \alpha \\\Rightarrow \beta = \alpha + \beta \alpha = \alpha (1 + \beta) \\\therefore \alpha = \frac{\beta}{1 + \beta}[/math]

Correct Answer: A.9

Solution:

 [math]=\frac{0.9}{1-0.9}=\frac{0.9}{0.1}=9[/math]

Question:
একটি ট্রানজিস্টর সাধারণ পীঠ সংযোগে আছে। যদি নিঃসারক প্রবাহ [math]0.85 \mathrm{~mA}[/math] এবং পীঠ প্রবাহ [math]50 \mu \mathrm{~A}[/math] হয়, তাহলে প্রবাহ বিবর্ধন গুণক কত?

Options:
ক. 0.06
খ. 0.94
গ. 1.06
ঘ. 16


Correct Answer: B.0.94

Solution:
• সংগ্রাহক প্রবাহ [math]I_{C}=I_{E}-I_{B}=0.85[/math]
 [math]\mathrm{mA}-50 \mu \mathrm{~A}=0.85 \times[/math]
 [math]10^{-3}-50 \times 10^{-6}[/math]
 [math]=8 \times 10^{-4} \mathrm{~A}=0.80 \mathrm{~mA}[/math]
• প্রবাহ বিবর্ধন গুণক, [math]\alpha=\frac{\mathrm{I}_{\mathrm{C}}}{\mathrm{I}_{\mathrm{E}}}=\frac{0.80}{0.85}=0.94[/math]

Question:
বর্তনীটির [math]\alpha[/math] এর মান কত?

Options:
ক. 0.67
খ. 0.80
গ. 1:5
ঘ. 2.0
ঙ. সঠিক উত্তর: 0.9877


Correct Answer: E. সঠিক উত্তর: 0.9877

Solution:
[math]\alpha[/math]
 হল সাধারণ-বেস কারেন্ট গেইন, যা কালেক্টর কারেন্ট ( [math]I_C[/math]
) এবং এমিটার কারেন্টের ( [math]I_E[/math]
) অনুপাত। অর্থাৎ, [math]\alpha = \frac{I_C}{I_E}[/math]
। প্রশ্নটিতে [math]I_C[/math]
 এবং [math]I_E[/math]
 এর মান দেওয়া থাকলে, তাদের ভাগ করে [math]\alpha[/math]
 এর মান নির্ণয় করা যায়। যদি [math]I_C = 0.8[/math]
 এবং [math]I_E = 1[/math]
 হয়, তাহলে [math]\alpha = \frac{0.8}{1} = 0.8[/math]
 হবে।

Explanation:
[math]\alpha=\frac{\beta}{1+\beta}=\frac{80}{1+80}=0.9877[/math]
