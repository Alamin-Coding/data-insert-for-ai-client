Question:
এক ব্যাপ্তি 40s সময়ে একটি কাঠের টুকরা 3m উচ্চতায় তুলতে পারেন। আর একজন 20 s সময়ে একই উচ্চতায় তুলতে পারেন।

তাদের ব্যায়িত শক্তি–

Options:
ক. সমান
খ. দ্বিতীয় জনের দ্বিগুণ বেশি
গ. দ্বিতীয় জনের তিন গুণ বেশি
ঘ. প্রথম জনের বেশি


Correct Answer: ক

Solution:
এই প্রশ্নের সঠিক উত্তর হল "প্রথম জনের বেশি"।**ব্যাখ্যা:**ক্ষমতা (Power) হলো কাজ করার হার। ক্ষমতা নির্ণয়ের সূত্র হলো:ক্ষমতা ( [math]P[/math]
) = কাজ ( [math]W[/math]
) / সময় ( [math]t[/math]
)এখানে, কাজটি হলো একটি বস্তুকে একটি নির্দিষ্ট উচ্চতায় তোলা। কাজ ( [math]W[/math]
) = বল ( [math]F[/math]
) × দূরত্ব ( [math]d[/math]
)। যেহেতু অভিকর্ষজ বলের বিরুদ্ধে বস্তুকে তোলা হচ্ছে, তাই বল ( [math]F[/math]
) বস্তুর ওজনের সমান, অর্থাৎ [math]F = mg[/math]
, যেখানে [math]m[/math]
 হলো বস্তুর ভর এবং [math]g[/math]
 হলো অভিকর্ষজ ত্বরণ। দূরত্ব ( [math]d[/math]
) হলো উচ্চতা ( [math]h[/math]
)।সুতরাং, কাজ ( [math]W[/math]
) = [math]mgh[/math]
প্রথম ব্যক্তির ক্ষেত্রে:সময় ( [math]t_1[/math]
) = 40 sউচ্চতা ( [math]h[/math]
) = 3 mক্ষমতা ( [math]P_1[/math]
) = [math]W / t_1[/math]
 = [math]mgh / 40[/math]
দ্বিতীয় ব্যক্তির ক্ষেত্রে:সময় ( [math]t_2[/math]
) = 20 sউচ্চতা ( [math]h[/math]
) = 3 mক্ষমতা ( [math]P_2[/math]
) = [math]W / t_2[/math]
 = [math]mgh / 20[/math]
এখন আমরা [math]P_1[/math]
 এবং [math]P_2[/math]
 এর তুলনা করব:[math]P_1 = mgh / 40[/math]
 [math]P_2 = mgh / 20[/math]
যেহেতু [math]m[/math]
, [math]g[/math]
, এবং [math]h[/math]
 একই, আমরা দেখতে পারি যে [math]1/40[/math]
 ভগ্নাংশটি [math]1/20[/math]
 ভগ্নাংশটির চেয়ে ছোট।অর্থাৎ, [math]P_1 < P_2[/math]
অতএব, দ্বিতীয় ব্যক্তির ক্ষমতা বেশি। প্রশ্নানুসারে, "ব্যায়িত শক্তি" বলতে ক্ষমতা বোঝানো হয়েছে। যেহেতু দ্বিতীয় ব্যক্তি কম সময়ে একই কাজ করছে, তার ক্ষমতা বেশি। সুতরাং, প্রথম ব্যক্তির ক্ষমতা দ্বিতীয় ব্যক্তির চেয়ে কম।**অতিরিক্ত তথ্য:**ক্ষমতা হলো প্রতি সেকেন্ডে কাজ করার সামর্থ্য। যার ক্ষমতা বেশি, সে কম সময়ে একই পরিমাণ কাজ করতে পারে অথবা বেশি সময়ে বেশি কাজ করতে পারে। এই ক্ষেত্রে, যেহেতু কাজটি একই (3m উচ্চতায় তোলা) এবং সময় ভিন্ন, তাই ক্ষমতা ভিন্ন হবে।

Question:
সুষম ত্রিভুজাকৃতি পাতের অভিকর্ষ কেন্দ্ৰ কোথায়?

Options:
ক. কর্ণদ্বয়ের ছেদ বিন্দুতে
খ. জ্যামিতিক কেন্দ্রে
গ. বাহুগুলোর মধ্যবিন্দুতে
ঘ. মধ্যমাগুলোর ছেদ বিন্দুতে


Correct Answer: ঘ

Solution:
একটি সুষম ত্রিভুজাকৃতি পাতের অভিকর্ষ কেন্দ্র হলো এর মধ্যমাগুলোর ছেদ বিন্দুতে। ত্রিভুজের মধ্যমা হলো শীর্ষবিন্দু থেকে বিপরীত বাহুর মধ্যবিন্দু পর্যন্ত অঙ্কিত রেখা। তিনটি মধ্যমাই একটি বিন্দুতে মিলিত হয়, যা ত্রিভুজটির ভারকেন্দ্র বা অভিকর্ষ কেন্দ্র নামে পরিচিত। অপশন A ভুল কারণ এটি বর্গের জন্য প্রযোজ্য, অপশন B ভুল কারণ জ্যামিতিক কেন্দ্র অভিকর্ষ কেন্দ্র নাও হতে পারে যদি পাতের ঘনত্ব সুষম না হয়, এবং অপশন C ভুল কারণ বাহুগুলোর মধ্যবিন্দুতে অভিকর্ষ কেন্দ্র অবস্থিত নয়।
