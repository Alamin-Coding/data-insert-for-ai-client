Question:
উদ্দীপকের আলোকে প্রশ্নের উত্তর দাও :


[Image]


P ও Q এর মধ্যবর্তী দূরত্ব 75 cm হলে 5 সেকেন্ডে স্থির তরঙ্গ সৃষ্টিকারী তরঙ্গের অতিক্রান্ত দূরত্ব–

Options:
ক. 120m
খ. 375m
গ. 600m
ঘ. 750m


Correct Answer: গ

Solution:
এখানে স্থির তরঙ্গ (standing wave) তৈরি হয়েছে। স্থির তরঙ্গের দুটি পরপর নিস্পন্দ বিন্দুর (node) মধ্যবর্তী দূরত্ব হল [math]\lambda/2[/math]

 Hz।সাধারণত, যখন "স্থির তরঙ্গ সৃষ্টিকারী তরঙ্গের অতিক্রান্ত দূরত্ব" বলা হয়, তখন এটি বোঝায় যে তরঙ্গটি 5 সেকেন্ডে কত দূরত্ব ভ্রমণ করেছে। কিন্তু স্থির তরঙ্গের প্রকৃতি এমন যে এর প্যাটার্ন স্থির থাকে।এই প্রশ্নটি সম্ভবত এমনভাবে তৈরি করা হয়েছে যেখানে 75 cm দূরত্বটি তরঙ্গের একটি অংশকে নির্দেশ করে যা 5 সেকেন্ডে একটি নির্দিষ্ট দূরত্ব অতিক্রম করেছে, এবং সেখানে পরোক্ষভাবে বেগের ধারণা ব্যবহার করা হয়েছে।যদি আমরা সরাসরি দূরত্বের সাথে দূরত্ব অনুপাত করি: 75 cm (দূরত্ব) এবং 5 s (সময়)।অনেক সময় এই ধরনের প্রশ্নে, তরঙ্গের বিস্তার (amplitude) বা নির্দিষ্ট দূরত্ব অতিক্রম করতে প্রয়োজনীয় সময় (time period) এর সাথে সম্পর্কিত তথ্য থাকে।এখানে একটি সাধারণ ধারণা ব্যবহার করা হয়েছে যেখানে 75 cm দূরত্ব 5 সেকেন্ডে অতিক্রান্ত হয়। যদি আমরা ধরে নিই যে 75 cm দূরত্ব তরঙ্গের একটি অংশ (যেমন একটি লুপের দৈর্ঘ্য বা এর অর্ধেক), তবে তা একটি বেগের সাথে সম্পর্কিত হতে পারে।যদি আমরা ধরে নিই যে 75 cm হল অর্ধ তরঙ্গদৈর্ঘ্য ( [math]\lambda/2[/math]

300 Hz কম্পাঙ্কের একটি শব্দ তরঙ্গের পানি ও বাতাসে তারঙ্গদৈর্ঘ্যের পার্থক্য 4.16m । শব্দের গতি বাতাসে [math]352 \mathrm{ms}^{-1} \beta[/math] পানিতে শব্দের গতি কত?

Options:
ক. [math]800 \mathrm{~ms}^{-1}[/math]
খ. [math]1200 \mathrm{~ms}^{-1}[/math]
গ. [math]1600 \mathrm{~ms}^{-1}[/math]
ঘ. [math]1800 \mathrm{\ cms}^{-1}[/math]


Correct Answer: C.[math]1600 \mathrm{~ms}^{-1}[/math]

Solution:
শব্দ তরঙ্গের তরঙ্গদৈর্ঘ্য ( [math]\lambda[/math]
), কম্পাঙ্ক ( [math]f[/math]
) এবং শব্দের গতি ( [math]v[/math]
) এর মধ্যে সম্পর্ক হলো [math]v = f \lambda[/math]
 ।সুতরাং, [math]\lambda = v/f[/math]
 ।বাতাসে শব্দের তরঙ্গদৈর্ঘ্য, [math]\lambda_{air} = 352 \mathrm{~ms}^{-1} / 300 \mathrm{~Hz} = 1.173 \mathrm{~m}[/math]
 ।প্রশ্নে দেওয়া আছে, পানি ও বাতাসে তারঙ্গদৈর্ঘ্যের পার্থক্য 4.16m।সুতরাং, পানিতে শব্দের তরঙ্গদৈর্ঘ্য ( [math]\lambda_{water}[/math]
) হবে:[math]\lambda_{water} = \lambda_{air} + 4.16 \mathrm{~m}[/math]
 [math]\lambda_{water} = 1.173 \mathrm{~m} + 4.16 \mathrm{~m} = 5.333 \mathrm{~m}[/math]
পানিতে শব্দের গতি ( [math]v_{water}[/math]
) হবে:[math]v_{water} = f \times \lambda_{water}[/math]
 [math]v_{water} = 300 \mathrm{~Hz} \times 5.333 \mathrm{~m} = 1600 \mathrm{~ms}^{-1}[/math]
শব্দের গতি মাধ্যমের ঘনত্বের উপর নির্ভর করে। পানি বাতাসের চেয়ে বেশি ঘন, তাই পানিতে শব্দের গতিও বেশি হয়।

Question:
একটি শব্দ ভরা বাবুতে 3 মিনিটে 1080 m দূরত্ব অতিক্রম করে। এ শব্দ তরঙ্গের তরঙ্গদৈর্ঘ্য 60 cm হলে তরঙ্গের পর্যায়কাল কত?

Options:
ক. 10 s
খ. 1 s
গ. 0.1 s
ঘ. এর কোনোটিই নয়


Correct Answer: গ

Solution:
শব্দটি 3 মিনিটে (অর্থাৎ 180 সেকেন্ডে) 1080 মিটার দূরত্ব অতিক্রম করে। সুতরাং, শব্দের বেগ ( [math]v[/math]
) হবে দূরত্ব/সময় [math]= 1080 m / 180 s = 6 m/s[/math]
.তরঙ্গদৈর্ঘ্য ( [math]\lambda[/math]
) দেওয়া আছে 60 cm, যা মিটারে 0.6 m.আমরা জানি, বেগ ( [math]v[/math]
) = কম্পাঙ্ক ( [math]f[/math]
) × তরঙ্গদৈর্ঘ্য ( [math]\lambda[/math]
).আবার, কম্পাঙ্ক ( [math]f[/math]
) = 1 / পর্যায়কাল ( [math]T[/math]
).সুতরাং, [math]v = (1/T) \times \lambda[/math]
.এখান থেকে পর্যায়কাল ( [math]T[/math]
) = [math]\lambda / v[/math]
.[math]T = 0.6 m / 6 m/s = 0.1 s[/math]
.শব্দের পর্যায়কাল হলো একটি পূর্ণ তরঙ্গ সম্পন্ন করতে শব্দের যে সময় লাগে। এই ক্ষেত্রে, 0.1 সেকেন্ডে শব্দটি 0.6 মিটার দূরত্ব অতিক্রম করে, যা এর তরঙ্গদৈর্ঘ্যের সমান।

Question:
P ও Q দুটি মাধ্যমে শব্দের বেগ যথাক্রমে [math]300 \mathrm{~ms}^{-1}[/math] এবং [math]350 \mathrm{~ms}^{-1}[/math] । মাধ্যম দুটিতে শব্দের তরঙ্গদৈর্ঘ্যে পার্থক্য 0.1 সুরশলাকার 55 কম্পনে শব্দ Q মাধ্যমে কতদূর যাবে?

Options:
ক. 77 m
খ. 60 m
গ. 38.5 m
ঘ. 30.5 m


Correct Answer: গ

Solution:
সুরশলাকার কম্পাঙ্ক ( [math]f[/math]
) হলো [math]55 \mathrm{~Hz}[/math]
।P মাধ্যমে শব্দের বেগ ( [math]v_P[/math]
) হলো [math]300 \mathrm{~ms}^{-1}[/math]
।Q মাধ্যমে শব্দের বেগ ( [math]v_Q[/math]
) হলো [math]350 \mathrm{~ms}^{-1}[/math]
।আমরা জানি, শব্দের বেগ ( [math]v[/math]
) = কম্পাঙ্ক ( [math]f[/math]
) [math]\times[/math]
 তরঙ্গদৈর্ঘ্য ( [math]\lambda[/math]
)।P মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্য ( [math]\lambda_P[/math]
) = [math]v_P / f = 300 \mathrm{~ms}^{-1} / 55 \mathrm{~Hz}[/math]
Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্য ( [math]\lambda_Q[/math]
) = [math]v_Q / f = 350 \mathrm{~ms}^{-1} / 55 \mathrm{~Hz}[/math]
প্রশ্নে দেওয়া আছে, মাধ্যম দুটিতে শব্দের তরঙ্গদৈর্ঘ্যে পার্থক্য 0.1 m। অর্থাৎ, [math]\lambda_Q - \lambda_P = 0.1 \mathrm{~m}[/math]
।[math]\lambda_Q - \lambda_P = (350/55) - (300/55) = 50/55 = 10/11 \mathrm{~m}[/math]
।তবে, প্রশ্নে দেওয়া তথ্য অনুযায়ী তরঙ্গদৈর্ঘ্যের পার্থক্য [math]0.1 \mathrm{~m}[/math]
। এই তথ্যের উপর ভিত্তি করে হিসেব করা যাক।Q মাধ্যমে সুরশলাকার 55 কম্পনে শব্দ কতদূর যাবে তা নির্ণয় করতে হবে। এটি Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্যের উপর নির্ভর করে।Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্য ( [math]\lambda_Q[/math]
) = [math]v_Q / f = 350 \mathrm{~ms}^{-1} / 55 \mathrm{~Hz}[/math]
আমরা জানি, শব্দের বেগ = কম্পাঙ্ক [math]\times[/math]
 তরঙ্গদৈর্ঘ্য।প্রশ্নে দেওয়া তথ্যে একটি অসঙ্গতি রয়েছে। যদি 55 কম্পাঙ্কের একটি সুরশলাকা ব্যবহার করা হয়, তবে P এবং Q মাধ্যমে তরঙ্গদৈর্ঘ্যের পার্থক্য হবে:[math]\Delta\lambda = \lambda_Q - \lambda_P = (v_Q/f) - (v_P/f) = (350/55) - (300/55) = 50/55 = 10/11 \approx 0.909 \mathrm{~m}[/math]
।কিন্তু প্রশ্নে তরঙ্গদৈর্ঘ্যের পার্থক্য [math]0.1 \mathrm{~m}[/math]
 দেওয়া হয়েছে।তবে, যদি প্রশ্নটি এমন হয় যে, 55 Hz কম্পাঙ্কের একটি শব্দতরঙ্গ P ও Q মাধ্যমে travelling করছে, এবং মাধ্যম দুটিতে শব্দের বেগের পার্থক্য [math]350-300 = 50 \mathrm{~ms}^{-1}[/math]
। এবং 55 কম্পাঙ্কের শব্দের জন্য কোনো একটি মাধ্যমে তরঙ্গদৈর্ঘ্যের পার্থক্য [math]0.1 \mathrm{~m}[/math]
।যদি 55 কম্পাঙ্কের শব্দের জন্য P মাধ্যমে তরঙ্গদৈর্ঘ্য [math]\lambda_P[/math]
 এবং Q মাধ্যমে [math]\lambda_Q[/math]
 হয়, তাহলে [math]\lambda_Q - \lambda_P = 0.1 \mathrm{~m}[/math]
।[math]\lambda_Q = v_Q / f[/math]
 এবং [math]\lambda_P = v_P / f[/math]
.[math]\Delta\lambda = (v_Q - v_P) / f = 0.1 \mathrm{~m}[/math]
 [math]f = (v_Q - v_P) / 0.1 = (350 - 300) / 0.1 = 50 / 0.1 = 500 \mathrm{~Hz}[/math]
.তাহলে কম্পাঙ্ক হবে 500 Hz, 55 Hz নয়।যদি আমরা 55 Hz কম্পাঙ্কের সুরশলাকা এবং প্রদত্ত বেগ ব্যবহার করে Q মাধ্যমে কতদূর যাবে তা জানতে চাই, তাহলে এটি Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্যের সঙ্গে সম্পর্কিত।Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্য ( [math]\lambda_Q[/math]
) = [math]v_Q / f = 350 \mathrm{~ms}^{-1} / 55 \mathrm{~Hz} \approx 6.36 \mathrm{~m}[/math]
।যদি প্রশ্নটি বোঝাতে চায় যে 55 কম্পাঙ্কের একটি সুরশলাকা ব্যবহার করা হচ্ছে এবং P ও Q মাধ্যমে শব্দের বেগ যথাক্রমে 300 m/s ও 350 m/s, এবং কোনো একটি কারণে (সম্ভবত অন্য কম্পাঙ্কের জন্য) তরঙ্গদৈর্ঘ্যের পার্থক্য 0.1 m, তবে ঐ 55 Hz কম্পাঙ্কের শব্দ Q মাধ্যমে কতদূর যাবে তা নির্ণয় করতে হবে।তবে, প্রদত্ত অপশনগুলি দেখে মনে হচ্ছে যে প্রশ্নটি সুরশলাকার কম্পাঙ্ক 55 Hz এবং Q মাধ্যমে শব্দের বেগ 350 m/s ব্যবহার করে Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্য বা অন্য কোনো প্রাসঙ্গিক দৈর্ঘ্য নির্ণয় করতে চাইছে।আসুন, আমরা তরঙ্গদৈর্ঘ্যের পার্থক্য [math]0.1 \mathrm{~m}[/math]
 এবং কম্পাঙ্ক 55 Hz কে একসাথে ব্যবহার করার চেষ্টা করি।যদি 55 Hz কম্পাঙ্কের জন্য তরঙ্গদৈর্ঘ্যের পার্থক্য [math]0.1 \mathrm{~m}[/math]
 হয়, তবে [math]\Delta v = f \times \Delta\lambda = 55 \mathrm{~Hz} \times 0.1 \mathrm{~m} = 5.5 \mathrm{~ms}^{-1}[/math]
।কিন্তু বেগের পার্থক্য [math]350-300 = 50 \mathrm{~ms}^{-1}[/math]
।সুতরাং, এখানে তথ্যে একটি অসামঞ্জস্য রয়েছে।অন্যভাবে চিন্তা করলে, যদি 55 কম্পাঙ্কের সুরশলাকা ব্যবহার করা হয়, তাহলে P মাধ্যমে তরঙ্গদৈর্ঘ্য [math]\lambda_P = 300/55[/math]
 এবং Q মাধ্যমে [math]\lambda_Q = 350/55[/math]
.প্রশ্নানুযায়ী, P ও Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্যে পার্থক্য 0.1 m।তাহলে, [math]\lambda_Q - \lambda_P = (350/55) - (300/55) = 50/55 = 10/11 \approx 0.909 \mathrm{~m}[/math]
।এই [math]0.1 \mathrm{~m}[/math]
 তথ্যটি ব্যবহার করা যাচ্ছে না।যদি আমরা প্রশ্নটিকে এইভাবে ব্যাখ্যা করি যে, 55 Hz কম্পাঙ্কের শব্দের জন্য Q মাধ্যমে শব্দের বেগ 350 m/s এবং P মাধ্যমে 300 m/s, এবং প্রশ্নে উল্লিখিত [math]0.1 \mathrm{~m}[/math]
 হলো P ও Q মাধ্যমে শব্দের কোনো সম্পর্কযুক্ত দৈর্ঘ্য।আসুন, আমরা বিকল্পগুলি দেখে একটি সম্ভাব্য প্রশ্ন গঠন করার চেষ্টা করি।যদি Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্য [math]\lambda_Q[/math]
 হয়, এবং প্রশ্নে पूछा गया है कि "সুরশলাকার 55 কম্পনে শব্দ Q মাধ্যমে কতদূর যাবে?"যদি এটি Q মাধ্যমে তরঙ্গদৈর্ঘ্য বোঝায়, তাহলে [math]\lambda_Q = 350/55 \approx 6.36 \mathrm{~m}[/math]
। এটি কোনো বিকল্পের সাথে মেলে না।যদি প্রশ্নটি বোঝাতে চায় যে, 55 Hz কম্পাঙ্কের শব্দের জন্য Q মাধ্যমে তরঙ্গদৈর্ঘ্য 0.1 m, তাহলে [math]\lambda_Q = 0.1 \mathrm{~m}[/math]
।তখন Q মাধ্যমে শব্দের বেগ হবে [math]v_Q = f \times \lambda_Q = 55 \mathrm{~Hz} \times 0.1 \mathrm{~m} = 5.5 \mathrm{~ms}^{-1}[/math]
।এটি প্রদত্ত 350 m/s এর সাথে মেলে না।যদি ধরা হয় যে, 55 Hz কম্পাঙ্কের একটি সুরশলাকার জন্য Q মাধ্যমে শব্দের বেগ 350 m/s। তাহলে Q মাধ্যমে তরঙ্গদৈর্ঘ্য [math]\lambda_Q = 350/55[/math]
.এখন, যদি P ও Q মাধ্যমে শব্দের তরঙ্গদৈর্ঘ্যে পার্থক্য 0.1 m দেওয়া থাকে, তবে [math]\lambda_Q - \lambda_P = 0.1 \mathrm{~m}[/math]
।[math]\lambda_Q = 350/55[/math]
.[math]350/55 - \lambda_P = 0.1[/math]
 [math]\lambda_P = 350/55 - 0.1 = (350 - 5.5)/55 = 344.5/55 \approx 6.26 \mathrm{~m}[/math]

Question:
নিম্নের কোনটি বাতাসে শব্দের বেগ বৃদ্ধি করে?

Options:
ক. চাপ
খ. আর্দ্রতা
গ. তাপমাত্রা
ঘ. ঘনত্ব
ঙ. b ও c


Correct Answer: খ

Solution:
বাতাসে শব্দের বেগ মূলত দুটি বিষয়ের উপর নির্ভর করে: গ্যাসের অণুর ভর এবং তাপমাত্রা। গ্যাসের তাপমাত্রা বাড়লে অণুর গড় গতিবেগ বৃদ্ধি পায়, ফলে শব্দের বেগ বাড়ে। চাপ বাড়লেও শব্দের বেগের উপর সরাসরি কোনো প্রভাব পড়ে না কারণ চাপ বাড়লে ঘনত্বও সমানুপাতিক হারে বাড়ে, যা শব্দের বেগ কমাতে সাহায্য করে। আর্দ্রতা বাড়লে বাতাসের গড় অণুর ভর কমে যায়, যা শব্দের বেগ বাড়াতে সাহায্য করে, তবে তাপমাত্রার প্রভাব এর চেয়ে বেশি। ঘনত্ব বাড়লে শব্দের বেগ কমে যায়।সাধারণত, তাপমাত্রা [math]1^\circ\text{C}[/math]
 বাড়লে বাতাসে শব্দের বেগ প্রায় [math]0.6\text{ m/s}[/math]
 বৃদ্ধি পায়।শব্দের বেগের সূত্রটি হল: [math]v = \sqrt{\frac{\gamma RT}{M}}[/math]
যেখানে,[math]v[/math]
 = শব্দের বেগ[math]\gamma[/math]
 = গ্যাসটির রুদ্ধতাপীয় সহগ (adiabatic index)[math]R[/math]
 = সার্বজনীন গ্যাস ধ্রুবক[math]T[/math]
 = কেলভিন স্কেলে তাপমাত্রা[math]M[/math]
 = গ্যাসটির মোলার ভরএই সূত্র থেকে দেখা যায় যে, [math]v \propto \sqrt{T}[/math]
। অর্থাৎ, তাপমাত্রা বাড়লে শব্দের বেগ বৃদ্ধি পায়।

Question:
কোন মাধ্যমে [math]2.5m[/math] এবং [math]2.0m[/math] তরঙ্গ দৈর্ঘ্যের দুটি শব্দের কম্পাঙ্কের পার্থক্য [math]250Hz[/math] হলে শব্দের বেগ কত?

Options:
ক. 200 m/s
খ. 312.5 m/s
গ. 2500 m/s
ঘ. 625 m/s
ঙ. 5000 m/s


Correct Answer: গ

Solution:
[math]\begin{array}{l}\text{Hints: } f_{2}-f_{1}=250 ; v=f \lambda\\\text{Solve: } \mathrm{f}_{2}-\mathrm{f}_{1}=250 \\\Rightarrow\left(\frac{v}{\lambda_{2}}-\frac{v}{\lambda_{1}}\right)=250 \\\Rightarrow \mathrm{v}\left(\frac{1}{2}-\frac{1}{2.5}\right)=250 \\\Rightarrow \mathrm{v} \times 0.1=250 \\\Rightarrow \mathrm{v}=\frac{250}{0.1} \\\therefore \mathrm{v}=2500 \mathrm{~m} / \mathrm{s}\end{array}[/math]

Explanation:
ব্যাখ্যাঃ একই মাধ্যমে একই ধরণের সকল তরঙ্গের বেগ একই। একই ধরণের তরঙ্গ বলতে যান্ত্রিক হলে সকল যান্ত্রিক এবং তড়িৎ চৌম্বক তরঙ্গ হলে সকল তড়িৎ চৌম্বক তরঙ্গ বুঝানো হয়েছে।[math]\mathrm{v}=\mathrm{f} \lambda \Rightarrow \mathrm{f}=\frac{1}{\lambda} \mathrm{v} \therefore \mathrm{f} \propto \frac{1}{\lambda}[/math]
 [একই মাধ্যমে বেগ একই হওয়ায় v স্থির]অর্থাৎ কম্পাঙ্ক ও তরঙ্গ দৈর্ঘ্য একই মাধ্যমে পরস্পরের ব্যস্তানুপাতিক। তাই যে তরঙ্গের তরঙ্গদৈর্ঘ্য বড় তার কম্পাঙ্ক ছোট হবে।[math]\lambda_{1}>\lambda_{2}[/math]
 হওয়ায় [math]f_{2}>f_{1}[/math]
 হবে। [math]\therefore \mathrm{f}_{2}-\mathrm{f}_{1}=250[/math]

 ।এই সম্পর্কটি শব্দের তরঙ্গ প্রকৃতির একটি মৌলিক ধারণা। কম্পাঙ্ক হলো প্রতি সেকেন্ডে কম্পনের সংখ্যা, যা উৎসের ওপর নির্ভর করে এবং মাধ্যমের পরিবর্তনের সাথে পরিবর্তিত হয় না। অন্যদিকে, বেগ নির্ভর করে মাধ্যমের প্রকৃতির ওপর (যেমন ঘনত্ব, স্থিতিস্থাপকতা)। যখন বেগ পরিবর্তিত হয় এবং কম্পাঙ্ক স্থির থাকে, তখন তরঙ্গদৈর্ঘ্যও অবশ্যই পরিবর্তিত হবে যাতে [math]v=f\lambda[/math]
 সম্পর্কটি বজায় থাকে।

Question:
রেডিও তরঙ্গের বেগ [math]3 \times 10^{8} \mathrm{~m} \cdot \mathrm{s}^{-1}[/math]। একটি রেডিও স্টেশন 300 m তরঙ্গ দৈর্ঘ্যে অনুষ্ঠান প্রচার করে। এর কম্পাঙ্ক কত?

Options:
ক. 300 kHz
খ. 1 MHz
গ. 1 kHz
ঘ. 1 Hz


Correct Answer: খ

Solution:
আমরা জানি, বেগ ( [math]v[/math]
), কম্পাঙ্ক ( [math]f[/math]
) এবং তরঙ্গদৈর্ঘ্য ( [math]\lambda[/math]
) এর মধ্যে সম্পর্ক হল: [math]v = f \lambda[/math]
 ।এখানে, বেগ [math]v = 3 \times 10^8 \mathrm{~m} \cdot \mathrm{s}^{-1}[/math]
 এবং তরঙ্গদৈর্ঘ্য [math]\lambda = 300 \mathrm{~m}[/math]
 ।আমাদের কম্পাঙ্ক ( [math]f[/math]
) নির্ণয় করতে হবে।সমীকরণটি পুনর্বিন্যাস করে পাই: [math]f = \frac{v}{\lambda}[/math]

কম্পাঙ্কের একক হার্টজ (Hz) বা সেকেন্ড ইনভার্স ( [math]\mathrm{s}^{-1}[/math]
) ।সুতরাং, [math]f = 10^6 \mathrm{~Hz}[/math]
 ।আমরা জানি, [math]1 \mathrm{~MHz} = 10^6 \mathrm{~Hz}[/math]

Question:
কোন সুরশলাকা একটি মাধ্যমে 5 মিটার দৈর্ঘ্যের এবং 350 মিটার/সেকেন্ড বেগের তরঙ্গ উৎপন্ন করে। অপর একটি মাধ্যমে এর বেগ 700 মিটার/সেকেন্ড হলে ঐ মাধ্যমে সুরশলাকার 100 কম্পনে শব্দ কত দূরে যাবে?

Options:
ক. 1200m
খ. 1000m
গ. 10m
ঘ. 120m


Correct Answer: খ

Solution:
শব্দের বেগ ( [math]v[/math]
), কম্পাঙ্ক ( [math]f[/math]
) এবং তরঙ্গদৈর্ঘ্যের ( [math]\lambda[/math]
) মধ্যে সম্পর্ক হলো [math]v = f \lambda[/math]
। সুরশলাকার কম্পাঙ্ক ( [math]f[/math]
) মাধ্যমে পরিবর্তিত হয় না। প্রথম মাধ্যমে, [math]v_1 = 350[/math]
 মি/সেকেন্ড এবং [math]\lambda_1 = 5[/math]
 মিটার। সুতরাং, কম্পাঙ্ক [math]f = \frac{v_1}{\lambda_1} = \frac{350 \text{ মি/সেকেন্ড}}{5 \text{ মিটার}} = 70[/math]
 হার্টজ।দ্বিতীয় মাধ্যমে, সুরশলাকার কম্পাঙ্ক একই থাকে, অর্থাৎ [math]f = 70[/math]
 হার্টজ এবং বেগ [math]v_2 = 700[/math]
 মি/সেকেন্ড।শব্দ 100 কম্পনে কত দূরে যাবে তা নির্ণয় করতে, আমাদের প্রথমে 100 কম্পনের জন্য মোট সময় বের করতে হবে। সময় ( [math]t[/math]
) = কম্পনের সংখ্যা ( [math]n[/math]
) / কম্পাঙ্ক ( [math]f[/math]
) = [math]100 / 70[/math]
 সেকেন্ড।শব্দ যে দূরত্ব অতিক্রম করবে তা হলো দূরত্ব ( [math]d[/math]
) = বেগ ( [math]v_2[/math]
) [math]\times[/math]
 সময় ( [math]t[/math]
) = [math]700 \text{ মি/সেকেন্ড} \times \frac{100}{70} \text{ সেকেন্ড} = 700 \times \frac{10}{7} \text{ মিটার} = 100 \times 10 \text{ মিটার} = 1000 \text{ মিটার}[/math]
।এখানে, সুরশলাকার কম্পাঙ্ক ধ্রুব থাকে। যখন শব্দের বেগ বৃদ্ধি পায়, তখন তরঙ্গদৈর্ঘ্যও আনুপাতিক হারে বৃদ্ধি পায়।

Question:
[math]y=0.6 \sin 0.12 x \cos 24 t[/math] একটি স্থির তরঙ্গের সমীকরণ যা নিম্নলিখিত চিত্র দ্বারা প্রকাশিত [ [math]x[/math] ও [math]y[/math] মিটার এককে]

[Image]


উদ্দীপকের তথ্যের আলোকে নিচের প্রশ্নের উত্তর দাও :

স্থির তরঙ্গ গঠনকারী তরঙ্গের বেগ —

Options:
ক. [math]100 \mathrm{~m} \cdot \mathrm{s}^{-1}[/math]
খ. [math]200 \mathrm{~m} \cdot \mathrm{s}^{-1}[/math]
গ. [math]300 \mathrm{~m} \cdot \mathrm{s}^{-1}[/math]
ঘ. [math]400 \mathrm{~m} \cdot \mathrm{s}^{-1}[/math]


Correct Answer: B.[math]200 \mathrm{~m} \cdot \mathrm{s}^{-1}[/math]

Solution:
প্রদত্ত তরঙ্গের সমীকরণ:[math]y = 0.6 \sin(0.12x) \cos(24t)[/math]
এটি একটি স্থির তরঙ্গের সমীকরণ, যার সাধারণ রূপ:[math]y = 2A \sin(kx) \cos(\omega t)[/math]
এখানে,[math]k = 0.12[/math]
, [math]\omega = 24[/math]
ধাপ ১: তরঙ্গদৈর্ঘ্য নির্ণয়[math]k = \dfrac{2\pi}{\lambda} \Rightarrow \lambda = \dfrac{2\pi}{k} = \dfrac{2\pi}{0.12} \approx \dfrac{6.28}{0.12} = 52.33 \, \text{m}[/math]
ধাপ ২: কম্পাংক (frequency) নির্ণয়[math]\omega = 2\pi f \Rightarrow f = \dfrac{\omega}{2\pi} = \dfrac{24}{2\pi} = \dfrac{24}{6.28} \approx 3.82 \, \text{Hz}[/math]
ধাপ ৩: তরঙ্গের বেগ নির্ণয়[math]v = f \lambda = 3.82 \times 52.33 \approx 200 \, \text{m/s}[/math]
অতএব, তরঙ্গের বেগঃ[math]\boxed{v \approx 200 \, \text{m/s}}[/math]

Question:
একটি তরঙ্গ [math]200[/math] টি পূর্ণ কম্পন সম্পন্ন করে [math]8\ m[/math] দূরত্ব [math]0.25\ s[/math] এ অতিক্রম করলে তরঙ্গ দৈর্ঘ্য হবে—

Options:
ক. 8 cm
খ. 4 cm
গ. 32 cm
ঘ. 25 cm


Correct Answer: খ

Solution:
তরঙ্গের বেগ ( [math]v[/math]
) হল দূরত্ব ( [math]d[/math]
) কে সময় ( [math]t[/math]
) দিয়ে ভাগ করলে যা পাওয়া যায়, অর্থাৎ [math]v = d/t[/math]
। এখানে, [math]d = 8\ m[/math]
 এবং [math]t = 0.25\ s[/math]
। সুতরাং, [math]v = 8\ m / 0.25\ s = 32\ m/s[/math]
।তরঙ্গের বেগ ( [math]v[/math]
), তরঙ্গ দৈর্ঘ্য ( [math]\lambda[/math]
) এবং কম্পাঙ্ক ( [math]f[/math]
) এর মধ্যে সম্পর্ক হল [math]v = f\lambda[/math]
।কম্পাঙ্ক ( [math]f[/math]
) হল প্রতি সেকেন্ডে পূর্ণ কম্পনের সংখ্যা। এখানে, [math]200[/math]
 টি পূর্ণ কম্পন সম্পন্ন হয় [math]0.25\ s[/math]
 এ। তাহলে, [math]f = 200 \text{ কম্পন} / 0.25\ s = 800\ Hz[/math]
।এখন, আমরা তরঙ্গ দৈর্ঘ্য ( [math]\lambda[/math]
) বের করতে পারি:[math]\lambda = v / f[/math]
 [math]\lambda = 32\ m/s / 800\ Hz[/math]
 [math]\lambda = 0.04\ m[/math]
যেহেতু অপশনগুলো সেন্টিমিটারে দেওয়া আছে, তাই মিটারকে সেন্টিমিটারে রূপান্তর করতে হবে:[math]0.04\ m \times 100\ cm/m = 4\ cm[/math]
অতএব, তরঙ্গ দৈর্ঘ্য হবে [math]4\ cm[/math]
।তরঙ্গ দৈর্ঘ্য হলো দুটি পরপর অনুরূপ বিন্দুর মধ্যেকার দূরত্ব, যেমন দুটি পরপর শীর্ষবিন্দু বা দুটি পরপর পাদবিন্দু। কম্পাঙ্ক হলো প্রতি সেকেন্ডে একটি পূর্ণ কম্পন সম্পন্ন করার সংখ্যা।

Question:
[math]P[/math] মাধ্যমে শব্দের বেগ [math]Q[/math] মাধ্যমে শব্দের বেগের [math]4[/math] গুণ। শব্দের তরঙ্গদৈর্ঘ্যের পার্থক্য [math]2\ m[/math] এবং [math]Q[/math] মাধ্যমে শব্দের বেগ [math]350\ ms^{-1}[/math]।

উদ্দীপকের তথ্যের আলোকে নিচের প্রশ্নের উত্তর দাও :

[math]Q[/math] মাধ্যমে শব্দের কম্পাঙ্ক—

Options:
ক. 425 Hz
খ. 525 Hz
গ. 625 Hz
ঘ. 725 Hz


Correct Answer: খ

Solution:
শব্দের বেগ ( [math]v[/math]
), কম্পাঙ্ক ( [math]f[/math]
) এবং তরঙ্গদৈর্ঘ্য ( [math]\lambda[/math]
) এর মধ্যে সম্পর্ক হলো [math]v = f\lambda[/math]
.দেওয়া আছে [math]Q[/math]
 মাধ্যমে শব্দের বেগ [math]v_Q = 350\ ms^{-1}[/math]

 মাধ্যমে শব্দের বেগ [math]Q[/math]
 মাধ্যমে শব্দের বেগের [math]4[/math]
 গুণ, অর্থাৎ [math]v_P = 4v_Q[/math]

উদ্দীপকের তথ্যের আলোকে নিচের প্রশ্নের উত্তর দাও :

, অর্থাৎ [math]|\lambda_P - \lambda_Q| = 2\ m[/math]
 ।আমরা জানি, [math]v = f\lambda[/math]
, যেখানে [math]v[/math]
 শব্দের বেগ, [math]f[/math]
