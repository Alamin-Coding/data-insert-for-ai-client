Question:
দুটি সুরশলাকার কম্পাঙ্ক যথাক্রমে 256 Hz ও 512 Hz। বাতাসে এদের তরঙ্গদৈর্ঘ্যের তুলনা করো।

Options:


Correct Answer: 

Solution:
( Answer : 2 : 1 )

Explanation:
বাতাসে তরঙ্গগত গতি একই থাকে এবং তরঙ্গদৈর্ঘ্য ও কম্পাঙ্কের মাঝের সম্পর্ক হলো [math]\lambda = \dfrac{v}{f}[/math]. তাই [math]\lambda_1=\dfrac{v}{256},\;\lambda_2=\dfrac{v}{512}[/math], ফলে [math]\lambda_1=2\lambda_2[/math] — অর্থাৎ 256 Hz সুরশলাকারটির তরঙ্গদৈর্ঘ্য 512 Hz সুরশলাকারটির দ্বিগুণ।

Question:
দুটি সুর শলাকার কম্পাঙ্ক যথাক্রমে 128 Hz ও 384 Hz। বায়ুতে শলাকা দুটি হতে সৃষ্ট শব্দের তরঙ্গ দৈর্ঘ্যের অনুপাত নির্ণয় করো।

Options:


Correct Answer: 

Solution:
( Answer : [math]3: 1[/math] )

Explanation:
ব্যাখ্যা: বায়ুতে শব্দের বেগ ধ্রুবক হওয়ায় [math]\lambda = v/f[/math] এবং তরঙ্গদৈর্ঘ্য ফ্রিকোয়েন্সির বিপরীত সহগত; তাই [math]\lambda_1:\lambda_2 = f_2:f_1 = 384:128 = 3:1[/math], অর্থাৎ প্রথমটির তরঙ্গদৈর্ঘ্য দ্বিতীয়টির তিনগুণ।

Question:
তিনটি সুর শলাকার কম্পাঙ্ক যথাক্রমে 123, 369 এবং 615Hz। বায়ুতে এদের তরঙ্গ দৈর্ঘ্যের অনুপাত নির্ণয় কর ।

Options:


Correct Answer: 

Solution:
( Answer : [math]15: 5: 3[/math] )

Explanation:
ব্যাখ্যা: একই মাধ্যম হলে তরঙ্গদৈর্ঘ্য ফ্রিকোয়েন্সির উল্টো অনুপাত, [math]\lambda = v/f[/math]. এখানে [math]369=3\times123,\;615=5\times123[/math], সুতরাং তরঙ্গদৈর্ঘ্যের অনুপাত [math]1:1/3:1/5[/math] = [math]15:5:3[/math].

Options:


Correct Answer:

( Answer : 3 : 1 )

Explanation:
ব্যাখ্যা: বায়ুতে শব্দের গতি একই হওয়ায় [math]\lambda = v/f[/math] এবং তাই [math]\lambda_1/\lambda_2 = f_2/f_1 = 384/128 = 3[/math], অর্থাৎ 128 Hz তরঙ্গের তরঙ্গদৈর্ঘ্য 384 Hz‑এর তুলনায় তিন গুণ বড়: [math]\lambda_{128}=3\lambda_{384}[/math].
