Question:
27°C তাপমাত্রায় 5g নাইট্রোজেনের গতিশক্তি নির্ণয় করো। (নাইট্রোজেনের গ্রাম আনবিক ভর = 28g )

Options:


Correct Answer: 

Solution:
( Answer : [math]667.8 \mathrm{~J}[/math] )

Explanation:
প্রথমে তাপমাত্রা কেলভিনে নাও: [math]T=27+273=300\ \text{K}[/math] এবং মোল সংখ্যা [math]n=\dfrac{5}{28}\ \text{mol}[/math]।  
নাইট্রোজেনের মোট গতিশক্তি [math]E_{\text{kin}}=\tfrac{3}{2}nRT[/math], তাই [math]E_{\text{kin}}=\tfrac{3}{2}\times\dfrac{5}{28}\times8.314\times300\approx6.68\times10^2\ \text{J}\ (\approx668\ \text{J})[/math]।

Correct Answer: 

Solution:
( Answer : [math]400.66 \mathrm{~J}[/math] )

Question:
27°C তাপমাত্রায় প্রতি মোল হিলিয়ামের গতিশক্তি নির্ণয় করো।

Options:


Correct Answer: 

Solution:
( Answer : [math]3.74 \times 10^{3} \mathrm{~J}[/math] )

Explanation:
ব্যাখ্যা: একটি মনৌপরমাণু গ্যাস (হিলিয়াম)‑এর প্রতি মোল গড় গতিশক্তি তিনটি অনুবর্তী মাত্রার কারণে [math]\frac{3}{2}RT[/math] হয়। এখানে [math]T=27^\circ\text{C}=300\ \text{K}[/math], তাই [math]E=\frac{3}{2}RT=\frac{3}{2}\times 8.314\ \text{J·mol}^{-1}\text{K}^{-1}\times 300\ \text{K}=3741.3\ \text{J·mol}^{-1}\approx 3.74\times10^{3}\ \text{J·mol}^{-1}.[/math]

Question:
27°C তাপমাত্রায় 1 gm নাইট্রোজেনের গতিশক্তি নির্ণয় করো ৷

Options:


Correct Answer: 

Solution:
( Answer : [math]133.65 \mathrm{~J}[/math] )

Explanation:
ব্যাখ্যা: গ্যাসের মোট গতিশক্তি ক্যালকুল করতে equipartition theorem অনুযায়ী ব্যবহার করি [math]E_{kin}=\frac{3}{2}nRT[/math]। এখানে [math]n=\frac{1\ \text{g}}{28\ \text{g mol}^{-1}}=\frac{1}{28}\ \text{mol}[/math], [math]T=27+273=300\ \text{K}[/math] এবং [math]R=8.314\ \text{J mol}^{-1}\text{K}^{-1}[/math], তাই [math]E_{kin}=\frac{3}{2}\times\frac{1}{28}\times 8.314\times300\approx133.6\ \text{J}\ (\approx134\ \text{J})[/math]। কারণ প্রতিটি অনুপরিমাণে ট্রান্সলেশনাল ডিগ্রি‑অফ‑ফ্রিডমের জন্য এনার্জি [math]\frac{3}{2}kT[/math] (মোলের জন্য [math]\frac{3}{2}RT[/math]) পাওয়া যায়।

Question:
29°C তাপমাত্রায় 3g নাইট্রোজেন গ্যাসের গতিশক্তি নির্ণয় করো? [ নাইট্রোজেনের গ্রাম আণবিক ভর 28 gm ]

Options:


Correct Answer: 

Solution:
( Answer : [math]402.79 \mathrm{~J}[/math] )

Explanation:
বস্তুগত ব্যাখ্যা: নিঃসন্দেহে গ্যাসের মোট অনুক্রমিক গতিশক্তি দেওয়া হয় [math]E=\frac{3}{2}nRT[/math] সূত্রে, যেখানে [math]n[/math] মোল সংখ্যা। এখানে [math]n=\frac{3\ \text{g}}{28\ \text{g mol}^{-1}}=\frac{3}{28}[/math], [math]T=29+273=302\ \text{K}[/math] এবং [math]R=8.314\ \text{J mol}^{-1}\text{K}^{-1}[/math]। হিসাব করলে: [math]E=\frac{3}{28}\times\frac{3}{2}\times8.314\times302\approx403.5\ \text{J}\ ( \approx4.03\times10^{2}\ \text{J})[/math]।

Correct Answer:

( Answer : 3739.5 J)
