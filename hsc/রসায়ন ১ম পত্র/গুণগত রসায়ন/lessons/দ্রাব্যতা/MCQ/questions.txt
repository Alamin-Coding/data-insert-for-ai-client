Question:
[math]_{92}^{\mathrm{x}}\mathrm{U}+_0^1\mathrm{n}=_{36}^{92}\mathrm{Kr}+_{_{56}}^{\mathrm{Y}}\mathrm{Ba}+3_0^1\mathrm{n}+\text{ Energy }[/math], X এবং Y হচ্ছে -

Options:
A. 235, 141
B. 234,150
C. 236, 142
D. 233,139


Correct Answer: ক

Solution:
পারমাণবিক বিক্রিয়ার আগে ও পরে ভর সংখ্যা এবং পারমাণবিক সংখ্যা সংরক্ষিত থাকে।প্রথমে X এবং Y এর মান নির্ণয় করি:[math]_{92}^{\mathrm{X}}\mathrm{U}+_0^1\mathrm{n}=_{36}^{92}\mathrm{Kr}+_{56}^{\mathrm{Y}}\mathrm{Ba}+3_0^1\mathrm{n}+\text{ Energy }[/math]
ভর সংখ্যা সংরক্ষণ:X + 1 = 92 + Y + 3(1)X + 1 = 92 + Y + 3X = 94 + Y ...(1)পারমাণবিক সংখ্যা সংরক্ষণ:92 + 0 = 36 + 5692 = 92এখানে পারমাণবিক সংখ্যা সংরক্ষিত আছে।এখন, প্রশ্ন অনুযায়ী X এবং Y এর মান বের করতে হবে। বিকল্পগুলি দেখে X এবং Y এর মান বসিয়ে দেখি কোনটিতে সমীকরণ (1) সিদ্ধ হয়।বিকল্প A: X = 235, Y = 141X = 94 + Y235 = 94 + 141235 = 235এটি সিদ্ধ হয়।বিকল্প C: X = 236, Y = 142X = 94 + Y236 = 94 + 142236 = 236এটিও সিদ্ধ হয়।কিন্তু ইউরেনিয়াম-২৩৫ ( [math]_{92}^{235}\mathrm{U}[/math]
) নিউট্রন দ্বারা আঘাতপ্রাপ্ত হলে সাধারণত ইউরেনিয়াম-২৩৬ ( [math]_{92}^{236}\mathrm{U}[/math]
) গঠিত হয় যা পরবর্তীতে বিভাজিত হয়। প্রশ্নে পারমাণবিক বিক্রিয়াটি ইউরেনিয়াম-২৩৫ এর বিভাজনকে নির্দেশ করছে। তাই X এর মান 235 হওয়া অধিকতর যুক্তিযুক্ত।অতএব, X=235 হলে Y=141 হয়। এই বিক্রিয়াটি পারমাণবিক বোমা বা চুল্লিতে ইউরেনিয়াম-২৩৫-এর নিউক্লিয়ার ফিশন (nuclear fission) প্রক্রিয়ার একটি উদাহরণ। এই প্রক্রিয়ায় একটি ভারী নিউক্লিয়াস (যেমন ইউরেনিয়াম) দুটি অপেক্ষাকৃত হালকা নিউক্লিয়াসে বিভক্ত হয় এবং প্রচুর শক্তি নির্গত করে।

Question:
তেজক্রিয় মৌলের নিউক্লিয়াস হতে বিটা-রশ্মির বিকিরণে কি বৃদ্ধি পায়?

Options:
ক. ইলেকট্রন
খ. প্রোটন
গ. নিউট্রন
ঘ. কোনটাই নয়


Correct Answer: খ

Solution:
বিটা-রশ্মি বিকিরণের সময়, তেজস্ক্রিয় মৌলের নিউক্লিয়াসে একটি নিউট্রন একটি প্রোটনে রূপান্তরিত হয়, যার ফলে প্রোটন সংখ্যা বৃদ্ধি পায়। এই প্রক্রিয়ায় একটি ইলেকট্রন (বিটা-কণা) এবং একটি অ্যান্টিনিউট্রিনো নির্গত হয়। তাই, বিটা-রশ্মির বিকিরণে প্রোটনের সংখ্যা বৃদ্ধি পায়।এই রূপান্তরকে নিম্নোক্তভাবে লেখা যেতে পারে:[math]n \rightarrow p + e^- + \bar{
u}_e[/math]
যেখানে:- [math]n[/math]
 হলো একটি নিউট্রন- [math]p[/math]
 হলো একটি প্রোটন- [math]e^-[/math]
 হলো একটি ইলেকট্রন (বিটা-কণা)- [math]\bar{
u}_e[/math]
 হলো একটি ইলেকট্রন অ্যান্টিনিউট্রিনোএই কারণে, পারমাণবিক সংখ্যা (প্রোটন সংখ্যা) এক বৃদ্ধি পায় এবং ভর সংখ্যা (প্রোটন ও নিউট্রনের মোট সংখ্যা) অপরিবর্তিত থাকে।

এই বিক্রিয়াটিতে, একটি নিকেল নিউক্লিয়াস (Nickel nucleus) একটি প্রোটন (proton) গ্রহণ করে এবং একটি তামা নিউক্লিয়াসে (copper nucleus) রূপান্তরিত হয়, সাথে একটি নিউট্রন (neutron) নির্গত হয়। এই প্রক্রিয়াটি একটি মৌলকে অন্য মৌলে রূপান্তর করে, যা ট্রান্সম্যুটেশন (Transmutation) নামে পরিচিত।নিউক্লীয় ফিউশন (Nuclear fusion) হল হালকা নিউক্লিয়াস একত্রিত হয়ে ভারী নিউক্লিয়াস গঠন করার প্রক্রিয়া, যেমন সূর্যের শক্তি উৎপাদন। নিউক্লীয় ফিশন (Nuclear fission) হল একটি ভারী নিউক্লিয়াস ভেঙে দুটি হালকা নিউক্লিয়াস ও শক্তি উৎপন্ন করার প্রক্রিয়া, যেমন পারমাণবিক চুল্লিতে ঘটে। ট্রান্সফরমেশন (Transformation) একটি ব্যাপক শব্দ, তবে পারমাণবিক পদার্থবিদ্যায় একটি মৌলের অন্য মৌলে রূপান্তরকে সুনির্দিষ্টভাবে বোঝানোর জন্য ট্রান্সম্যুটেশন শব্দটি ব্যবহৃত হয়।

Type: MCQ | Book: রসায়ন ১ম পত্র | Years: 2014-2015, 2015-2016 | Boards:  | Tags: University of Dhaka (A Unit), Jagannath University (A Unit)

Question:
নিম্নের নিউক্লিয়ার বিক্রিয়ায় X- কণাটি কী ?

[math]_4^9\mathrm{Be}+\mathrm{X}\rightarrow_{\ 6}^{12}\mathrm{C}+_0^1\mathrm{n}[/math]

Options:
ক. [math]\alpha \text { - particle }[/math]
খ. [math]\beta-\text { particle }[/math]
গ. [math]\gamma-\text { ray }[/math]
ঘ. [math]\text { Neutron }[/math]


Correct Answer: ক

Solution:
নিউক্লিয়ার বিক্রিয়ায় ভরসংখ্যা এবং পারমাণবিক সংখ্যা উভয়ই সংরক্ষিত থাকে। প্রদত্ত বিক্রিয়াটি হলো: [math]_4^9\mathrm{Be}+\mathrm{X}\rightarrow_{\ 6}^{12}\mathrm{C}+_0^1\mathrm{n}[/math]
।বামপক্ষের মোট ভরসংখ্যা = [math]9 + (\mathrm{X}-এর \ ভরসংখ্যা)[/math]
বামপক্ষের মোট পারমাণবিক সংখ্যা = [math]4 + (\mathrm{X}-এর \ পারমাণবিক সংখ্যা)[/math]
ডানপক্ষের মোট ভরসংখ্যা = [math]12 + 1 = 13[/math]
ডানপক্ষের মোট পারমাণবিক সংখ্যা = [math]6 + 0 = 6[/math]
ভরসংখ্যার সংরক্ষণশীলতা থেকে পাই:[math]9 + (\mathrm{X}-এর \ ভরসংখ্যা) = 13[/math]
 [math]\mathrm{X}-এর \ ভরসংখ্যা = 13 - 9 = 4[/math]
পারমাণবিক সংখ্যার সংরক্ষণশীলতা থেকে পাই:[math]4 + (\mathrm{X}-এর \ পারমাণবিক সংখ্যা) = 6[/math]
 [math]\mathrm{X}-এর \ পারমাণবিক সংখ্যা = 6 - 4 = 2[/math]
অতএব, X কণাটির পারমাণবিক সংখ্যা ২ এবং ভরসংখ্যা ৪। এই বৈশিষ্ট্যটি [math]\alpha[/math]
 কণার সাথে মেলে, যা একটি হিলিয়াম নিউক্লিয়াস ( [math]_2^4\mathrm{He}[/math]
) ।আলফা কণা ( [math]_2^4\mathrm{He}[/math]
) হল হিলিয়াম পরমাণুর নিউক্লিয়াস, যা দুটি প্রোটন এবং দুটি নিউট্রন নিয়ে গঠিত। এটি তেজস্ক্রিয় ক্ষয়ের একটি সাধারণ রূপ।

--------------------------------------------------------------------------------
Type: MCQ | Book: রসায়ন ১ম পত্র | Years: 2014-2015, 2015-2016 | Boards:  | Tags: University of Dhaka (A Unit), Jagannath University (A Unit)

Question:
নিম্নের নিউক্লিয়ার বিক্রিয়ায় X- কণাটি কী ?

[math]_4^9\mathrm{Be}+\mathrm{X}\rightarrow_{\ 6}^{12}\mathrm{C}+_0^1\mathrm{n}[/math]

Options:
ক. [math]\alpha \text { - particle }[/math]
খ. [math]\beta-\text { particle }[/math]
গ. [math]\gamma-\text { ray }[/math]
ঘ. [math]\text { Neutron }[/math]


Correct Answer: ক

Solution:
নিউক্লিয়ার বিক্রিয়ায় ভরসংখ্যা এবং পারমাণবিক সংখ্যা উভয়ই সংরক্ষিত থাকে। প্রদত্ত বিক্রিয়াটি হলো: [math]_4^9\mathrm{Be}+\mathrm{X}\rightarrow_{\ 6}^{12}\mathrm{C}+_0^1\mathrm{n}[/math]
।বামপক্ষের মোট ভরসংখ্যা = [math]9 + (\mathrm{X}-এর \ ভরসংখ্যা)[/math]
বামপক্ষের মোট পারমাণবিক সংখ্যা = [math]4 + (\mathrm{X}-এর \ পারমাণবিক সংখ্যা)[/math]
ডানপক্ষের মোট ভরসংখ্যা = [math]12 + 1 = 13[/math]
ডানপক্ষের মোট পারমাণবিক সংখ্যা = [math]6 + 0 = 6[/math]
ভরসংখ্যার সংরক্ষণশীলতা থেকে পাই:[math]9 + (\mathrm{X}-এর \ ভরসংখ্যা) = 13[/math]
 [math]\mathrm{X}-এর \ ভরসংখ্যা = 13 - 9 = 4[/math]
পারমাণবিক সংখ্যার সংরক্ষণশীলতা থেকে পাই:[math]4 + (\mathrm{X}-এর \ পারমাণবিক সংখ্যা) = 6[/math]
 [math]\mathrm{X}-এর \ পারমাণবিক সংখ্যা = 6 - 4 = 2[/math]
অতএব, X কণাটির পারমাণবিক সংখ্যা ২ এবং ভরসংখ্যা ৪। এই বৈশিষ্ট্যটি [math]\alpha[/math]
 কণার সাথে মেলে, যা একটি হিলিয়াম নিউক্লিয়াস ( [math]_2^4\mathrm{He}[/math]
) ।আলফা কণা ( [math]_2^4\mathrm{He}[/math]
) হল হিলিয়াম পরমাণুর নিউক্লিয়াস, যা দুটি প্রোটন এবং দুটি নিউট্রন নিয়ে গঠিত। এটি তেজস্ক্রিয় ক্ষয়ের একটি সাধারণ রূপ।

Question:
নিচের নিউক্লিয়ার বিক্রিয়ায় X-কে বলা যায়?

নিউক্লিয়ার বিক্রিয়ায় ভর সংখ্যা ও পারমাণবিক সংখ্যা সংরক্ষিত থাকে। প্রদত্ত বিক্রিয়াটি হলো:[math]{ }_{7}^{14} \mathrm{N} + { }_{2}^{4} \mathrm{He} \longrightarrow { }_{8}^{17} \mathrm{O} + \mathrm{X}[/math]
এখানে, নাইট্রোজেন ( [math]{ }_{7}^{14} \mathrm{N}[/math]
) একটি আলফা কণা ( [math]{ }_{2}^{4} \mathrm{He}[/math]
) শোষণ করে অক্সিজেন ( [math]{ }_{8}^{17} \mathrm{O}[/math]
) এবং X উৎপন্ন করে।ভর সংখ্যা সংরক্ষণ নীতি অনুসারে:[math]14 + 4 = 17 + \text{X-এর ভর সংখ্যা}[/math]

পারমাণবিক সংখ্যা সংরক্ষণ নীতি অনুসারে:[math]7 + 2 = 8 + \text{X-এর পারমাণবিক সংখ্যা}[/math]
 [math]9 = 8 + \text{X-এর পারমাণবিক সংখ্যা}[/math]
