Question:
x এবং y উভয়ই বিজোড় সংখ্যা হলে কোনটি অবশ্যই জোড় সংখ্যা?

Options:


Correct Answer: গ

Solution:
x এবং y উভয়ই বিজোড় সংখ্যা জানা আছে, বিজোড় × বিজোড় = বিজোড়

বিজোড় + বিজোড় = জোড়

∴ x + y + 1 = (x + y) + 1 = জোড় + 1 = বিজোড়

xy = বিজোড় × বিজোড় = বিজোড়

x + y = বিজোড় + বিজোড় = জোড়

xy + 2 = বিজোড় × বিজোড় + 2 = বিজোড় + 2 = বিজোড়

∴ সঠিক উত্তর (x + y)

Explanation:
ব্যাখ্যা: যদি x ও y বিজোড় হয়, ধরে নিই [math]x=2m+1[/math] এবং [math]y=2n+1[/math] (m,n পূর্ণসংখ্যা)। তখন [math]x+y=2(m+n+1)[/math] যা 2 দ্বারা বিভাজ্য, অর্থাৎ জোড়, তাই গ সঠিক।

Correct Answer: খ

Solution:
ধরি, দৈর্ঘ্য = x মি এবং প্রন্থ = y মি

প্রশ্নমতে, xy = 7200 .....(i)

আবার, x(y – 3) = 7200 — 576 ⇒ xy - 3x = 7200 – 576

⇒ 7200 - 3x = 7200 – 576 [ (i) নং এর সাহায্যে]

⇒ - 3x = 576 ⸫ x = 192 মি.

(i) নং হতে, xy = 7200 = 192y = 7200

⸫ y = 37.50 মিটার
