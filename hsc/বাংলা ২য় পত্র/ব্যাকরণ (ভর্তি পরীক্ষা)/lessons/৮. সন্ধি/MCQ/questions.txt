Question #15 (ID: 275740)
--------------------------------------------------------------------------------
Type: MCQ | Book: বাংলা ২য় পত্র | Years: 2023-2024, 2019-2020 | Boards:  | Tags: University of Chittagong (Unit D/D1), Khulna University C Unit)
--------------------------------------------------------------------------------
Question:
‘শিরশ্ছেদ’ এর সন্ধি-বিচ্ছেদ কী?

Options:
ক. শির + ছেদ
খ. শিরশ + ছেদ
গ. শিরঃ + ছেদ
ঘ. শির + শ্ছেদ


Correct Answer: C.শিরঃ + ছেদ

Solution:
সন্ধির নিয়মে সঠিক শিরঃ + ছেদ = শিরশ্ছেদ।

বিসর্গের পরে তালব্য অঘোষ ধ্বনি (চ, ছ) থাকলে বিসর্গের জায়গায় তালব্য শিশ (শ) ধ্বনি,

বিসর্গের পরে মূর্ধণ্য অঘোষ ধ্বনি (ট, ঠ) থাকলে বিসর্গের জায়গায় মূর্ধণ্য শিশ (ষ) ধ্বনি,

বিসর্গের পরে দন্ত্য অঘোষ ধ্বনি (ত, থ) থাকলে বিসর্গের জায়গায় দন্ত্য শিশ (স) ধ্বনি হয়।

অর্থাৎ,

‘ঃ’ - এর পরে ‘চ/ছ’ (তালব্য) থাকলে ‘ঃ’ - এর জায়গায় ‘শ’

‘ঃ’ - এর পরে ‘ট/ঠ’ (মূর্ধণ্য) থাকলে ‘ঃ’ - এর জায়গায় ‘ষ’

‘ঃ’ - এর পরে ‘ত/থ’ (দন্ত্য) থাকলে ‘ঃ’ - এর জায়গায় ‘স’ হয়। যেমন - ঃ

+ চ/ছ = শ + চ/ছ

নিঃ + চয় = নিশ্চয়,

শিরঃ + ছেদ = শিরশ্ছেদ।

'চলোর্মি' এর সন্ধিবিচ্ছেদ-

Options:
ক. চল + উর্মি
খ. চলা + উর্মি
গ. চল+ ঊর্মি
ঘ. চলা + উর্মি


Correct Answer: C. চল+ ঊর্মি

Solution:
অ+ঊ = ও, চল+ঊর্মি = চলোর্মি

স্বরধ্বনির সাথে স্বরধ্বনি মিলে যে সন্ধি হয় তাকে স্বরসন্ধি বলে।

স্বরসন্ধি গঠনের নিয়ম:

অ-কার কিংবা আ-কারের পর উ-কার কিংবা উ-কার থাকলে উভয়ে মিলে ও-কার হয়; ও-কার পূর্ববর্তী ব্যঞ্জনের সঙ্গে যুক্ত হয়। যেমন: চলোর্মি = চল+ঊর্মি , নবোঢ়া = নব+উঢ়া, জ্বলোর্ধ্ব = জল+ঊর্ধ্ব।

Question #125 (ID: 275671)
--------------------------------------------------------------------------------
Type: MCQ | Book: বাংলা ২য় পত্র | Years: 2018 | Boards:  | Tags: শিক্ষক নিয়োগ '2021
--------------------------------------------------------------------------------
Question:
সন্ধি বিচ্ছেদ করুন : ‘ক্ষুৎপিপাসা'

Options:
ক. ক্ষুধা + পিপাসা
খ. কোনোটিই নয়।
গ. ক্ষুধ + পিপাসা
ঘ. ক্ষুৎ + পিপাসা


Correct Answer: গ

Solution:
ক্ষুধ + পিপাসা

Explanation:
ব্যাখ্যা : দ্ / ধূ পরে অঘোষ ধ্বনি (ক, খ, চ, ছ, ট, ঠ, ত, থ; প, ফ ) থাকলে, দ/ধু স্থলে অঘোষ অল্পপ্রাণ ধ্বনি হয়। 

■ দ্ + ক(বর্গীয় ১ম ধ্বনি) = ত্ (বর্গীয় ১ম ধ্বনি) + ক

⇒ তদ্ + কাল = তৎকাল, হৃদ্ + কম্প = হৃৎকম্প

■ ধৃ + প (বর্গীয় ১ম) = ত্ (বর্গীয় ১ম ধ্বনি) + প

→ ক্ষুধ + পিপাসা = ক্ষুৎপিপাসা। এরূপ- দৃ + প = ত্ + ত + পর = তৎপর

দৃ + ত = ত্ + ত ⇒ তদ্‌ + ত = তত্ত্ব
