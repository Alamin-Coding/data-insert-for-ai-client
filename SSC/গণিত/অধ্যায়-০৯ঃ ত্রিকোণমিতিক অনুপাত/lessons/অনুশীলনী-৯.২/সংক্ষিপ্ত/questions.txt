Question:
[math]A - B = 30^\circ[/math] এবং [math]\cot A = 1[/math] হলে, [math]B[/math] এর মান কত?

Options:


Correct Answer: 

Solution:
এখানে, [math]A - B = 30^\circ[/math]
 এবং [math]\cot A = 1[/math]
∴ [math]\cot A = \cot 45^\circ[/math]
∴ [math]A = 45^\circ[/math]
আবার, [math]A - B = 30^\circ[/math]
∴ [math]45^\circ - B = 30^\circ[/math]
∴ [math]B = 15^\circ[/math]


Explanation:
ব্যাখ্যা: [math]\cot A=1[/math] হলে [math]\tan A=1[/math] তাই প্রাথমিক কোণটি [math]A=45^\circ[/math]. এরপর [math]A-B=30^\circ[/math] থেকে [math]B=A-30^\circ=45^\circ-30^\circ=15^\circ[/math]; সাধারণভাবে [math]A=45^\circ+180^\circ k[/math] হলে [math]B=15^\circ+180^\circ k[/math], তাই মূল মানটি [math]15^\circ[/math].

Options:


Correct Answer: 

Solution:
দেওয়া আছে, [math]\sin (90^\circ - \theta) = \dfrac{5}{7}[/math]

Question:
[math]\sin (90^\circ - \theta) = \dfrac{\sqrt{3}}{2}[/math] হলে [math]\sec^2 \theta[/math] এর মান কত?

Options:


Correct Answer: 

Solution:
দেওয়া আছে, [math]\sin (90^\circ - \theta) = \dfrac{\sqrt{3}}{2}[/math]
∴ [math]\sin (90^\circ - \theta) = \sin 60^\circ[/math]
∴ [math]90^\circ - \theta = 60^\circ[/math]
∴ [math]\theta = 90^\circ - 60^\circ = 30^\circ[/math]
∴ [math]\sec^2 \theta = \sec^2 30^\circ = \left( \dfrac{2}{\sqrt{3}} \right)^2 = \dfrac{4}{3}[/math]
নির্ধারিত মান: [math]\dfrac{4}{3}[/math]


Explanation:
sin(90^\circ - \theta) = \cos\theta হওয়ায় [math]\cos\theta = \dfrac{\sqrt{3}}{2}[/math]. তাই [math]\sec\theta = \dfrac{1}{\cos\theta} = \dfrac{2}{\sqrt{3}}[/math] এবং [math]\sec^2\theta = \dfrac{4}{3}[/math].

Question:
[math]\sec^2 \theta + \tan^2 \theta = ৩[/math] হলে, [math]\csc \theta[/math] এর মান কত? (যেখানে [math]\theta[/math] সূক্ষ্মকোণ)

Options:


Correct Answer: 

Solution:
দেওয়া আছে, [math]\sec^2 \theta + \tan^2 \theta = 3[/math]
∴ [math]\tan^2 \theta + 1 + \tan^2 \theta = 3[/math]
∴ [math]2 \tan^2 \theta = 3 - 1 = 2[/math]
∴ [math]\tan^2 \theta = \dfrac{2}{2} = 1[/math]
∴ [math]\tan \theta = \sqrt{1} = 1[/math]
∴ [math]\tan \theta = \tan 45^\circ \Rightarrow \theta = 45^\circ[/math]
∴ [math]\csc \theta = \csc 45^\circ = \sqrt{2}[/math]
নির্ধারিত মান: [math]\sqrt{2}[/math]


Explanation:
ব্যাখ্যা: [math]\sec^2\theta=1+\tan^2\theta[/math] হওয়ায় [math]\sec^2\theta+\tan^2\theta=(1+\tan^2\theta)+\tan^2\theta=1+2\tan^2\theta=3[/math], ফলে [math]\tan^2\theta=1[/math] এবং শুচ্মকোণে [math]\tan\theta=1[/math].  
[math]\tan\theta=1[/math] মানে [math]\sin\theta=\cos\theta=\frac{\sqrt2}{2}</math], তাই [math]\csc\theta=\frac{1}{\sin\theta}=\sqrt2[/math].

Correct Answer: 

Solution:
দেওয়া আছে, [math]\sin (90^\circ - \theta) = \dfrac{5}{7}[/math]
∴ [math]\cos \theta = \dfrac{5}{7}[/math]

 [math]= \dfrac{2\sqrt{6} \cdot 7 + 5 \cdot 5}{35} = \dfrac{14\sqrt{6} + 25}{35}[/math]
