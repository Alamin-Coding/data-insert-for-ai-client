Question:
উৎপাদক বিশ্লেষণ কর: [math]x^3 + 6x^2 + 11x + 6[/math]

Options:


Correct Answer: 

Solution:
[math]x^3 + 6x^2 + 11x + 6[/math]
 [math]= x^2(x + 1) + 5x(x + 1) + 6(x + 1)[/math]
 [math]= (x + 1)(x^2 + 5x + 6)[/math]
 [math]= (x + 1)(x^2 + 3x + 2x + 6)[/math]
 [math]= (x + 1)[x(x + 3) + 2(x + 3)][/math]
 [math]= (x + 1)(x + 2)(x + 3)[/math]
সুতরাং, উৎপাদক বিশ্লেষণ: [math](x + 1)(x + 2)(x + 3)[/math]


Explanation:
উৎপাদক: [math](x+1)(x+2)(x+3)[/math]। কারণ ধ্রুবক 6-এর বিভাজক পরীক্ষা করলে [math]x=-1,-2,-3[/math] এ বহুপদী শূন্য হয়, তাই তাদের জন্য সংশ্লিষ্ট গুণফল ফ্যাক্টরগুলো [math](x+1),(x+2),(x+3)[/math] এবং এই গুণফল বিস্তারে আবার মূল বহুপদীটি পাওয়া যায়।

Question:
[math]2p^3 - 3p^2 + 3p - 1[/math] এর একটি উৎপাদক নির্ধারণ কর।

Options:


Correct Answer: 

Solution:
ধরি, [math]f(p) = 2p^3 - 3p^2 + 3p - 1[/math]
 [math]f\left(\dfrac{1}{2}\right) = 2\left(\dfrac{1}{2}\right)^3 - 3\left(\dfrac{1}{2}\right)^2 + 3\left(\dfrac{1}{2}\right) - 1[/math]
 [math]= \dfrac{2}{8} - \dfrac{3}{4} + \dfrac{3}{2} - 1[/math]
 [math]= \dfrac{2 - 6 + 12 - 8}{8} = \dfrac{0}{8} = 0[/math]
যেহেতু, [math]f\left(\dfrac{1}{2}\right) = 0[/math]
,অতএব, [math]p - \dfrac{1}{2} = 0[/math]
বা, [math]2p - 1 = 0[/math]
সুতরাং, [math]2p - 1[/math]
 হলো [math]f(p)[/math]
 এর একটি উৎপাদক।

Explanation:
ব্যাখ্যা: যদি [math]p=\tfrac{1}{2}[/math] ধরে বসানো হয়, তাহলে [math]2p^3-3p^2+3p-1=0[/math], তাই [math]2p-1[/math] একটি উৎপাদক। বিভাজন করলে আমরা পাই [math](2p-1)(2p^2-2p+2)[/math], অতএব [math]2p-1[/math] নিশ্চিতভাবে একটি উৎপাদক।

 [math]f(x)[/math]

f(x)-কে [math](ax+b)[/math] দ্বারা ভাগ করলে থাকবে [math]f(x)=(ax+b)q(x)+r[/math], যেখানে [math]q(x)[/math] ভাগফল ও [math]r[/math] ধ্রুবক ভাগশেষ (কারণ বিভাজকটির মাত্রা 1)। এখন [math]x=-\dfrac{b}{a}[/math] বসালে [math]ax+b=0[/math] হয়ে যায় এবং তাই [math]f\!\left(-\dfrac{b}{a}\right)=r[/math], অর্থাৎ ভাগশেষই [math]f\!\left(-\dfrac{b}{a}\right)[/math]।

Options:


Correct Answer: 

Solution:

 [math]f(x)[/math]
 কে [math]x + 2[/math]
 দ্বারা ভাগ করলে ভাগশেষ হবে [math]f(-2)[/math]

 [math]= 12 + 2 - 16 = 14 - 16 = -2[/math]
সুতরাং, ভাগশেষ = [math]-2[/math]

Options:


Correct Answer: 

Solution:
ধরি, [math]f(x) = 6x^2 - 7x + 5[/math]
 [math]f(x)[/math]
 কে [math]x - 1[/math]
 দ্বারা ভাগ করলে ভাগশেষ হবে [math]f(1)[/math]
এখন,[math]f(1) = 6(1)^2 - 7(1) + 5[/math]
 [math]= 6 - 7 + 5 = 4[/math]
সুতরাং, ভাগশেষ = [math]4[/math]
অতএব, [math]6x^2 - 7x + 5[/math]
 কে [math]x - 1[/math]
 দ্বারা ভাগ করলে ভাগশেষ [math]4[/math]
 হবে। (দেখানো হলো)
