Question:
[math]x^2 + y^2 = 4xy[/math] হলে, প্রমাণ কর যে, [math]\dfrac{x^2}{y^2} + \dfrac{y^2}{x^2} = 14[/math]

Options:


Correct Answer: 

Solution:
এখানে, [math]x^2 + y^2 = 4xy[/math]
বামপক্ষ = [math]\dfrac{x^2}{y^2} + \dfrac{y^2}{x^2}[/math]
\hspace{3em} = [math]\left(\dfrac{x}{y}\right)^2 + \left(\dfrac{y}{x}\right)^2[/math]
\hspace{3em} = [math]\left(\dfrac{x}{y} + \dfrac{y}{x}\right)^2 - 2[/math]
\hspace{3em} = [math]\left(\dfrac{x^2 + y^2}{xy}\right)^2 - 2[/math]
\hspace{3em} = [math]\left(\dfrac{4xy}{xy}\right)^2 - 2[/math]
\hspace{3em} = [math](4)^2 - 2 = 16 - 2 = 14[/math]
∴ [math]\dfrac{x^2}{y^2} + \dfrac{y^2}{x^2} = 14[/math]
 (প্রমাণিত)

Explanation:
প্রথমে [math]y\neq0[/math] ধরে সমীকরণকে [math]y^2[/math] দিয়ে ভাগ করলে [math]\left(\dfrac{x}{y}\right)^2-4\left(\dfrac{x}{y}\right)+1=0[/math] হয়; এখানে [math]t=\dfrac{x}{y}[/math] নিলে [math]t^2-4t+1=0[/math]. এ থেকে [math]t+\dfrac{1}{t}=4[/math] পাওয়া যায়, তাই [math]t^2+\dfrac{1}{t^2}=(t+\dfrac{1}{t})^2-2=16-2=14[/math], অর্থাৎ [math]\dfrac{x^2}{y^2}+\dfrac{y^2}{x^2}=14[/math].

Question:
[math]x = \sqrt{5} + \sqrt{3}[/math] হলে, [math]\dfrac{2}{x}[/math] এর মান নির্ণয় কর।

Options:


Correct Answer: 

Solution:
এখন, [math]x = \sqrt{5} + \sqrt{3}[/math]
∴ [math]\dfrac{2}{x} = \dfrac{2}{\sqrt{5} + \sqrt{3}}[/math]
 [math]\hspace{2em}[/math]
 = [math]\dfrac{2}{\sqrt{5} + \sqrt{3}} \cdot \dfrac{\sqrt{5} - \sqrt{3}}{\sqrt{5} - \sqrt{3}}[/math]
 [math]\hspace{2em}[/math]
 = [math]\dfrac{2(\sqrt{5} - \sqrt{3})}{(\sqrt{5})^2 - (\sqrt{3})^2}[/math]
 [math]\hspace{2em}[/math]
 = [math]\dfrac{2(\sqrt{5} - \sqrt{3})}{5 - 3}[/math]
 [math]\hspace{2em}[/math]
 = [math]\sqrt{5} - \sqrt{3}[/math]
নির্দেশিত মান : [math]\sqrt{5} - \sqrt{3}[/math]


Explanation:
ব্যাখ্যা: [math]\frac{2}{x}=\frac{2}{\sqrt5+\sqrt3}=\frac{2(\sqrt5-\sqrt3)}{(\sqrt5)^2-(\sqrt3)^2}=\frac{2(\sqrt5-\sqrt3)}{5-3}=\sqrt5-\sqrt3[/math]. কনজুগেট [math]\sqrt5-\sqrt3[/math] দ্বারা গুণ করলে ভগ্নাংশের ভগ্নশেষ থেকে.IRRATIONAL অংশ মুছে গিয়ে সরল ফল পাওয়া যায়।

Options:


Correct Answer: 

Solution:
আমরা জানি, [math](a + b)^2 = a^2 + 2ab + b^2[/math]
 .......... (i)\hspace{4em} এবং [math](a - b)^2 = a^2 - 2ab + b^2[/math]
 .......... (ii)(i) নং হতে (ii) নং বিয়োগ করে পাই,[math]\hspace{2em}[/math]
 [math](a + b)^2 - (a - b)^2 = a^2 + 2ab + b^2 - (a^2 - 2ab + b^2)[/math]
 [math]\hspace{2em}[/math]
 বা, [math](a + b)^2 - (a - b)^2 = a^2 + 2ab + b^2 - a^2 + 2ab - b^2[/math]
 [math]\hspace{2em}[/math]
 বা, [math]4ab = (a + b)^2 - (a - b)^2[/math]
 [math]\hspace{2em}[/math]
 বা, [math]ab = \dfrac{(a + b)^2 - (a - b)^2}{4}[/math]
∴ [math]ab = \left(\dfrac{a + b}{2}\right)^2 - \left(\dfrac{a - b}{2}\right)^2[/math]
 (দেখানো হলো)

Correct Answer: 

Solution:
দেওয়া আছে, [math]a + b = \sqrt{7}[/math]
 এবং [math]a - b = \sqrt{3}[/math]

Solution:

∴ [math]\sqrt{a} + \dfrac{1}{\sqrt{a}} = \sqrt{2}[/math]
∴ প্রদত্ত রাশি = [math]\sqrt{2} \cdot \sqrt{2} = 2[/math]

 [math]2x(3x - \dfrac{1}{2x}) = 6x^2 - 1[/math]

[math]x + \dfrac{1}{x} = 4[/math] হলে, প্রমাণ কর যে, [math]\left(x^2 - \dfrac{1}{x^2}\right) = 8\sqrt{3}[/math]

Options:


Correct Answer: 

Solution:
দেওয়া আছে, [math]x + \dfrac{1}{x} = 4[/math]
আমরা জানি, [math]\left(x - \dfrac{1}{x}\right)^2 = \left(x + \dfrac{1}{x}\right)^2 - 4x \cdot \dfrac{1}{x}[/math]
 [math]\hspace{2em}[/math]
 = [math]4^2 - 4 = 16 - 4 = 12[/math]
∴ [math]x - \dfrac{1}{x} = \sqrt{12} = 2\sqrt{3}[/math]
এখন, [math]x^2 - \dfrac{1}{x^2} = (x - \dfrac{1}{x}) (x + \dfrac{1}{x})[/math]
 [math]\hspace{2em}[/math]
 = [math]2\sqrt{3} \cdot 4 = 8\sqrt{3}[/math]
∴ [math]\left(x^2 - \dfrac{1}{x^2}\right) = 8\sqrt{3}[/math]
 (প্রমাণিত)

Explanation:
প্রথমে উভয় পাশে বর্গ করলে পাই [math]x^2+2+\dfrac{1}{x^2}=16[/math], তাই [math]x^2+\dfrac{1}{x^2}=14[/math]। ফলে [math]\left(x^2-\dfrac{1}{x^2}\right)^2=(x^2+\dfrac{1}{x^2})^2-4=14^2-4=192[/math], অর্থাৎ পরিমাপ [math]8\sqrt{3}[/math]। এছাড়া [math]x^2-4x+1=0[/math] থেকে [math]x=2\pm\sqrt{3}[/math] এবং বড় মূলটি [math]2+\sqrt{3}>1[/math] হলে প্রকাশটি ধনাত্মক, তাই [math]x^2-\dfrac{1}{x^2}=8\sqrt{3}[/math]।

Correct Answer: 

Solution:
এখানে, [math]a + \dfrac{1}{a} = 2[/math]
∴ [math]\dfrac{a^2 + 1}{a} = 2[/math]
∴ [math]a^2 - 2a + 1 = 0[/math]
∴ [math](a - 1)^2 = 0[/math]
∴ [math]a = 1[/math]
∴ [math]a^3 + \dfrac{1}{a^3} = 1^3 + \dfrac{1}{1^3} = 1 + 1 = 2[/math]
∴ [math]a^3 + \dfrac{1}{a^3} = 2[/math]
 (প্রমাণিত)

Explanation:
ব্যাখ্যা: [math](a+\frac{1}{a})^3 = a^3 + \frac{1}{a^3} + 3\bigl(a+\frac{1}{a}\bigr)[/math]. যদি [math]a+\frac{1}{a}=2[/math] হয়, তাহলে [math]a^3+\frac{1}{a^3}=(a+\frac{1}{a})^3-3\bigl(a+\frac{1}{a}\bigr)=2^3-3\cdot2=8-6=2[/math]. বিকল্পভাবে সমীকরণ থেকে [math]a=1[/math] পাওয়া যায় এবং তাতেও [math]1^3+\frac{1}{1^3}=1+1=2[/math]।
