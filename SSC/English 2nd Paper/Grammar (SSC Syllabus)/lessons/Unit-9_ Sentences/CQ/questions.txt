Question:
Make six sentences using the parts of sentences from each column of the table below:

$\begin{array}{|l|l|l|}\hline \text { (a) Wildlife } & \text { (i) is } & \text { (i) on wildlife. } \\\hline \text { (b) Balance of ecosystem largely } & \text { (ii) refers } & \text { (ii) harmful to wildlife. } \\\hline \text { (c) Any development } & \text { (iii) depends } & \begin{array}{l}\text { (iii) to all animals, plants } \\\text { found in the forests. }\end{array} \\\hline \begin{array}{l}\text { (d) Urbanization and } \\\text { industrialization }\end{array} & \text { (iv) should take } & \text { (iv) be extinct. } \\\hline \text { (e) As a result, many animals } & \text { (v) have also increased } & \begin{array}{l}\text { (v) united efforts to protect } \\\text { wildlife. }\end{array} \\\hline \text { (f) For our own existence, we } & \text { (vi) are going to } & \begin{array}{l}\text { (vi) deforestation to a great } \\\text { extent. }\end{array} \\\hline\end{array}$

Options:


Correct Answer: 

Solution:
Matching:

(a+ii+iii) = Wildlife refers to all animals, plants found in the forests. (বন্যপ্রাণী বলতে বনে পাওয়া সমস্ত প্রাণী, উদ্ভিদকে বোঝায়।)

(b+iii+i) = Balance of ecosystem largely depends on wildlife. (বাস্তুতন্ত্রের ভারসাম্য অনেকাংশে বন্যপ্রাণীর উপর নির্ভর করে।)

(c+i+ii) = Any development is harmful to wildlife. (যেকোনো উন্নয়ন বন্যপ্রাণীর জন্য ক্ষতিকর।)

(d+v+vi) = Urbanization and industrialization have also increased deforestation to a great extent. (নগরায়ণ এবং শিল্পায়নও বন উজাড়কে ব্যাপক মাত্রায় বাড়িয়ে দিয়েছে।)

(e+vi+iv) = As a result, many animals are going to be extinct. (ফলস্বরূপ, অনেক প্রাণী বিলুপ্ত হতে চলেছে।)

(f+iv+v) = For our own existence, we should take united efforts to protect wildlife. (আমাদের নিজেদের অস্তিত্বের জন্য, বন্যপ্রাণী রক্ষার জন্য আমাদের সম্মিলিত প্রচেষ্টা নেওয়া উচিত।)

Explanation:
Explanation: Each sentence is made by combining a subject from the left column, an appropriate verb from the middle column, and a complement from the right column so that both grammar and meaning fit (subject–verb agreement and correct verb collocation). For example, "Wildlife refers to all animals, plants found in the forests" uses "refers to" to define the subject; "Balance of ecosystem largely depends on wildlife" uses "depends on" with "on wildlife"; and "Urbanization and industrialization have also increased deforestation to a great extent" uses a plural subject with "have also increased" taking "deforestation…" as its object. This matching ensures each sentence is grammatically correct and semantically logical.

Question #3 (ID: 481774)
--------------------------------------------------------------------------------
Type: CQ | Book: English 2nd Paper | Years: 2023 | Boards: Rajshahi Board | Tags: Rajshahi Board
--------------------------------------------------------------------------------
Question:
Change the sentences according to the directions:

(a) Newspaper is one of the most useful sources of knowledge. (Positive)

(b) It carries both local and international news.(Negative)

(c) It reflects the whole world like a mirror. (Interrogative)

(d) We can not help reading newspaper daily. (Affirmative)

(e) It contains daily news. (Passive)

(f) We are helped in many ways by it. (Active)

(g) By reading newspaper, we can enrich our knowledge. (Negative)

(h) It is called the people's parliament. (Active)

(i) Nothing is so important as it. ( Comparative)

(j) We should read newspaper to know the outside world more. (Imperative)

Options:


Correct Answer: 

Solution:
(a) The newspaper is unquestionably one of the most valuable sources of knowledge.

(b) Unfortunately, it lacks both local and international news.

(c) Does it mirror the entire world like a reflection?

(d) Reading the newspaper daily is unavoidable. Daily news is contained within.

(f) It actively aids us in numerous ways.

(g) Enriching our knowledge can be achieved by reading the newspaper.

(h) People fondly refer to it as the people's parliament.

(i) There is nothing more important than it.

(j) We should read the newspaper to gain a better understanding of the outside world.

Explanation:
(a) Newspaper is a useful source of knowledge.
Explanation: Changed the superlative “one of the most useful” to the positive degree by using “a useful” to state a general quality.

(b) It does not carry only local news; it also carries international news.
Explanation: Made the sentence negative by adding “does not” and clarified the contrast with “also” to keep the original meaning.

(c) Does it reflect the whole world like a mirror?
Explanation: Turned the statement into a yes/no question by using the auxiliary “does” and inverting subject and verb.

(d) We read the newspaper daily.
Explanation: Removed the negative structure “cannot help” and used a simple affirmative present tense to express the habitual action.

(e) Daily news is contained in it.
Explanation: Formed the passive by making the object “daily news” the subject and using the passive verb “is contained.”

(f) It helps us in many ways.
Explanation: Converted the passive “are helped in many ways by it” to active with “it” as subject and “helps” as the verb.

(g) We cannot enrich our knowledge by reading the newspaper.
Explanation: Made the sentence negative by changing “can” to “cannot,” which negates the possibility expressed originally.

(h) People call it the people's parliament.
Explanation: Changed the passive “is called” to an active form with the agent “People” and the verb “call.”

(i) It is more important than anything.
Explanation: Converted the original “Nothing is so important as it” into a comparative form using “more... than.”

(j) Read the newspaper to know the outside world better.
Explanation: Turned the sentence into an imperative by using the base verb “Read,” implying the subject “you,” and used “better” for clearer comparative meaning.
