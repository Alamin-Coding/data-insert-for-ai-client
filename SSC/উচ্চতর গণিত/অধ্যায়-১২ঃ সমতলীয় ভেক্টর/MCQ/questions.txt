Question:

Options:
A. [math]2\overrightarrow{AB}[/math]

C. [math]0[/math]
D. 2


Correct Answer: গ

Solution:
[math]\overrightarrow{AB}[/math]

 হলো B থেকে A বিন্দুর দিকে নির্দেশিত একটি ভেক্টর। [math]\overrightarrow{BA} = -\overrightarrow{AB}[/math]
। সুতরাং, [math]\overrightarrow{AB} + \overrightarrow{BA} = \overrightarrow{AB} + (-\overrightarrow{AB}) = \overrightarrow{AB} - \overrightarrow{AB} = 0[/math]
।

Question:
A, B, C এর অবস্থান ভেক্টর যথাক্রমে a, b ও c এবং C, AB কে 6 : 11 অনুপাতে অন্তর্বিভক্ত করলে c = ?

Options:
ক. [math]\frac{6b+11a}{17}[/math]
খ. [math]\frac{11b+6a}{17}[/math]
গ. [math]\frac{6b-11a}{17}[/math]
ঘ. [math]\frac{11b-6a}{17}[/math]


Correct Answer: ক

Solution:
C বিন্দুটি AB রেখাকে 6 : 11 অনুপাতে অন্তর্বিভক্ত করে। এর মানে হলো, AC : CB = 6 : 11।একটি রেখাকে নির্দিষ্ট অনুপাতে অন্তর্বিভক্ত করার সূত্র অনুযায়ী, যদি P বিন্দুটি AB রেখাকে m : n অনুপাতে অন্তর্বিভক্ত করে, তাহলে P এর অবস্থান ভেক্টর হবে [math]\frac{n\vec{a} + m\vec{b}}{m+n}[/math]
।এখানে, C বিন্দুটি AB রেখাকে 6 : 11 অনুপাতে অন্তর্বিভক্ত করেছে। A এর অবস্থান ভেক্টর [math]\vec{a}[/math]
 এবং B এর অবস্থান ভেক্টর [math]\vec{b}[/math]
। C এর অবস্থান ভেক্টর [math]\vec{c}[/math]
।অনুপাত m = 6 এবং n = 11।সুতরাং, C এর অবস্থান ভেক্টর [math]\vec{c}[/math]
 হবে:[math]\vec{c} = \frac{11\vec{a} + 6\vec{b}}{6+11}[/math]
 [math]\vec{c} = \frac{11\vec{a} + 6\vec{b}}{17}[/math]
এখানে প্রশ্নে প্রদত্ত অপশনগুলো ভিন্নভাবে লেখা হয়েছে। অপশন 1 হলো [math]\frac{6b+11a}{17}[/math]
। এটি আমাদের প্রাপ্ত [math]\frac{11\vec{a} + 6\vec{b}}{17}[/math]
 এর সমতুল্য।তাই, সঠিক উত্তর হলো 1।

Question:
AB সরলরেখাকে C বিন্দুটি 2 : 3 ভাগে অন্তর্বিভক্ত করলে C বিন্দুর অবস্থান ভেক্টর c কে নিচের কোনটি দ্বারা নির্দেশ করা যাবে যেখানে, মূল বিন্দুর সাপেক্ষে A, B ও C বিন্দুর অবস্থান ভেক্টর যথাক্রমে a, b ও c?

Options:
ক. [math]\frac{2a+3b}{6}[/math]
খ. [math]\frac{2a+3b}{5}[/math]
গ. [math]\frac{3a+2b}{6}[/math]
ঘ. [math]\frac{3a+2b}{5}[/math]


Correct Answer: ঘ

Solution:
AB সরলরেখাকে C বিন্দুটি 2 : 3 ভাগে অন্তর্বিভক্ত করে। এর অর্থ হলো, AC : CB = 2 : 3।যদি মূল বিন্দুর সাপেক্ষে A বিন্দুর অবস্থান ভেক্টর [math]a[/math]
 এবং B বিন্দুর অবস্থান ভেক্টর [math]b[/math]
 হয়, তবে C বিন্দুর অবস্থান ভেক্টর [math]c[/math]
 নির্ণয় করার সূত্রটি হলো:[math]c = \frac{m \times b + n \times a}{m+n}[/math]
এখানে, [math]m[/math]
 হলো C বিন্দু থেকে B বিন্দুর দূরত্ব অংশের অনুপাত, যা এখানে 3। আর [math]n[/math]
 হলো C বিন্দু থেকে A বিন্দুর দূরত্ব অংশের অনুপাত, যা এখানে 2।সুতরাং, [math]m=3[/math]
 এবং [math]n=2[/math]
 বসিয়ে পাই:[math]c = \frac{3 \times b + 2 \times a}{2+3}[/math]
 [math]c = \frac{3b + 2a}{5}[/math]
 [math]c = \frac{2a + 3b}{5}[/math]
এখানে একটি ভুল হয়েছে। প্রশ্ন অনুযায়ী C বিন্দুটি AB সরলরেখাকে 2:3 অনুপাতে অন্তর্বিভক্ত করে। এর মানে হলো, C বিন্দুটি A থেকে 2 ভাগ এবং B থেকে 3 ভাগ দূরে অবস্থিত।সুতরাং, C বিন্দুর অবস্থান ভেক্টর [math]c[/math]
 হবে:[math]c = \frac{3 \times a + 2 \times b}{2+3}[/math]
 [math]c = \frac{3a + 2b}{5}[/math]

Question:
নিচের কোনটি দ্বারা [math]\overrightarrow{NO}[/math] নির্দেশ করে?

Options:
ক. [math]\overrightarrow{MN}-\overrightarrow{MO}[/math]
খ. [math]\overrightarrow{MO}-\overrightarrow{NM}[/math]
গ. [math]\overrightarrow{MO}+\overrightarrow{NM}[/math]
ঘ. [math]\overrightarrow{MN}+\overrightarrow{MO}[/math]


Correct Answer: গ

Solution:
[math]\overrightarrow{NO}[/math]
 নির্দেশ করে [math]\overrightarrow{NM}+\overrightarrow{MO}[/math]
 ।কারণ, ভেক্টর যোগের ত্রিভুজ সূত্র অনুযায়ী, যদি তিনটি ভেক্টর একটি ত্রিভুজের তিনটি বাহুকে একই ক্রমে নির্দেশ করে, তবে তাদের সমষ্টি শূন্য হয়। এখানে, [math]\overrightarrow{MN}+\overrightarrow{NO}+\overrightarrow{OM} = \vec{0}[/math]
 ।আমরা জানি, [math]\overrightarrow{OM} = -\overrightarrow{MO}[/math]
 ।সুতরাং, [math]\overrightarrow{MN}+\overrightarrow{NO}-\overrightarrow{MO} = \vec{0}[/math]
 ।বা, [math]\overrightarrow{NO} = \overrightarrow{MO} - \overrightarrow{MN}[/math]
 ।আবার, [math]\overrightarrow{MO} - \overrightarrow{MN} = \overrightarrow{MO} + (-\overrightarrow{MN}) = \overrightarrow{MO} + \overrightarrow{NM}[/math]
 ।অতএব, [math]\overrightarrow{NO} = \overrightarrow{MO} + \overrightarrow{NM}[/math]
 ।

ক. [math]a-8b[/math]
খ. [math]8b-a[/math]
গ. [math]a+8b[/math]
ঘ. [math]a+b[/math]


Correct Answer: ক

Solution:
P বিন্দুর অবস্থান ভেক্টর হলো [math]2a + 3b[/math]
 এবং Q বিন্দুর অবস্থান ভেক্টর হলো [math]3a - 5b[/math]
।[math]\overrightarrow{PQ}[/math]
 হলো P থেকে Q পর্যন্ত ভেক্টর। এটি নির্ণয় করতে Q বিন্দুর অবস্থান ভেক্টর থেকে P বিন্দুর অবস্থান ভেক্টর বিয়োগ করতে হবে।[math]\overrightarrow{PQ}[/math]
 = (Q বিন্দুর অবস্থান ভেক্টর) - (P বিন্দুর অবস্থান ভেক্টর)[math]\overrightarrow{PQ}[/math]
 = [math](3a - 5b) - (2a + 3b)[/math]
 [math]\overrightarrow{PQ}[/math]
 = [math]3a - 5b - 2a - 3b[/math]
 [math]\overrightarrow{PQ}[/math]
 = [math](3a - 2a) + (-5b - 3b)[/math]
 [math]\overrightarrow{PQ}[/math]
 = [math]a - 8b[/math]
সুতরাং, সঠিক উত্তরটি হলো [math]a-8b[/math]

Question:
নিচের কোনটি দ্বারা [math]\overrightarrow{NO}[/math] নির্দেশ করে?

Options:
ক. [math]\overrightarrow{MN}-\overrightarrow{MO}[/math]
খ. [math]\overrightarrow{MO}-\overrightarrow{NM}[/math]
গ. [math]\overrightarrow{MO}+\overrightarrow{NM}[/math]
ঘ. [math]\overrightarrow{MN}+\overrightarrow{MO}[/math]


Correct Answer: গ

Solution:
[math]\overrightarrow{NO}[/math]
 নির্দেশ করে [math]\overrightarrow{NM}+\overrightarrow{MO}[/math]
 ।কারণ, ভেক্টর যোগের ত্রিভুজ সূত্র অনুযায়ী, যদি তিনটি ভেক্টর একটি ত্রিভুজের তিনটি বাহুকে একই ক্রমে নির্দেশ করে, তবে তাদের সমষ্টি শূন্য হয়। এখানে, [math]\overrightarrow{MN}+\overrightarrow{NO}+\overrightarrow{OM} = \vec{0}[/math]
 ।আমরা জানি, [math]\overrightarrow{OM} = -\overrightarrow{MO}[/math]
 ।সুতরাং, [math]\overrightarrow{MN}+\overrightarrow{NO}-\overrightarrow{MO} = \vec{0}[/math]
 ।বা, [math]\overrightarrow{NO} = \overrightarrow{MO} - \overrightarrow{MN}[/math]
 ।আবার, [math]\overrightarrow{MO} - \overrightarrow{MN} = \overrightarrow{MO} + (-\overrightarrow{MN}) = \overrightarrow{MO} + \overrightarrow{NM}[/math]
 ।অতএব, [math]\overrightarrow{NO} = \overrightarrow{MO} + \overrightarrow{NM}[/math]
 ।

Question:

খ. [math]\frac{11b+6a}{17}[/math]
গ. [math]\frac{6b-11a}{17}[/math]
ঘ. [math]\frac{11b-6a}{17}[/math]


Correct Answer: ক

Solution:
C বিন্দুটি AB রেখাকে 6 : 11 অনুপাতে অন্তর্বিভক্ত করে। এর মানে হলো, AC : CB = 6 : 11।একটি রেখাকে নির্দিষ্ট অনুপাতে অন্তর্বিভক্ত করার সূত্র অনুযায়ী, যদি P বিন্দুটি AB রেখাকে m : n অনুপাতে অন্তর্বিভক্ত করে, তাহলে P এর অবস্থান ভেক্টর হবে [math]\frac{n\vec{a} + m\vec{b}}{m+n}[/math]
।এখানে, C বিন্দুটি AB রেখাকে 6 : 11 অনুপাতে অন্তর্বিভক্ত করেছে। A এর অবস্থান ভেক্টর [math]\vec{a}[/math]
 এবং B এর অবস্থান ভেক্টর [math]\vec{b}[/math]
। C এর অবস্থান ভেক্টর [math]\vec{c}[/math]
।অনুপাত m = 6 এবং n = 11।সুতরাং, C এর অবস্থান ভেক্টর [math]\vec{c}[/math]
 হবে:[math]\vec{c} = \frac{11\vec{a} + 6\vec{b}}{6+11}[/math]
 [math]\vec{c} = \frac{11\vec{a} + 6\vec{b}}{17}[/math]
এখানে প্রশ্নে প্রদত্ত অপশনগুলো ভিন্নভাবে লেখা হয়েছে। অপশন 1 হলো [math]\frac{6b+11a}{17}[/math]
। এটি আমাদের প্রাপ্ত [math]\frac{11\vec{a} + 6\vec{b}}{17}[/math]
 এর সমতুল্য।তাই, সঠিক উত্তর হলো 1।

Options:
ক. [math]a-8b[/math]
খ. [math]8b-a[/math]
গ. [math]a+8b[/math]
ঘ. [math]a+b[/math]


Correct Answer: ক

Solution:
P বিন্দুর অবস্থান ভেক্টর হলো [math]2a + 3b[/math]
 এবং Q বিন্দুর অবস্থান ভেক্টর হলো [math]3a - 5b[/math]
।[math]\overrightarrow{PQ}[/math]
 হলো P থেকে Q পর্যন্ত ভেক্টর। এটি নির্ণয় করতে Q বিন্দুর অবস্থান ভেক্টর থেকে P বিন্দুর অবস্থান ভেক্টর বিয়োগ করতে হবে।[math]\overrightarrow{PQ}[/math]
 = (Q বিন্দুর অবস্থান ভেক্টর) - (P বিন্দুর অবস্থান ভেক্টর)[math]\overrightarrow{PQ}[/math]
 = [math](3a - 5b) - (2a + 3b)[/math]
 [math]\overrightarrow{PQ}[/math]
 = [math]3a - 5b - 2a - 3b[/math]
 [math]\overrightarrow{PQ}[/math]
 = [math](3a - 2a) + (-5b - 3b)[/math]
 [math]\overrightarrow{PQ}[/math]
 = [math]a - 8b[/math]
সুতরাং, সঠিক উত্তরটি হলো [math]a-8b[/math]
।

Question:
AB সরলরেখাকে C বিন্দুটি 2 : 3 ভাগে অন্তর্বিভক্ত করলে C বিন্দুর অবস্থান ভেক্টর c কে নিচের কোনটি দ্বারা নির্দেশ করা যাবে যেখানে, মূল বিন্দুর সাপেক্ষে A, B ও C বিন্দুর অবস্থান ভেক্টর যথাক্রমে a, b ও c?

Options:
ক. [math]\frac{2a+3b}{6}[/math]
খ. [math]\frac{2a+3b}{5}[/math]
গ. [math]\frac{3a+2b}{6}[/math]
ঘ. [math]\frac{3a+2b}{5}[/math]


Correct Answer: ঘ

Solution:
AB সরলরেখাকে C বিন্দুটি 2 : 3 ভাগে অন্তর্বিভক্ত করে। এর অর্থ হলো, AC : CB = 2 : 3।যদি মূল বিন্দুর সাপেক্ষে A বিন্দুর অবস্থান ভেক্টর [math]a[/math]
 এবং B বিন্দুর অবস্থান ভেক্টর [math]b[/math]
 হয়, তবে C বিন্দুর অবস্থান ভেক্টর [math]c[/math]
 নির্ণয় করার সূত্রটি হলো:[math]c = \frac{m \times b + n \times a}{m+n}[/math]
এখানে, [math]m[/math]
 হলো C বিন্দু থেকে B বিন্দুর দূরত্ব অংশের অনুপাত, যা এখানে 3। আর [math]n[/math]
 হলো C বিন্দু থেকে A বিন্দুর দূরত্ব অংশের অনুপাত, যা এখানে 2।সুতরাং, [math]m=3[/math]
 এবং [math]n=2[/math]
 বসিয়ে পাই:[math]c = \frac{3 \times b + 2 \times a}{2+3}[/math]
 [math]c = \frac{3b + 2a}{5}[/math]
 [math]c = \frac{2a + 3b}{5}[/math]
এখানে একটি ভুল হয়েছে। প্রশ্ন অনুযায়ী C বিন্দুটি AB সরলরেখাকে 2:3 অনুপাতে অন্তর্বিভক্ত করে। এর মানে হলো, C বিন্দুটি A থেকে 2 ভাগ এবং B থেকে 3 ভাগ দূরে অবস্থিত।সুতরাং, C বিন্দুর অবস্থান ভেক্টর [math]c[/math]
 হবে:[math]c = \frac{3 \times a + 2 \times b}{2+3}[/math]
 [math]c = \frac{3a + 2b}{5}[/math]

A, B ও C বিন্দুর অবস্থান ভেক্টর যথাক্রমে a, b, c এবং C বিন্দুতে AB রেখাংশ 1:2 অনুপাতে অন্তর্বিভক্ত হলে সঠিক সম্পর্কটি হলো c = (2a + b) / 3 বা c = (2a + b) / (1+2).এই সূত্রটি ভেতরের বিভাজন সূত্র থেকে আসে, যেখানে C বিন্দুর অবস্থান ভেক্টর (c) হলো A ও B বিন্দুর অবস্থান ভেক্টরের (a ও b) একটি মিশ্রণ, যা বিভাজন অনুপাতের উপর ভিত্তি করে নির্ধারিত হয়.

Explanation:
a: A বিন্দুর অবস্থান ভেক্টর।b: B বিন্দুর অবস্থান ভেক্টর।c: C বিন্দুর অবস্থান ভেক্টর।m:n: অন্তর্বিভক্ত অনুপাত (এই ক্ষেত্রে m=1 এবং n=2).সূত্র প্রয়োগঅন্তর্বিভাজন সূত্র অনুযায়ী, C বিন্দুর অবস্থান ভেক্টর c এর সূত্র হলো:Codec = (ma + nb) / (m + n)এখানে, m = 1 এবং n = 2 বসিয়ে পাই:c = (1*a + 2*b) / (1 + 2)c = (a + 2b) / 3সুতরাং, সঠিক সম্পর্কটি হলো c = (a + 2b) / 3 বা c = (2a + b) / 3 (যেহেতু a এবং b এর সাথে 2 এবং 1 এর সম্পর্ক পরিবর্তন করা যায় কিন্তু চূড়ান্ত ফলাফল একই থাকে).
