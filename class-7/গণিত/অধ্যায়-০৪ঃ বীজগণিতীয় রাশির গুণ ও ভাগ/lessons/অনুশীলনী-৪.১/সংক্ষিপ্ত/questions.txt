Question:
[math](x^2 + 2x + 1)^2[/math] কে [math](x - 1)^2[/math] দ্বারা গুণ কর।

Options:


Correct Answer: 

Solution:
প্রথমে [math](x^2 + 2x + 1)^2 = x^4 + 4x^3 + 6x^2 + 4x + 1[/math]
তারপর [math](x^4 + 4x^3 + 6x^2 + 4x + 1)(x^2 - 2x + 1)[/math]
 [math]= x^6 + 2x^5 - x^4 - 4x^5 - 8x^4 + 4x^3 + 6x^4 + 12x^3 - 6x^2 + 4x^3 + 8x^2 - 4x + x^2 - 2x + 1[/math]
সব সংযোজন করে: [math]= x^6 - 2x^5 - 3x^4 + 20x^3 + 3x^2 - 6x + 1[/math]


Explanation:
প্রথমত, [math]x^2+2x+1=(x+1)^2[/math] হওয়ায় গুণফলটি হয় [math](x+1)^4(x-1)^2[/math]. প্রয়োজনে এটিকে আবার লিখতে পারি যেমন [math][(x+1)^2(x-1)]^2[/math], যা একইই ফল।

Question:
[math](a^2 + b^2)[/math] কে [math]a^3b^4[/math] দ্বারা গুণ কর।

Options:


Correct Answer: 

Solution:
[math](a^2 + b^2) \times a^3b^4[/math]
 [math]= a^2 \times a^3b^4 + b^2 \times a^3b^4[/math]
 [math]= a^{2+3}b^4 + a^3b^{2+4}[/math]
 [math]= a^5b^4 + a^3b^6[/math]


Explanation:
বিস্তরণ নিয়ম অনুযায়ী প্রতিটি পদকে [math]a^3b^4[/math] দ্বারা গুণ করলে এবং ঘাতের নিয়মে একই ভিত্তির ঘাতগুলো যোগ করলে ফল হয়: [math](a^2+b^2)a^3b^4=a^2\cdot a^3b^4+b^2\cdot a^3b^4=a^{5}b^{4}+a^{3}b^{6}[/math].

Question:
[math]A = x^2 + xy + y^2[/math] এবং [math]B = x - y[/math] হলে, প্রমাণ কর যে [math]AB = x^3 - y^3[/math]

Options:


Correct Answer: 

Solution:
দেওয়া আছে,[math]A = x^2 + xy + y^2[/math]
, [math]B = x - y[/math]
সুতরাং,[math]AB = (x^2 + xy + y^2)(x - y)[/math]
এখন গুণ করি:[math]= x^2(x - y) + xy(x - y) + y^2(x - y)[/math]
 [math]= x^3 - x^2y + x^2y - xy^2 + xy^2 - y^3[/math]
মধ্যবর্তী পদগুলো কেটে যায়:[math]= x^3 - y^3[/math]
প্রমাণিত: [math]AB = x^3 - y^3[/math]


Explanation:
কারণ [math]x^3 - y^3 = (x - y)(x^2 + xy + y^2)[/math], তাই [math]A = x^2+xy+y^2[/math] এবং [math]B = x-y[/math] হলে তাদের গুণফল হলো [math]AB=(x^2+xy+y^2)(x-y)=x^3 - x^2y + x^2y - xy^2 + xy^2 - y^3 = x^3 - y^3[/math], যেখানে মধ্যবর্তী পদসমূহ কেটে যায়।

 [math]= 2x^2 + 6x - 5x - 15[/math]
 [math]= 2x^2 + x - 15[/math]


Explanation:
বহুগুণকের নিয়ম অনুযায়ী প্রতিটি পদকে গুণ করলে পাই [math](2x-5)(x+3)=2x^2+6x-5x-15[/math]. একই ধরনের পদ যোগ করলে ফল হবে [math]2x^2+x-15[/math].
