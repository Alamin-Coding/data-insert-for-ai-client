Question:
[math]x^4 + x^3y + x^2y^2 + xy^3 + y^4[/math] কে [math]x^2 - xy + y^2[/math] দ্বারা ভাগ কর

Options:


Correct Answer: 

Solution:
ধাপে ধাপে ভাগ করা হয়েছে:[math]x^4 + x^3y + x^2y^2 + xy^3 + y^4[/math]
 কে [math]x^2 - xy + y^2[/math]
 দ্বারা ভাগ করতে হবে।প্রথম ধাপে:[math]x^2 \times (x^2 - xy + y^2) = x^4 - x^3y + x^2y^2[/math]
বিয়োগ করলে পাই:[math](x^4 + x^3y + x^2y^2) - (x^4 - x^3y + x^2y^2) = 2x^3y[/math]
দ্বিতীয় ধাপে:[math]+xy \times (x^2 - xy + y^2) = x^3y - x^2y^2 + xy^3[/math]
বিয়োগ করলে পাই:[math]2x^3y - (x^3y - x^2y^2 + xy^3) = x^3y + x^2y^2 - xy^3[/math]
তৃতীয় ধাপে:[math]+y^2 \times (x^2 - xy + y^2) = x^2y^2 - x y^3 + y^4[/math]
বিয়োগ করলে সব পদ কেটে যায়, অবশিষ্ট থাকে [math]0[/math]
নির্ণীত ভাগফল: [math]x^2 + xy + y^2[/math]


Explanation:
ব্যাখ্যা: y-কে ধ্রুব ধরে x-এ দীর্ঘ ভাগ করলে শীর্ষ পদ থেকে ক্রমান্বয়ে ভাগ করলে পাওয়া যায় ভাগফল [math]x^2+2xy+2y^2[/math] এবং অবশিষ্ট অংশ (remainder) [math]xy^3-y^4[/math]. যাচাই: [math](x^2-xy+y^2)(x^2+2xy+2y^2)+(xy^3-y^4)=x^4+x^3y+x^2y^2+xy^3+y^4[/math], সুতরাং উত্তর সঠিক।

Question:
[math]a^5 + 11a - 12[/math] কে [math]a^2 - 2a + 3[/math] দ্বারা ভাগ কর

Options:


Correct Answer: 

Solution:
ধাপে ধাপে ভাগ করা হয়েছে:প্রথম ধাপে:[math]a^3 \times (a^2 - 2a + 3) = a^5 - 2a^4 + 3a^3[/math]
বিয়োগ করলে পাই:[math](a^5 + 0a^4 + 0a^3 + 0a^2 + 11a - 12) - (a^5 - 2a^4 + 3a^3) = 2a^4 - 3a^3[/math]
দ্বিতীয় ধাপে:[math]+2a \times (a^2 - 2a + 3) = 2a^3 - 4a^2 + 6a[/math]
বিয়োগ করলে পাই:[math]-3a^3 - (2a^3 - 4a^2 + 6a) = -5a^3 + 4a^2 - 6a[/math]
তৃতীয় ধাপে:[math]+1 \times (a^2 - 2a + 3) = a^2 - 2a + 3[/math]
শেষে বিয়োগ করলে অবশিষ্ট থাকে [math]0[/math]
নির্ণীত ভাগফল: [math]a^3 + 2a + 1[/math]


Explanation:
ভাগ করলে ভাগফল হয় [math]a^3+2a^2+a-4[/math] এবং অবশিষ্টাংশ শূন্য। কারণ [math](a^2-2a+3)(a^3+2a^2+a-4)=a^5+11a-12[/math], তাই পূর্ণ ভাগ সম্ভব।

Question:
[math]30x^6y^8 - 45x^4y^7[/math] কে [math]3x^2y^4[/math] দ্বারা ভাগ কর।

Options:


Correct Answer: 

Solution:
[math]\dfrac{30x^6y^8 - 45x^4y^7}{3x^2y^4} = \dfrac{30x^6y^8}{3x^2y^4} - \dfrac{45x^4y^7}{3x^2y^4}[/math]
 [math]= 10x^{6-2}y^{8-4} - 15x^{4-2}y^{7-4}[/math]
 [math]= 10x^4y^4 - 15x^2y^3[/math]


Explanation:
প্রতিটি পদকে আলাদাভাবে [math]3x^2y^4[/math] দ্বারা ভাগ করলে প্রথম পদ থেকে পাওয়া যায় [math]30/3=10,\;x^{6-2}=x^4,\;y^{8-4}=y^4[/math] অর্থাৎ [math]10x^4y^4[/math], এবং দ্বিতীয় পদ থেকে পাওয়া যায় [math]-45/3=-15,\;x^{4-2}=x^2,\;y^{7-4}=y^3[/math] অর্থাৎ [math]-15x^2y^3[/math]. তাই ফলাফল হচ্ছে [math]\frac{30x^6y^8-45x^4y^7}{3x^2y^4}=10x^4y^4-15x^2y^3[/math].

 [math]= 3x^2[/math]
 দিয়ে গুণ: [math]3x^2 \times (3x - 2) = 9x^3 - 6x^2[/math]
বিয়োগ: [math]0 - 20x + 12[/math]
 [math]-20x \div 3x = -\dfrac{20}{3}[/math]
... (অসমানভাবে বিভাজ্য, ভাগশেষ থাকবে)।ভাগফল: [math]3x^2[/math]

Explanation:

Question:
[math]-125x^{10}y^7z^9[/math] কে [math]-25x^5y^3z^4[/math] দ্বারা ভাগ কর।

Options:


Correct Answer: 

Solution:
[math]\dfrac{-125x^{10}y^7z^9}{-25x^5y^3z^4}[/math]
 [math]= \dfrac{-125}{-25} \times \dfrac{x^{10}}{x^5} \times \dfrac{y^7}{y^3} \times \dfrac{z^9}{z^4}[/math]
 [math]= 5x^5y^4z^5[/math]


Explanation:
ব্যাখ্যা: সহগগুলো ভাগ করলে [math]\frac{-125}{-25}=5[/math] এবং ঘাতগুলোর নিয়ম অনুযায়ী ঘাতগুলো বিয়োগ করলে [math]x^{10-5}=x^5,\; y^{7-3}=y^4,\; z^{9-4}=z^5[/math]. তাই ভাগফল হলো [math]5x^5y^4z^5[/math].

Question:
[math]144m^9n^8p^6[/math] কে [math]-12m^3n^2p^3[/math] দ্বারা ভাগ কর।

Options:


Correct Answer: 

Solution:
[math]\dfrac{144m^9n^8p^6}{-12m^3n^2p^3}[/math]
 [math]= \dfrac{144}{-12} \times \dfrac{m^9}{m^3} \times \dfrac{n^8}{n^2} \times \dfrac{p^6}{p^3}[/math]
 [math]= -12m^6n^6p^3[/math]


Explanation:
বিভাজনের ফল: [math]144m^9n^8p^6 \div (-12m^3n^2p^3)= -12m^6n^6p^3[/math]. কারণ সহগগুলো ভাগ করলে [math]144\div(-12)=-12[/math] এবং ঘাতগুলো ভাগের সময় বিয়োগ হয়: [math]m^{9-3}=m^6,\ n^{8-2}=n^6,\ p^{6-3}=p^3[/math].
