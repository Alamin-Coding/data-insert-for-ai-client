Question:
[math]x + y[/math], [math]x - y[/math], [math]x^2 + y^2[/math] তিনটি দ্বিপদী রাশি

Options:
ক. প্রথম রাশিকে দ্বিতীয় রাশি দ্বারা গুণ কর।
খ. প্রথম, দ্বিতীয় ও তৃতীয় রাশিকে সূচকের সাহায্যে গুণফল নির্ণয় কর।
গ. খ হতে প্রাপ্ত গুণফলকে তৃতীয় রাশি দ্বারা ভাগ কর।


Correct Answer: 

Solution:
ক(x+y)(x−y)=x2−y2খগুণফল:(x+y)(x−y)(x2+y2)প্রথম দুইটি গুণফল: [math]x^2 - y^2[/math]
এরপর, (x2−y2)(x2+y2)=x4−y4গএখন, x2+y2x4−y4​[math]x^4 - y^4 = (x^2 + y^2)(x^2 - y^2)[/math]
সুতরাং ভাগফল: x2−y2

Explanation:
ব্যাখ্যা: ক থেকে [math](x+y)(x-y)=x^2-y^2[/math] (difference of squares)। খ সব তিনটি গুণ করলে [math](x^2-y^2)(x^2+y^2)=x^4-y^4[/math] পাওয়া যায়, আর গ এ সেই গুণফলকে তৃতীয় রাশি [math](x^2+y^2)[/math] দ্বারা ভাগ করলে পুনরায় [math]x^2-y^2[/math] পাওয়া যায় কারণ [math]x^4-y^4=(x^2-y^2)(x^2+y^2)[/math]।

Question:
[math]1 - x^3[/math], [math]1 - x + x^2[/math], [math]1 + x + x^2[/math] — তিনটি দ্বিপদী রাশি

Options:
ক. দ্বিতীয় রাশির যোগফলকে বিপরীত রাশি লিখ এবং ঘাতের অবক্রম অনুসারে সাজাও।
খ. সূত্রের সাহায্যে ২য় রাশিকে ৩য় রাশি দ্বারা গুণ কর।
গ. ১ম রাশিকে ২য় রাশি দ্বারা ভাগ কর।


Correct Answer: 

Solution:
কদ্বিতীয় রাশি: [math]1 - x + x^2[/math]
এর বিপরীত: [math]-1 + x - x^2[/math]
ঘাত অনুযায়ী সাজানো হলে:−x2+x−1খ(1−x+x2)(1+x+x2)সূত্র: [math](a - b)(a + b) = a^2 - b^2[/math]
এখানে [math]a = 1 + x + x^2[/math]
, [math]b = x[/math]
 নয়, তাই স্বাভাবিক গুণফলাফল হবে: [math]1 + x + x^2 - x - x^2 - x^3 + x^4 = 1 - x^3[/math]
অর্থাৎ: [math]1 - x^3[/math]
 গ1−x+x21−x3​=1+x+x2কারণ: (1−x3)=(1−x+x2)(1+x+x2)সুতরাং ভাগফল: [math]1 + x + x^2[/math]


Explanation:
ব্যাখ্যা:
ক দ্বিতীয় রাশি [math]1 - x + x^2[/math] এর যোগফলকের বিপরীত রাশি হলো [math]-1 + x - x^2[/math], ঘাতের অবক্রম (উত্থিত ঘাত অর্ধক্রমে) অনুসারে সাজালে [math]-x^2 + x - 1[/math] পাওয়া যায়।  
খ গুণ করলে (সরল গুণফল বা পরিচিত সূত্র প্রয়োগে) [math](1 - x + x^2)(1 + x + x^2)=1 + x^2 + x^4[/math] হয়, কারণ মধ্যবর্তী ধনাত্মক এবং ঋণাত্মক x ও x^3 ধরা টার্মগুলো পরস্পরে বাতিল হয়।  
গ ভাগ করলে বর্গে ব্যাবহার করে দেখা যায় [math]1 - x^3=(1 - x + x^2)(-x -1)+2[/math], তাই ভাগফল হচ্ছে [math]-x-1+\dfrac{2}{1 - x + x^2}[/math].
