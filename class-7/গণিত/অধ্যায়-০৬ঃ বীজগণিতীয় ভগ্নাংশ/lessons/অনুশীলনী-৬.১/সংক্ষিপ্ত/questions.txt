Question:
[math]\dfrac{a}{8x}[/math] ও [math]\dfrac{b}{4y}[/math] কে সমহর বিশিষ্ট ভগ্নাংশে প্রকাশ কর।

Options:


Correct Answer: 

Solution:
[math]\text{হর 8x এবং 4y এর ল.সা.গু. = } 8xy[/math]
∴ [math]\dfrac{a}{8x} = \dfrac{a \times y}{8x \times y} = \dfrac{ay}{8xy}[/math]
এবং, [math]\dfrac{b}{4y} = \dfrac{b \times 2x}{4y \times 2x} = \dfrac{2bx}{8xy}[/math]
∴ সমহর বিশিষ্ট ভগ্নাংশ দুটি হলো [math]\dfrac{ay}{8xy}[/math]
 এবং [math]\dfrac{2bx}{8xy}[/math]
 (Ans.)

Explanation:
সমাধান: দুই ভগ্নাংশের লঘিষ্ঠ সাধারণ হর হলো [math]8xy[/math] (কারণ [math]\text{lcm}(8,4)=8[/math] এবং উভয় চলক [math]x,y[/math] দরকার)। তাই

ক [math]\frac{a}{8x}=\frac{a\cdot y}{8xy}[/math].

খ [math]\frac{b}{4y}=\frac{b\cdot 2x}{8xy}=\frac{2bx}{8xy}[/math].

 [math]= \dfrac{p(p + 3) + 2(p + 3)}{p + 3} = \dfrac{(p + 3)(p + 2)}{p + 3}[/math]
 [math]= p + 2[/math]
অতএব, [math]A[/math]
 কে [math]p + 2[/math]
 দ্বারা ভাগ করলে পাই, [math]\dfrac{p + 2}{p + 2} = 1[/math]

Question:

Options:


Correct Answer: 

Solution:
[math]x^2 - 1 = (x - 1)(x + 1)[/math]
 [math]x^2 + 2x + 1 = (x + 1)^2[/math]
∴ সাধারণ গুণিত হর (ল.সা.গু.) = [math](x - 1)(x + 1)^2[/math]
∴ [math]\dfrac{1}{(x - 1)(x + 1)} = \dfrac{x + 1}{(x - 1)(x + 1)^2}[/math]
∴ [math]\dfrac{1}{(x + 1)^2} = \dfrac{x - 1}{(x - 1)(x + 1)^2}[/math]
∴ সমহর বিশিষ্ট ভগ্নাংশদ্বয়: [math]\dfrac{x + 1}{(x - 1)(x + 1)^2}[/math]
 এবং [math]\dfrac{x - 1}{(x - 1)(x + 1)^2}[/math]
 (Ans.)

Explanation:
ব্যাখ্যা: দুইটি ভগ্নাংশকে পার্শ্বধারী ভগ্নাংশে লিখে অজ্ঞাত ধ্রুবকগুলো নির্ণয় করা হল।

ক [math]\dfrac{1}{x^2-1}=\dfrac{A}{x-1}+\dfrac{B}{x+1}[/math] ধরা হলে [math]A(x+1)+B(x-1)=1[/math] থেকে [math]A+B=0,\;A-B=1[/math] হয়; সমাধানে [math]A=\tfrac{1}{2},\;B=-\tfrac{1}{2}[/math], ফলে [math]\dfrac{1}{x^2-1}=\tfrac{1}{2}\Big(\dfrac{1}{x-1}-\dfrac{1}{x+1}\Big)[/math]।

খ [math]\dfrac{1}{x^2+2x+1}=\dfrac{1}{(x+1)^2}=\dfrac{A}{x+1}+\dfrac{B}{(x+1)^2}[/math] হলে [math]A(x+1)+B=1[/math], x-ঘাতের সহগ থেকে [math]A=0[/math] এবং তাই [math]B=1[/math]; ফলে এটি ইতিমধ্যে অংশীয় ভগ্নাংশে [math]\dfrac{1}{(x+1)^2}[/math]।

Question:
[math]\dfrac{1}{x^2 - 4x}[/math] এবং [math]\dfrac{1}{x^2 - 2x - 8}[/math] ভগ্নাংশদ্বয় সমহর বিশিষ্ট ভগ্নাংশে প্রকাশ কর।

Options:


Correct Answer: 

Solution:
[math]x^2 - 4x = x(x - 4)[/math]
 [math]x^2 - 2x - 8 = (x - 4)(x + 2)[/math]
∴ সাধারণ গুণিত হর (ল.সা.গু.) = [math]x(x - 4)(x + 2)[/math]
∴ [math]\dfrac{1}{x(x - 4)} = \dfrac{x + 2}{x(x - 4)(x + 2)}[/math]
∴ [math]\dfrac{1}{(x - 4)(x + 2)} = \dfrac{x}{x(x - 4)(x + 2)}[/math]
∴ সমহর বিশিষ্ট ভগ্নাংশদ্বয়: [math]\dfrac{x + 2}{x(x - 4)(x + 2)}[/math]
 এবং [math]\dfrac{x}{x(x - 4)(x + 2)}[/math]
 (Ans.)

Explanation:
ব্যাখ্যা:
ক প্রথম ভগ্নাংশের হরঃ [math]x^2-4x=x(x-4)[/math] এবং ধরে নিই [math]\frac{1}{x(x-4)}=\frac{A}{x}+\frac{B}{x-4}[/math]; ফলে [math]1=A(x-4)+Bx[/math], রেখে [math]x=0\Rightarrow A=-\tfrac{1}{4},\; x=4\Rightarrow B=\tfrac{1}{4}[/math], তাই [math]\frac{1}{x^2-4x}=\frac{1}{4}\Big(\frac{1}{x-4}-\frac{1}{x}\Big)[/math].

খ দ্বিতীয় ভগ্নাংশের হরঃ [math]x^2-2x-8=(x-4)(x+2)[/math] এবং ধরে নিই [math]\frac{1}{(x-4)(x+2)}=\frac{A}{x-4}+\frac{B}{x+2}[/math]; ফলে [math]1=A(x+2)+B(x-4)[/math], রেখে [math]x=4\Rightarrow A=\tfrac{1}{6},\; x=-2\Rightarrow B=-\tfrac{1}{6}[/math], তাই [math]\frac{1}{x^2-2x-8}=\frac{1}{6}\Big(\frac{1}{x-4}-\frac{1}{x+2}\Big)[/math].

Solution:
[math]x^2 - 6x + 9 = (x - 3)^2[/math]
 [math]x^2 - 9 = (x - 3)(x + 3)[/math]

∴ [math]\dfrac{1}{(x - 3)(x + 3)} = \dfrac{x - 3}{(x - 3)^2(x + 3)}[/math]
∴ সমহর বিশিষ্ট ভগ্নাংশদ্বয়: [math]\dfrac{x + 3}{(x - 3)^2(x + 3)}[/math]

 (Ans.)

Explanation:
ব্যাখ্যা:

∴ উভয় ভগ্নাংশের সাধারণ গুণিত হর (ল.সা.গু.) = [math](x + 4)(x + 5)(x + 6)[/math]
∴ ১ম ভগ্নাংশ: [math]\dfrac{1}{(x + 5)(x + 4)} = \dfrac{x + 6}{(x + 4)(x + 5)(x + 6)}[/math]
∴ ২য় ভগ্নাংশ: [math]\dfrac{1}{(x + 5)(x + 6)} = \dfrac{x + 4}{(x + 4)(x + 5)(x + 6)}[/math]
∴ সমহর বিশিষ্ট ভগ্নাংশদ্বয়: [math]\dfrac{x + 6}{(x + 4)(x + 5)(x + 6)}[/math]
 এবং [math]\dfrac{x + 4}{(x + 4)(x + 5)(x + 6)}[/math]
 (Ans.)

Explanation:
ব্যাখ্যা: সমানরূপ ভগ্নাংশে লিখতে প্রতি ভগ্নাংশকে গুণোত্তর বর্গশূন্যাংশে ভেঙে ধ্রুবক নির্ণয় করা হয়।  
ক প্রথমটি ফ্যাক্টর করলে [math]x^2+9x+20=(x+4)(x+5)[/math], ধরে নিই [math]\frac{1}{(x+4)(x+5)}=\frac{A}{x+4}+\frac{B}{x+5}[/math]; গুণ করলে [math]1=A(x+5)+B(x+4)[/math], [math]x=-4[/math] দিলে [math]A=1[/math] এবং [math]x=-5[/math] দিলে [math]B=-1[/math], ফলে [math]\frac{1}{(x+4)(x+5)}=\frac{1}{x+4}-\frac{1}{x+5}[/math].  
খ দ্বিতীয়টি ধরে নিই [math]\frac{1}{(x+5)(x+6)}=\frac{C}{x+5}+\frac{D}{x+6}[/math]; সমানভাবে [math]1=C(x+6)+D(x+5)[/math], [math]x=-5[/math] থেকে [math]C=1[/math], [math]x=-6[/math] থেকে [math]D=-1[/math], ফলে [math]\frac{1}{(x+5)(x+6)}=\frac{1}{x+5}-\frac{1}{x+6}[/math].

Question:
[math]\dfrac{1}{9 + 8x^2 + 16x^4}[/math] ভগ্নাংশটির হরকে উৎপাদকে বিশ্লেষণ কর।

Options:


Correct Answer: 

Solution:
[math]A[/math]
 এর হর = [math]9 + 8x^2 + 16x^4[/math]
 [math]= (3)^2 + 2 \cdot 3 \cdot 4x^2 + (4x^2)^2 = (4x^2 + 3)^2[/math]
আবার, [math]= (4x^2 + 3)^2 - (4x)^2[/math]
 [math]= (4x^2 + 3 + 4x)(4x^2 + 3 - 4x)[/math]
 [math]= (4x^2 + 4x + 3)(4x^2 - 4x + 3)[/math]
 (Ans.)

Explanation:
একটি সহজ ধরন ধরে নেওয়া যায়: ধরে নিই [math](4x^2+px+3)(4x^2+qx+3)[/math]. গুণ করলে মধ্যবর্তী শব্দগুলো শূন্য হওয়ার শর্ত থেকে [math]p+q=0[/math] এবং সহগ থেকে [math]pq+24=8[/math] পাওয়া যায়, যা দিয়ে [math]p^2=16[/math] এবং [math]p=±4,\;q=∓4[/math] হয়; তাই হরকে উৎপাদকে লেখা যায় [math](4x^2+4x+3)(4x^2-4x+3)[/math].

Question:
[math]\dfrac{1}{2p^2 - 2p - 40}[/math] ভগ্নাংশটির হরকে উৎপাদকে বিশ্লেষণ কর।

Options:


Correct Answer: 

Solution:
ভগ্নাংশটির হর = [math]2p^2 - 2p - 40 = 2(p^2 - p - 20)[/math]
 [math]= 2(p^2 - 5p + 4p - 20)[/math]
 [math]= 2(p(p - 5) + 4(p - 5))[/math]
 [math]= 2(p - 5)(p + 4)[/math]
 (Ans.)

Explanation:
সমাধান: প্রথমে সাধারণ গুণক বের করলে [math]2p^2-2p-40=2(p^2-p-20)[/math]. এরপর বর্গঘাত সমীকরণটি বিভাজন করলে [math]p^2-p-20=(p-5)(p+4)[/math] হয়, তাই ভগ্নাংশটি হয়ে যায় [math]\dfrac{1}{2(p-5)(p+4)}[/math].

Question:
[math]\dfrac{1}{x^2 - 6x + 9}[/math] ভগ্নাংশটির হরকে উৎপাদকে বিশ্লেষণ কর।

Options:


Correct Answer: 

Solution:
[math]x^2 - 6x + 9 = (x - 3)^2[/math]
∴ হর = [math](x - 3)(x - 3)[/math]
 (Ans.)

Explanation:
ব্যাখ্যা: কারণ [math]x^2-6x+9=(x-3)^2[/math] — এটা একটি perfect square trinomial (কারণ [math]2\cdot3=6[/math] এবং [math]3^2=9[/math]). তাই ভগ্নাংশটি হয় [math]\dfrac{1}{(x-3)^2}[/math] (শর্ত: [math]x\neq3[/math]).
