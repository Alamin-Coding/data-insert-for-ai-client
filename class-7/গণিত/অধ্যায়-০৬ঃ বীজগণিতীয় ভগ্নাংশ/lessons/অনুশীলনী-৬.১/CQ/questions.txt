Question:
তিনটি বীজগণিতীয় ভগ্নাংশ: [math]\dfrac{1}{x(x+2)},\ \dfrac{1}{x^2+5x+6},\ \dfrac{1}{x^2 - x - 6}[/math]।

Options:
ক. প্রথম ভগ্নাংশের একটি সমতুল ভগ্নাংশ নির্ণয় কর যার লব [math]x-2[/math]।
খ. ভগ্নাংশ তিনটির হর এর ল.সা.গু. নির্ণয় করো।
গ. ভগ্নাংশ তিনটিকে সমহর বিশিষ্ট ভগ্নাংশে প্রকাশ করো।


Correct Answer: 

Solution:
ক. প্রথম ভগ্নাংশটি [math]\dfrac{1}{x(x+2)}[/math]
লব ও হর উভয়কে [math]x-2[/math]
 দ্বারা গুণ করলে পাই,[math]\dfrac{1}{x(x+2)} \times \dfrac{x-2}{x-2} = \dfrac{x-2}{x(x+2)(x-2)} = \dfrac{x-2}{x(x^2-4)}[/math]
যা [math]x-2[/math]
 লববিশিষ্ট প্রথম ভগ্নাংশের একটি সমতুল ভগ্নাংশ।খ. ১ম ভগ্নাংশের হর = [math]x(x+2)[/math]
২য় ভগ্নাংশের হর = [math]x^2+5x+6 = x^2+2x+3x+6 = x(x+2) + 3(x+2) = (x+2)(x+3)[/math]
৩য় ভগ্নাংশের হর = [math]x^2-x-6 = x^2-3x+2x-6 = x(x-3) + 2(x-3) = (x-3)(x+2)[/math]
ভগ্নাংশ তিনটির হরের ল.সা.গু = [math]x(x+2)(x+3)(x-3) = x(x+2)(x^2-9)[/math]
 (Ans.)গ.ল.সা.গু = [math]x(x+2)(x^2-9)[/math]
প্রথম ল.সা.গু. কে প্রতিটি হর দ্বারা ভাগ করলে ভাগফল যথাক্রমে [math](x+3)(x-3)[/math]
, [math]x(x-3)[/math]
 এবং [math]x(x+3)[/math]
এখন,[math]\dfrac{1}{x(x+2)} \times \dfrac{(x+3)(x-3)}{(x+3)(x-3)} = \dfrac{x^2-9}{x(x+2)(x-3)(x+3)} = \dfrac{x^2-9}{x(x+2)(x^2-9)}[/math]
 [math]\dfrac{1}{x^2+5x+6} \times \dfrac{x(x-3)}{x(x-3)} = \dfrac{x^2-3x}{(x+2)(x+3),x(x-3)} = \dfrac{x^2-3x}{x(x+2)(x^2-9)}[/math]
 [math]\dfrac{1}{x^2-x-6} \times \dfrac{x(x+3)}{x(x+3)} = \dfrac{x^2+3x}{x(x-3)(x+2)(x+3)} = \dfrac{x^2+3x}{x(x+2)(x^2-9)}[/math]
অতএব সমহর বিশিষ্ট ভগ্নাংশ তিনটি হলো:[math]\dfrac{x^2-9}{x(x+2)(x^2-9)},\ \dfrac{x^2-3x}{x(x+2)(x^2-9)},\ \dfrac{x^2+3x}{x(x+2)(x^2-9)}[/math]
 (Ans.)

Explanation:
ব্যাখ্যা:

ক প্রথম ভগ্নাংশকে [math]\dfrac{x-2}{x-2}[/math] দ্বারা গুণ করলে সমতুল ভগ্নাংশ পাওয়া যায়: [math]\dfrac{x-2}{x(x+2)(x-2)}[/math]।

খ ভাগহারগুলোকে বিশ্লেষণ করলে [math]x^2+5x+6=(x+2)(x+3)[/math] এবং [math]x^2-x-6=(x+2)(x-3)[/math]; তাই ল.সা.গু হবে [math]x(x+2)(x+3)(x-3)[/math]।

গ প্রতিটি ভগ্নাংশকে ল.সা.গু নামকরণ করতে প্রয়োজনীয় ঘাতক দিয়ে গুণ করলে পাওয়া যায়: 
[math]\dfrac{(x+3)(x-3)}{x(x+2)(x+3)(x-3)},\quad \dfrac{x(x-3)}{x(x+2)(x+3)(x-3)},\quad \dfrac{x(x+3)}{x(x+2)(x+3)(x-3)}[/math]।

Question:
তিনটি বীজগণিতীয় ভগ্নাংশ:

[math]\dfrac{a(a-4)}{a^2+3a},\ \dfrac{a}{a^2+5a+6},\ \dfrac{a}{a^2-a-12}[/math]

Options:
ক. [math]x^2 - 25y^2[/math] এর উৎপাদকে বিশ্লেষণ কর।
খ. প্রদত্ত ভগ্নাংশের হরগুলোর ল.সা.গু. নির্ণয় কর।
গ. ২য় ও ৩য় ভগ্নাংশ দুইটিকে সমহরবিশিষ্ট ভগ্নাংশে প্রকাশ কর।


Correct Answer: 

Solution:
ক. প্রদত্ত রাশি [math]x^2 - 25y^2 = (x)^2 - (5y)^2 = (x + 5y)(x - 5y)[/math]
 (Ans.)খ.১ম ভগ্নাংশের হর = [math]a^2 + 3a = a(a + 3)[/math]
২য় ভগ্নাংশের হর = [math]a^2 + 5a + 6 = a^2 + 2a + 3a + 6 = a(a + 2) + 3(a + 2) = (a + 2)(a + 3)[/math]
৩য় ভগ্নাংশের হর = [math]a^2 - a - 12 = a^2 - 4a + 3a - 12 = a(a - 4) + 3(a - 4) = (a - 4)(a + 3)[/math]
প্রদত্ত রাশিগুলোর সর্বাধিক সাধারণ গুণিতকগুলোর গুণফল = [math](a + 2), (a + 3), (a - 4)[/math]
নির্ণীত ল.সা.গু. = [math]a(a + 2)(a + 3)(a - 4)[/math]
 (Ans.)গ.২য় ভগ্নাংশ = [math]\dfrac{a}{a^2 + 5a + 6} = \dfrac{a}{(a + 2)(a + 3)}[/math]
৩য় ভগ্নাংশ = [math]\dfrac{a}{a^2 - a - 12} = \dfrac{a}{(a - 4)(a + 3)}[/math]
২য় ও ৩য় ভগ্নাংশের হরের ল.সা.গু. = [math](a + 2)(a + 3)(a - 4)[/math]
এখন,২য় ভগ্নাংশ = [math]\dfrac{a}{(a + 2)(a + 3)} \times \dfrac{a - 4}{a - 4} = \dfrac{a(a - 4)}{(a + 2)(a + 3)(a - 4)}[/math]
৩য় ভগ্নাংশ = [math]\dfrac{a}{(a - 4)(a + 3)} \times \dfrac{a + 2}{a + 2} = \dfrac{a(a + 2)}{(a + 2)(a + 3)(a - 4)}[/math]
অতএব সমহরবিশিষ্ট ভগ্নাংশ দুটি হলো:[math]\dfrac{a(a - 4)}{(a + 2)(a + 3)(a - 4)},\quad \dfrac{a(a + 2)}{(a + 2)(a + 3)(a - 4)}[/math]
 (Ans.)

Explanation:
ব্যাখ্যা:
ক [math]x^2-25y^2[/math] একটি পার্থক্যের বর্গ, তাই এটি বিশ্লেষণ করা যায়: [math](x-5y)(x+5y)[/math]।

খ হারগুলোর এগুলি বিভাজ্য: [math]a^2+3a=a(a+3),\;a^2+5a+6=(a+2)(a+3),\;a^2-a-12=(a+3)(a-4)[/math]; অতএব ল.সা.গু. হলো [math]a(a+2)(a+3)(a-4)[/math] (শর্ত: [math]a\neq0,-2,-3,4[/math])।

গ সাধারণ বাদামালার সাথে আনুপাতিক করার জন্য ২য়টি [math]\times\dfrac{a(a-4)}{a(a-4)}[/math] ও ৩য়টি [math]\times\dfrac{a(a+2)}{a(a+2)}[/math] করলে সমহরবিশিষ্ট ভগ্নাংশ পাওয়া যায়: ২য় = [math]\dfrac{a^2(a-4)}{a(a+2)(a+3)(a-4)}[/math], ৩য় = [math]\dfrac{a^2(a+2)}{a(a+2)(a+3)(a-4)}[/math]।

Question:
তিনটি বীজগণিতীয় রাশি:

[math]\dfrac{x^2 + x}{xy + y},\quad \dfrac{x y^2}{x y^2 z^2},\quad \dfrac{x y^3}{x^2 z^2 (x - 1)}[/math]

Options:
ক. ১ম ভগ্নাংশকে লঘুকরণ কর।
খ. ভগ্নাংশ তিনটি সমতুল কিনা যাচাই কর।
গ. ভগ্নাংশ তিনটিকে সমহর বিশিষ্ট ভগ্নাংশে প্রকাশ কর।


Correct Answer: 

Solution:
ক. ১ম ভগ্নাংশকে লঘুকরণ করপ্রথম ভগ্নাংশ: [math]\frac{x^2 + x}{xy + y}[/math]
লব: [math]x^2 + x = x(x + 1)[/math]
হর: [math]xy + y = y(x + 1)[/math]
অতএব, [math]\frac{x^2 + x}{xy + y} = \frac{x(x + 1)}{y(x + 1)}[/math]
এখানে [math](x + 1)[/math]
 কেটে দিলে পাই [math]\frac{x}{y}[/math]
।অতএব সরলীকৃত রূপ: [math]\frac{x}{y}[/math]
খ. ভগ্নাংশ তিনটি সমতুল কিনা যাচাই কর১ম ভগ্নাংশ (সরলীকৃত): [math]\frac{x}{y}[/math]
২য় ভগ্নাংশ: [math]\frac{xz^2}{yz^2(x - 1)} = \frac{x}{y(x - 1)}[/math]
৩য় ভগ্নাংশ: [math]\frac{xy^2}{y^2(x - 1)} = \frac{x}{x - 1}[/math]
কারণ [math]\frac{x}{y} 
e \frac{x}{y(x - 1)} 
e \frac{x}{x - 1}[/math]
,অতএব সমতুল নয়।গ. ভগ্নাংশ তিনটিকে সমহর বিশিষ্ট ভগ্নাংশে প্রকাশ করতিনটি ভগ্নাংশ:[math]\frac{x}{y},\ \frac{x}{y(x - 1)},\ \frac{x}{x - 1}[/math]
হরের ল.সা.গু = [math]y(x - 1)[/math]
প্রথম ভগ্নাংশ: [math]\frac{x}{y} \times \frac{x - 1}{x - 1} = \frac{x(x - 1)}{y(x - 1)}[/math]
দ্বিতীয় ভগ্নাংশ: [math]\frac{x}{y(x - 1)}[/math]
 (ইতিমধ্যেই সমহর)তৃতীয় ভগ্নাংশ: [math]\frac{x}{x - 1} \times \frac{y}{y} = \frac{xy}{y(x - 1)}[/math]
সমহর বিশিষ্ট রূপ:[math]\frac{x(x - 1)}{y(x - 1)},\ \frac{x}{y(x - 1)},\ \frac{xy}{y(x - 1)}[/math]
 (Ans.)

Explanation:
ব্যাখ্যা:

ক প্রথম ভগ্নাংশের উভয় অংশে [math](x+1)[/math] সাধারণ আছে: [math]\dfrac{x^2+x}{xy+y}=\dfrac{x(x+1)}{y(x+1)}=\dfrac{x}{y}[/math] (যদি [math]x\neq -1[/math])।

খ দ্বিতীয়টি সরলীকরণে হয় [math]\dfrac{1}{z^2}[/math] এবং তৃতীয়টি সরলীকরণে হয় [math]\dfrac{y^3}{x z^2 (x-1)}[/math]; সাধারণভাবে [math]\dfrac{x}{y},\ \dfrac{1}{z^2},\ \dfrac{y^3}{x z^2 (x-1)}[/math] সমতুল নয় (শুধু বিশেষভাবে পরিবর্তনশীলগুলোর নির্দিষ্ট সম্পর্ক থাকলে সমান হতে পারে)।

গ সাধারণ হর হিসাবে নেয়া যেতে পারে [math]x y z^2 (x-1)[/math], তখন তিনটি ভগ্নাংশ হয়  
[math]\dfrac{x}{y}=\dfrac{x^2 z^2 (x-1)}{x y z^2 (x-1)},\quad \dfrac{1}{z^2}=\dfrac{x y (x-1)}{x y z^2 (x-1)},\quad \dfrac{y^3}{x z^2 (x-1)}=\dfrac{y^4}{x y z^2 (x-1)}.[/math]
