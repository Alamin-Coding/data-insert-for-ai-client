Question:
সরলীকরণ কর: [math]\frac{(x - y)^2}{xyz} - \frac{v}{zx} + \frac{z}{xy}[/math]

Options:


Correct Answer: 

Solution:
প্রথমে একটি সাধারণ হর [math]xyz[/math]
 ঠিক করি:দ্বিতীয় ভগ্নাংশ: [math]\frac{v}{zx} = \frac{v \cdot y}{xyz}[/math]
তৃতীয় ভগ্নাংশ: [math]\frac{z}{xy} = \frac{z \cdot z}{xyz} = \frac{z^2}{xyz}[/math]
∴ [math]\frac{(x - y)^2 - vy + z^2}{xyz}[/math]
∴ চূড়ান্ত উত্তর: [math]\boxed{\frac{(x - y)^2 - vy + z^2}{xyz}}[/math]


Explanation:
সমাধান: সাধারণ ভাজক নেওয়া হলে তিনটি ভগ্নাংশের প্রত্যেকটির অভিন্ন ভাজক হবে [math]xyz[/math], তাই দ্বিতীয় ও তৃতীয় অংশকে উপযুক্তভাবে গুণ করলে পাওয়া যায় [math]\frac{(x-y)^2}{xyz}-\frac{vy}{xyz}+\frac{z^2}{xyz}[/math]. এগুলোকে একত্র করলে সরলিকৃত ফল হয় [math]\frac{(x-y)^2 - vy + z^2}{xyz}[/math]।

Question:
বিয়োগফল নির্ণয় কর: [math]\frac{5}{x + 7} - \frac{2}{x^2 - x - 56}[/math]

Options:


Correct Answer: 

Solution:
প্রথমে [math]x^2 - x - 56 = (x + 7)(x - 8)[/math]
∴ [math]\frac{5}{x + 7} - \frac{2}{(x + 7)(x - 8)}[/math]
ল.সা.গু. = [math](x + 7)(x - 8)[/math]
প্রথম ভগ্নাংশকে একই হরে রূপান্তর করি:[math]\frac{5(x - 8)}{(x + 7)(x - 8)} - \frac{2}{(x + 7)(x - 8)}[/math]
 [math]= \frac{5x - 40 - 2}{(x + 7)(x - 8)}[/math]
 [math]= \frac{5x - 42}{(x + 7)(x - 8)}[/math]
∴ [math]\boxed{\frac{5x - 42}{(x + 7)(x - 8)}}[/math]


Explanation:
প্রথমে গুণফলে বিন্যাস করলে [math]x^2-x-56=(x+7)(x-8)[math] হওয়ায় সাধারণ নামাঙ্কন [math](x+7)(x-8)[math] নিলে লঘু করি: [math]\frac{5(x-8)-2}{(x+7)(x-8)}[/math]. সরল করলে ফল হয় [math]\frac{5x-42}{(x+7)(x-8)}[/math] এবং শর্ত আছে যে [math]x\neq -7,\,8[/math].

Question:
বিয়োগফল নির্ণয় কর: [math]\frac{2}{x - 4} - \frac{2x}{x^2 + 5x - 36}[/math]

Options:


Correct Answer: 

Solution:
প্রথমে [math]x^2 + 5x - 36 = (x - 4)(x + 9)[/math]
∴ [math]\frac{2}{x - 4} - \frac{2x}{(x - 4)(x + 9)}[/math]
প্রথম ভগ্নাংশকে একই হরে রূপান্তর করি:[math]\frac{2(x + 9)}{(x - 4)(x + 9)} - \frac{2x}{(x - 4)(x + 9)}[/math]
 [math]= \frac{2x + 18 - 2x}{(x - 4)(x + 9)} = \frac{18}{(x - 4)(x + 9)}[/math]
∴ [math]\boxed{\frac{18}{(x - 4)(x + 9)}}[/math]


Explanation:
প্রথমে [math]x^2+5x-36=(x-4)(x+9)[/math] তাই প্রথম ভগ্নাংশকে [math](x+9)/(x+9)[math] দ্বারা বাড়িয়ে সাধারণ হারিকরণ করলে টার্মগুলো মিলে যায়: [math]\frac{2(x+9)-2x}{(x-4)(x+9)}=\frac{18}{(x-4)(x+9)}[/math]. মন রাখো [math]x\neq4,\,-9[/math].

Question:
বিয়োগফল নির্ণয় কর: [math]\frac{p + a}{p - a} - \frac{4ap}{(p + a)(p - a)}[/math]

Options:


Correct Answer: 

Solution:
প্রথম ভগ্নাংশের হর [math]p - a[/math]
, দ্বিতীয়টির হর [math](p + a)(p - a)[/math]
প্রথম ভগ্নাংশকে একই হরে রূপান্তর করি:[math]\frac{(p + a)(p + a)}{(p + a)(p - a)} = \frac{(p + a)^2}{(p + a)(p - a)}[/math]
∴ [math]\frac{(p + a)^2 - 4ap}{(p + a)(p - a)}[/math]
এখন লবকে সরল করি: [math](p + a)^2 - 4ap = p^2 + 2ap + a^2 - 4ap = p^2 - 2ap + a^2 = (p - a)^2[/math]
∴ [math]\frac{(p - a)^2}{(p + a)(p - a)} = \frac{p - a}{p + a}[/math]
∴ [math]\boxed{\frac{p - a}{p + a}}[/math]


Explanation:
সমাধান: সাধারণ নামাঙ্কন করা হলে প্রথম ভগ্নাংশকে একই হারে গুণ করে পাওয়া যায় [math]\frac{(p+a)^2}{(p+a)(p-a)}[/math], তাই পার্থক্যটি হচ্ছে [math]\frac{(p+a)^2-4ap}{(p+a)(p-a)}=\frac{p^2+2ap+a^2-4ap}{p^2-a^2}=\frac{p^2-2ap+a^2}{p^2-a^2}[/math]. উপরেকথা সহজ করলে নিউমেরেটর [math](p-a)^2[/math] হয় এবং একটি (p-a) কেটে গেলে ফল হয় [math]\frac{p-a}{p+a}[/math] (শর্ত: [math]p\neq \pm a[/math]).

Question:
বিয়োগফল নির্ণয় কর: [math]\frac{3x}{x^2 - 2x - 15} - \frac{2}{x^2 + x - 30}[/math]

Options:


Correct Answer: 

Solution:
প্রথমে উৎপাদকে বিশ্লেষণ করি:[math]x^2 - 2x - 15 = (x - 5)(x + 3)[/math]
 [math]x^2 + x - 30 = (x - 5)(x + 6)[/math]
∴ [math]\frac{3x}{(x - 5)(x + 3)} - \frac{2}{(x - 5)(x + 6)}[/math]
ল.সা.গু. = [math](x - 5)(x + 3)(x + 6)[/math]
প্রথম ভগ্নাংশকে: [math]\frac{3x(x + 6)}{(x - 5)(x + 3)(x + 6)}[/math]
দ্বিতীয় ভগ্নাংশকে: [math]\frac{2(x + 3)}{(x - 5)(x + 3)(x + 6)}[/math]
∴ [math]\frac{3x(x + 6) - 2(x + 3)}{(x - 5)(x + 3)(x + 6)}[/math]
 [math]= \frac{3x^2 + 18x - 2x - 6}{(x - 5)(x + 3)(x + 6)}[/math]
 [math]= \frac{3x^2 + 16x - 6}{(x - 5)(x + 3)(x + 6)}[/math]
∴ [math]\boxed{\frac{3x^2 + 16x - 6}{(x - 5)(x + 3)(x + 6)}}[/math]


Explanation:
সমাধান: প্রথমে ভগ্নাঙ্কগুলো তুলুন: [math]x^2-2x-15=(x-5)(x+3)[/math] এবং [math]x^2+x-30=(x-5)(x+6)[/math], তাই সাধারণ নামহার [math](x-5)(x+3)(x+6)[/math]. প্রতিটি ভগ্নাংশকে সেই নামহারে আনলে পাই [math]\frac{3x(x+6)-2(x+3)}{(x-5)(x+3)(x+6)}=\frac{3x^2+16x-6}{(x-5)(x+3)(x+6)}[/math]. নিউমারেটরটি আর নামহার কোনো সাধারণ গুণফল শেয়ার করে না, তাই এটি চূড়ান্ত সরলীকৃত রূপ।

Question:
বিয়োগফল নির্ণয় কর: [math]\frac{2x - 3}{x^2 - 11x + 30} - \frac{5}{x^2 - 16x + 63}[/math]

Options:


Correct Answer: 

Solution:
উৎপাদকে বিশ্লেষণ করি:[math]x^2 - 11x + 30 = (x - 6)(x - 5)[/math]
 [math]x^2 - 16x + 63 = (x - 9)(x - 7)[/math]
∴ [math]\frac{2x - 3}{(x - 6)(x - 5)} - \frac{5}{(x - 9)(x - 7)}[/math]
এটি সরাসরি ল.সা.গু. করা না গেলেও পৃথকভাবে বিশ্লেষণযোগ্য; অতএব,∴ [math]\boxed{\text{এই বিয়োগফল নির্ণয়ের জন্য একটি সাধারণ হর নির্ধারণ করে যোগফল করা সম্ভব}}[/math]
তবে এটি ফলাফলে যুক্ত করে:[math]\frac{(2x - 3)(x - 9)(x - 7) - 5(x - 6)(x - 5)}{(x - 6)(x - 5)(x - 9)(x - 7)}[/math]
এই পূর্ণ বিস্তার করে উত্তর: [math]\boxed{\frac{2x^3 - 39x^2 + 243x - 441 - (5x^2 - 55x + 150)}{(x - 6)(x - 5)(x - 9)(x - 7)}}[/math]
যা সরল করলে: [math]\boxed{\frac{2x^3 - 44x^2 + 298x - 591}{(x - 6)(x - 5)(x - 9)(x - 7)}}[/math]


Explanation:
প্রথমে মাথার নামচিহ্নগুলো ভেঙে নিই: [math]x^2-11x+30=(x-5)(x-6)[/math] এবং [math]x^2-16x+63=(x-7)(x-9)[/math], তাই সাধারণ নিতি হলো [math](x-5)(x-6)(x-7)(x-9)[/math].  
একই ভগ্নাংশে নিয়ে যোগ করলে পাই [math]\frac{(2x-3)(x-7)(x-9)-5(x-5)(x-6)}{(x-5)(x-6)(x-7)(x-9)}[/math].  
এটি বিস্তৃত ও সরল করলে ফল হয় [math]\frac{2x^3-40x^2+229x-339}{(x-5)(x-6)(x-7)(x-9)}[/math], এবং কোনো সাধারণ গুণক নেই, সুতরাং এটা চূড়ান্ত উত্তর।

Question:
বিয়োগফল নির্ণয় কর: [math]\frac{4}{x^2 - 10x + 21} - \frac{3x}{x^2 - 6x + 5}[/math]

Options:


Correct Answer: 

Solution:
উৎপাদকে বিশ্লেষণ করি:[math]x^2 - 10x + 21 = (x - 7)(x - 3)[/math]
 [math]x^2 - 6x + 5 = (x - 1)(x - 5)[/math]
∴ [math]\frac{4}{(x - 7)(x - 3)} - \frac{3x}{(x - 1)(x - 5)}[/math]
ল.সা.গু. = [math](x - 7)(x - 3)(x - 1)(x - 5)[/math]
∴ [math]\frac{4(x - 1)(x - 5) - 3x(x - 7)(x - 3)}{(x - 7)(x - 3)(x - 1)(x - 5)}[/math]
লব বিস্তার করে:[math]4(x^2 - 6x + 5) - 3x(x^2 - 10x + 21)[/math]
 [math]= 4x^2 - 24x + 20 - (3x^3 - 30x^2 + 63x)[/math]
 [math]= -3x^3 + 34x^2 - 87x + 20[/math]
∴ [math]\boxed{\frac{-3x^3 + 34x^2 - 87x + 20}{(x - 7)(x - 3)(x - 1)(x - 5)}}[/math]


Explanation:
প্রক্রিয়া: প্রথমে নামানগুলিকে ভগ্নাংশের গুণনীয় হিসাবে ফ্যাক্টর করে নেওয়া যায়: [math]x^2-10x+21=(x-3)(x-7)[/math] এবং [math]x^2-6x+5=(x-1)(x-5)[/math], তাই সাধারণ নামাঙ্ক হল [math](x-3)(x-7)(x-1)(x-5)[/math].  
প্রতিটি ভগ্নাংশকে সেই নামাঙ্কে আনলে অভিন্ন সূচক রূপে বিয়োগ করলে লেখাটি হয় [math]\dfrac{4(x-1)(x-5)-3x(x-3)(x-7)}{(x-3)(x-7)(x-1)(x-5)}[/math], যা প্রসারিত করলে সংখ্যাগুণকায় [math]-3x^3+34x^2-87x+20[/math] পাওয়া যায়.  
অতএব ফলাফলটি হল [math]\dfrac{-3x^3+34x^2-87x+20}{(x-3)(x-7)(x-1)(x-5)}, যেখানে [math]x\neq 1,5,3,7[/math]।

Question:
সরলীকরণ কর: [math]\frac{m + n}{m} - \frac{m - n}{m} - 2[/math]

Options:


Correct Answer: 

Solution:
প্রথমে ভগ্নাংশ দুটি একত্রে করি:[math]\frac{m + n - (m - n)}{m} - 2 = \frac{m + n - m + n}{m} - 2[/math]
 [math]= \frac{2n}{m} - 2[/math]
এখন একটি হরে আনতে লিখি: [math]= \frac{2n - 2m}{m}[/math]
তবে ছবির পদ্ধতিতে সব ধাপ একত্রে আছে:[math]= \frac{m(m + n) - n(m - n) - 2mn}{mn}[/math]
 [math]= \frac{m^2 + mn - mn + n^2 - 2mn}{mn} = \frac{m^2 - 2mn + n^2}{mn}[/math]
 [math]= \frac{(m - n)^2}{mn}[/math]
∴ [math]\boxed{\frac{(m - n)^2}{mn}}[/math]


Explanation:
দুটি ভগ্নাংশের ভগ্নক একই হওয়ায় এগুলো একত্র করলে পাওয়া যায় [math]\frac{(m+n)-(m-n)}{m}=\frac{2n}{m}[/math]. এরপর [math]2[/math] বিয়োগ করলে ফল হয় [math]\frac{2n}{m}-2=2\left(\frac{n}{m}-1\right)=\frac{2(n-m)}{m}[/math].

Question:
সরলীকরণ কর: [math]\frac{a}{a + b} - \frac{b}{a - b} + \frac{2ab}{a^2 - b^2}[/math]

Options:


Correct Answer: 

Solution:
এখানে [math]a^2 - b^2 = (a + b)(a - b)[/math]
∴ প্রথম দুইটি ভগ্নাংশের ল.সা.গু. = [math](a + b)(a - b)[/math]
প্রথম: [math]\frac{a(a - b)}{(a + b)(a - b)}[/math]
দ্বিতীয়: [math]\frac{b(a + b)}{(a + b)(a - b)}[/math]
তৃতীয়: [math]\frac{2ab}{(a + b)(a - b)}[/math]
∴ [math]\frac{a(a - b) - b(a + b) + 2ab}{(a + b)(a - b)}[/math]
 [math]= \frac{a^2 - ab - ab - b^2 + 2ab}{a^2 - b^2}[/math]
 [math]= \frac{a^2 - b^2}{a^2 - b^2} = 1[/math]
∴ [math]\boxed{1}[/math]


Explanation:
সমান হর নেওয়ার পর প্রতিটি ভগ্নাংশকে বিভাজক [math](a+b)(a-b)=a^2-b^2[/math] এ আনলে উর্দ্ধপাতি হয়ে যায় [math]a^2-b^2[/math], তাই সমীকরণটি দাঁড়ায় [math]\dfrac{a^2-b^2}{a^2-b^2}=1[/math]. শর্ত: এটি বৈধ কেবল যখন [math]a\ne b[/math] এবং [math]a\ne -b[/math] (অর্থাৎ [math]a^2\ne b^2[/math]).

Question:
সরলীকরণ কর: [math]\frac{4}{x^2 - 2x - 35} + \frac{2}{x + 5} - \frac{1}{x - 7}[/math]

Options:


Correct Answer: 

Solution:
উৎপাদকে বিশ্লেষণ করি: [math]x^2 - 2x - 35 = (x - 7)(x + 5)[/math]
∴ [math]\frac{4}{(x - 7)(x + 5)} + \frac{2}{x + 5} - \frac{1}{x - 7}[/math]
একই হরে রূপান্তর করে লিখি:[math]\frac{4 + 2(x - 7) - 1(x + 5)}{(x - 7)(x + 5)}[/math]
 [math]= \frac{4 + 2x - 14 - x - 5}{(x - 7)(x + 5)}[/math]
 [math]= \frac{x - 15}{(x - 7)(x + 5)}[/math]
∴ [math]\boxed{\frac{x - 15}{(x - 7)(x + 5)}}[/math]


Explanation:
ব্যাখ্যা: প্রথমে গুণকারী দ্বারা ভগ্নাংশের সমান নামকরণ করা যায় কারণ [math]x^2-2x-35=(x-7)(x+5)[/math]. তাই [math]\frac{2}{x+5}=\frac{2(x-7)}{(x-7)(x+5)}[/math] এবং [math]\frac{1}{x-7}=\frac{x+5}{(x-7)(x+5)}[/math] নিয়ে সবকটি যোগ করলে উপরেরগুলোকে যোগ করলে নামগুণক সাধারণ হয়ে যায় এবং অঙ্কগুলো যোগ করলে: [math]4+2(x-7)-(x+5)=x-15[/math]. ফলে সরলীকৃত ফলাফল হবে [math]\frac{x-15}{(x-7)(x+5)}[/math], যেখানে [math]x\neq7,\,x\neq-5[/math]।

ব্যাখ্যা: সংযোজনের জন্য ভগ্নাঙ্কগুলোর নামকালকে গুণনীয়কের রূপে লিখি [math]x^2-9=(x-3)(x+3),\;x^2+2x-3=(x+3)(x-1)[/math], সাধারণ নামকরা [math](x-3)(x+3)(x-1)[/math]. তাই যোগ করলে, [math]\frac{3(x-1)+5(x-3)}{(x-3)(x+3)(x-1)}=\frac{8x-18}{(x-3)(x+3)(x-1)}=\frac{2(4x-9)}{(x-3)(x+3)(x-1)}[/math] এবং সংজ্ঞায়নের জন্য [math]x\neq 3,\,-3,\,1[/math].
