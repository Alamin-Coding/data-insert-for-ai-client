Question:
[math]\frac{4}{x^2 - 3x - 54} + \frac{3}{x - 9}[/math] এর যোগফল নির্ণয় কর।

Options:


Correct Answer: 

Solution:
প্রথমে, [math]x^2 - 3x - 54 = (x - 9)(x + 6)[/math]
∴ [math]\frac{4}{x^2 - 3x - 54} + \frac{3}{x - 9} = \frac{4}{(x - 9)(x + 6)} + \frac{3}{x - 9}[/math]
এখন, দ্বিতীয় ভগ্নাংশটির হরকে সমপার্থক হরে রূপান্তর করি:[math]\frac{3}{x - 9} = \frac{3(x + 6)}{(x - 9)(x + 6)}[/math]
∴ [math]\frac{4}{(x - 9)(x + 6)} + \frac{3(x + 6)}{(x - 9)(x + 6)}[/math]
এখন একই হরে যোগ করি:[math]\frac{4 + 3(x + 6)}{(x - 9)(x + 6)} = \frac{4 + 3x + 18}{(x - 9)(x + 6)}[/math]
∴ [math]\frac{3x + 22}{(x - 9)(x + 6)}[/math]
অতএব, যোগফল = [math]\boxed{\frac{3x + 22}{(x - 9)(x + 6)}}[/math]


Explanation:
ব্যাখ্যা: বহুগুণকে ভগ্নাংশের নামমাত্রে ভাঙলে [math]x^2-3x-54=(x-9)(x+6)[/math] পাওয়া যায়, তাই প্রথম ভগ্নাংশটি লেখতে পারি [math]\frac{4}{(x-9)(x+6)}[/math] এবং দ্বিতীয়টিকে ওসই নামমাত্রে আনলে [math]\frac{3}{x-9}=\frac{3(x+6)}{(x-9)(x+6)}[/math]। যোগ করলে উভয়ের উপরাংশ যোগ হয়ে [math]\frac{4+3(x+6)}{(x-9)(x+6)}=\frac{3x+22}{(x-9)(x+6)}[/math] হয়, যেখানে [math]x\neq 9,\,-6[/math]।

Question:
[math]A = \frac{1}{p^2 + 4}[/math] এবং [math]B = \frac{2}{p^2 - 16}[/math] হলে, [math]A + B[/math] নির্ণয় কর।

Options:


Correct Answer: 

Solution:
প্রথমে, [math]A + B = \frac{1}{p^2 + 4} + \frac{2}{p^2 - 16}[/math]
এখানে [math]p^2 - 16 = (p - 4)(p + 4) = (p^2 - 4^2)[/math]
কিন্তু [math]p^2 + 4[/math]
 কে সরাসরি উৎপাদকে ভাঙ্গা যাবে না, তাই সাধারণ হর ব্যবহার করব:∴ [math]A + B = \frac{1(p^2 - 16) + 2(p^2 + 4)}{(p^2 + 4)(p^2 - 16)}[/math]
 [math]= \frac{p^2 - 16 + 2p^2 + 8}{(p^2 + 4)(p^2 - 16)}[/math]
 [math]= \frac{3p^2 - 8}{(p^2 + 4)(p^2 - 16)}[/math]
অতএব, [math]A + B = \boxed{\frac{3p^2 - 8}{(p^2 + 4)(p^2 - 16)}}[/math]


Explanation:
[math]A+B=\frac{1}{p^2+4}+\frac{2}{p^2-16}=\frac{(p^2-16)+2(p^2+4)}{(p^2+4)(p^2-16)}=\frac{3p^2-8}{(p^2+4)(p^2-16)}[/math] । শর্ত: [math]p\ne\pm4[/math] যাতে হারগুলো সংজ্ঞায়িত থাকে।

Question:
[math]M = \frac{a}{2a + 2}[/math] এবং [math]N = \frac{-3a}{a^2 - 1}[/math] হলে, [math]M + N[/math] নির্ণয় কর।

Options:


Correct Answer: 

Solution:
প্রথমে, [math]M = \frac{a}{2a + 2} = \frac{a}{2(a + 1)}[/math]
এবং [math]N = \frac{-3a}{a^2 - 1} = \frac{-3a}{(a + 1)(a - 1)}[/math]
এখন, [math]M + N = \frac{a}{2(a + 1)} + \frac{-3a}{(a + 1)(a - 1)}[/math]
সাধারণ হর হবে [math]2(a + 1)(a - 1)[/math]
∴ [math]\frac{a(a - 1) - 3a \cdot 2}{2(a + 1)(a - 1)}[/math]
 [math]= \frac{a^2 - a - 6a}{2(a + 1)(a - 1)} = \frac{a^2 - 7a}{2(a + 1)(a - 1)}[/math]
∴ [math]\boxed{\frac{a(a - 7)}{2(a + 1)(a - 1)}}[/math]


Explanation:
সমাধান: অমূলক ভগ্নাংশগুলোকে গুণফেক্টর করে লিখলে [math]M=\frac{a}{2(a+1)}[/math] এবং [math]N=\frac{-3a}{(a-1)(a+1)}[/math]। সাধারণ নামপাত [math]2(a-1)(a+1)[/math] নেওয়া হলে যোগফলটির হিসাব হয় [math]\frac{a(a-1)-6a}{2(a-1)(a+1)}=\frac{a(a-7)}{2(a-1)(a+1)}[/math] (শর্ত: [math]a\ne\pm1[/math]).

[math]\frac{a - b - c}{2a} + \frac{a + b + c}{3a}[/math] এর যোগফল নির্ণয় করে [math]6a[/math] দ্বারা গুণ কর।

Options:


Correct Answer: 

Solution:
প্রদত্ত রাশি: [math]\frac{a - b - c}{2a} + \frac{a + b + c}{3a}[/math]
ল.সা.গু. = [math]6a[/math]
∴ [math]\frac{3(a - b - c) + 2(a + b + c)}{6a}[/math]
 [math]= \frac{3a - 3b - 3c + 2a + 2b + 2c}{6a}[/math]
 [math]= \frac{5a - b - c}{6a}[/math]
এখন [math]6a[/math]
 দ্বারা গুণ করলে: [math]\frac{5a - b - c}{6a} \times 6a = 5a - b - c[/math]
∴ চূড়ান্ত উত্তর: [math]\boxed{5a - b - c}[/math]


Explanation:
প্রথমে ভাগগুলোকে সাধারণ নামাঙ্কনে নিয়ে আসলে [math]\frac{a-b-c}{2a}+\frac{a+b+c}{3a}=\frac{3(a-b-c)+2(a+b+c)}{6a}=\frac{5a-b-c}{6a}[/math]. এবার এটা [math]6a[/math] দ্বারা গুণ করলে নামাঙ্ক বাতিল হয়ে ফল হয় [math]5a-b-c[/math]।

Question:
[math]\frac{x - a}{x + a} + \frac{4ax}{x^2 - a^2}[/math] এর যোগফল নির্ণয় কর।

Options:


Correct Answer: 

Solution:
প্রদত্ত রাশি: [math]\frac{x - a}{x + a} + \frac{4ax}{x^2 - a^2}[/math]
যেহেতু [math]x^2 - a^2 = (x + a)(x - a)[/math]
∴ [math]\frac{x - a}{x + a} + \frac{4ax}{(x + a)(x - a)}[/math]
প্রথম ভগ্নাংশের হরকে একই হরে রূপান্তর করি:[math]\frac{(x - a)^2}{(x + a)(x - a)} + \frac{4ax}{(x + a)(x - a)}[/math]
∴ [math]\frac{(x - a)^2 + 4ax}{(x + a)(x - a)}[/math]
 [math]= \frac{x^2 - 2ax + a^2 + 4ax}{(x + a)(x - a)}[/math]
 [math]= \frac{x^2 + 2ax + a^2}{(x + a)(x - a)} = \frac{(x + a)^2}{(x + a)(x - a)}[/math]
∴ [math]\boxed{\frac{x + a}{x - a}}[/math]


Explanation:
ব্যাখ্যা: সাধারণ নামবাহক [math]x^2-a^2=(x-a)(x+a)[/math] নিয়ে যোগ করলে সংখ্যাগুণক করলে পাই numerator [math](x-a)^2+4ax=x^2-2ax+a^2+4ax=x^2+2ax+a^2=(x+a)^2[/math]. সুতরাং ফল [math]\frac{(x+a)^2}{x^2-a^2}=\frac{x+a}{x-a}[/math], শর্ত [math]x\neq\pm a[/math].

Options:


Correct Answer: 

Solution:
প্রদত্ত রাশি: [math]A + B = \frac{1}{x^2 + 2x} + \frac{4}{x^2 - 4}[/math]
 [math]x^2 + 2x = x(x + 2)[/math]
 এবং [math]x^2 - 4 = (x + 2)(x - 2)[/math]
∴ [math]\frac{1}{x(x + 2)} + \frac{4}{(x + 2)(x - 2)}[/math]
ল.সা.গু. = [math](x + 2)(x)(x - 2)[/math]
∴ [math]\frac{1(x - 2) + 4x}{(x + 2)(x)(x - 2)}[/math]
 [math]= \frac{x - 2 + 4x}{(x + 2)(x)(x - 2)} = \frac{5x - 2}{(x + 2)(x)(x - 2)}[/math]
∴ চূড়ান্ত উত্তর: [math]\boxed{\frac{5x - 2}{(x + 2)(x)(x - 2)}}[/math]

 [math]x^2 + 2x - 3 = (x + 3)(x - 1)[/math]
∴ [math]\frac{3}{(x - 3)(x + 3)} + \frac{5}{(x + 3)(x - 1)}[/math]

Question:
[math]\frac{2x}{x^2 - x - 6} + \frac{3}{x^2 - 4}[/math] = ?

Options:


Correct Answer: 

Solution:
প্রথমে উৎপাদকে বিশ্লেষণ করি:[math]x^2 - x - 6 = (x - 3)(x + 2)[/math]
 [math]x^2 - 4 = (x - 2)(x + 2)[/math]
∴ [math]\frac{2x}{(x - 3)(x + 2)} + \frac{3}{(x - 2)(x + 2)}[/math]
ল.সা.গু. = [math](x - 3)(x + 2)(x - 2)[/math]
∴ [math]\frac{2x(x - 2) + 3(x - 3)}{(x - 3)(x + 2)(x - 2)}[/math]
 [math]= \frac{2x^2 - 4x + 3x - 9}{(x - 3)(x + 2)(x - 2)} = \frac{2x^2 - x - 9}{(x - 3)(x + 2)(x - 2)}[/math]
∴ [math]\boxed{\frac{2x^2 - x - 9}{(x - 3)(x + 2)(x - 2)}}[/math]


Explanation:
Factor the denominators: [math]x^2-x-6=(x-3)(x+2)[/math] and [math]x^2-4=(x-2)(x+2)[/math], so the common denominator is [math](x-3)(x+2)(x-2)[/math]; the numerator becomes [math]2x(x-2)+3(x-3)=2x^2-x-9[/math]. Hence the sum is [math]\frac{2x^2-x-9}{(x-3)(x+2)(x-2)}[/math], with [math]x\neq3,-2,2[/math].

Options:


Correct Answer: 

Solution:
প্রথমে উৎপাদকে বিশ্লেষণ করি:[math]a^2 - 3a = a(a - 3)[/math]

∴ [math]\frac{(a + 3) - a}{a(a - 3)(a + 3)} = \frac{3}{a(a - 3)(a + 3)}[/math]
∴ [math]\boxed{\frac{3}{a(a^2 - 9)}}[/math]


Explanation:

Correct Answer: 

Solution:

∴ [math]\frac{(x - 3)(x + 3)}{(x + 2)(x + 3)} \div \frac{x - 3}{x + 2}[/math]
ভাগকে গুণে রূপান্তর করি:[math]\frac{(x - 3)(x + 3)}{(x + 2)(x + 3)} \times \frac{x + 2}{x - 3}[/math]

 [math]= \frac{1}{1} = \boxed{1}[/math]


Explanation:
প্রথমে গুণনীয় করে মুকলি করলে [math]\frac{x^2-9}{x^2+5x+6}=\frac{(x-3)(x+3)}{(x+2)(x+3)}=\frac{x-3}{x+2}[/math], তাই পুরো অভিব্যক্তি হয়ে যায় [math]\frac{x-3}{x+2}\div\frac{x-3}{x+2}=1[/math]. শর্ত: মূল সমস্যা থেকে ডোমেইন সীমাবদ্ধ থাকে, তাই [math]x\neq -2,\,-3,\;3[/math] (বিভাজ্যক বা বিভাজ্য দ্বারা ভাগ শূন্য হলে অক্ষম)।

Question:
বীজগণিতীয় ভগ্নাংশের সাধারণ হৃবিষ্টীকরণের প্রক্রিয়া কি?

Options:


Correct Answer: ভগ্নাংশের লব ও হরকে সাধারণ গুণনীয়ক দ্বারা ভাগ করলে ভগ্নাংশটি সাধারণ হৃবিষ্টীকৃত হয়।

Solution:
সমাধান:

উত্তরসূত্রগত ধারণা: বীজগণিতীয় ভগ্নাংশকে সাধারণীকরণ (simplification) করা মানে ভগ্নাংশের লব (numerator) ও হর (denominator) থেকে সাধারণ গুণনীয়ক(গুলো) বাদ দিয়ে সহজতর রূপে লেখাই। তবে শুধুমাত্র গুণনীয়ক (factor) গুলোই বাদ দেয়া যায়—যোগ বা বিয়োগের সদস্য সরাসরি বাতিল করা যায় না। সঙ্গে হরটি শূন্য করলে চলবে না এমন মানগুলো (domain restriction) উল্লেখ করতে হয়।

ধাপেঃ
1. পূর্ণভাবে গুণনীয়করণ করো: লব ও হরকে সম্ভাব্যভাবে ভগ্নাংশ, সাধারণ গুণিতক, পার্থক্য-ও-স্কোয়ারস, ত্রিঘাতীয় ইত্যাদি দ্বারা বিলিয়ন করো।  
2. সাধারণ গুণনীয়ক শনাক্ত করো: লব ও হরের মধ্যে সাধারণ গুণনীয়ক (factor) থাকলে সেগুলো চিহ্নিত করো।  
3. বাদ দাও (cancel) শুধুমাত্র সাধারণ গুণনীয়কগুলো—ফ্যাক্টর হিসেবে উপস্থিত থাকলে।  
4. অবশিষ্ট রূপ লেখো এবং হরের শূন্যকারী মানগুলি (যা আদতে বৈধ নয়) তালিকাভুক্ত করো।

উদাহরণ (সংক্ষিপ্ত, বোঝার সুবিধার জন্য):  
ধরা যাক ভগ্নাংশ [math]\dfrac{x^{2}-9}{x^{2}-3x}[/math] কে সাধারণীকরণ করো।  
- গুণনীয়ক করো: [math]x^{2}-9=(x-3)(x+3)[/math], [math]x^{2}-3x=x(x-3)[/math].  
- সাধারণ গুণনীয়ক [math](x-3)[/math] আছে; এটিকে বাদ দিলে পাই [math]\dfrac{x+3}{x}[/math].  
- শর্ত/সীমা: হর শূন্য হওয়া নিষিদ্ধ, সুতরাং প্রথমিক ভগ্নাংশে [math]x
eq 0[/math] এবং [math]x
eq 3[/math]। সাধারণীকৃত রূপে বলা যায় [math]\dfrac{x+3}{x}[/math] কিন্তু অরিজিনাল ভগ্নাংশের জন্য [math]x
eq 0,\,3[/math] বজায় রাখতে হবে।

অতিরিক্ত টীকা: সংখ্যাসূচক গুণনীয়ক থাকলে প্রথমে সাধারণ গ.সা.গু. (GCD) ব্যবহার করে সংখ্যাগুলো সরল করা যায়; বিষয়টি সর্বদা গুণনীয়ক হিসেবে উপস্থিত থাকার নিয়মেই বাতিল করা যায়।

Explanation:
ব্যাখ্যা: ভগ্নাংশ সাধারণ হৃবিষ্টীকরণে লব ও হর—উভয়ই যদি কোনো সাধারণ গুণনীয়ক দ্বারা বিভাজ্য হয়, তখন উভয়কে ওই গুণনীয়ক দিয়ে ভাগ করলে ভগ্নাংশটি সরল রূপে লেখাযায়; উদাহরণস্বরূপ [math]\frac{6}{8}[/math]-এ সাধারণ গুণনীয়ক [math]2[/math] দ্বারা ভাগ করলে [math]\frac{3}{4}[/math] পাওয়া যায়।

Question:
বীজগণিতীয় ভগ্নাংশের লঘুকরণের প্রক্রিয়া কিভাবে সম্পন্ন হয়?

Options:


Correct Answer: বীজগণিতীয় ভগ্নাংশের লঘুকরণে লব ও হরকে তাদের সাধারণ গুণনীয়ক দ্বারা ভাগ করা হয়।

Solution:
সমাধান:

বীজগণিতীয় ভগ্নাংশ লঘুকরণের মূল ধারণা হলো লব (উপরের অংশ) ও হর (নিচের অংশ) কে সাধারণ গুণনীয়ক দ্বারা ভাগ করা—অর্থাৎ দুটি অংশকে তাদের সাধারণ কারণগুলো বের করে তাগুলো বাতিল করা।

পদ্ধতি (ধাপে ধাপে):
1. প্রথমে লব ও হরকে গুণকরণ করে প্রকাশ বা ভগ্নাংশের প্রতিটি অংশকে পূর্ণভাবে গুণনীয়করণ করো। উদাহরণস্বরূপ, বহু সদস্যের বহুপদকে সাধারণভাবে ফ্যাক্টরাইজ করো।  
2. লব ও হর উভয়ের জন্য সাধারণ গুণনীয়ক (common factors) শনাক্ত করো।  
3. একই গুণনীয়ক উভয় অংশে উপস্থিত থাকলে তা ওপর থেকে নিম্নে ভাগ করে বাতিল করো (cancel)। এই ভাগ/বাতিল কেবল তখনই বৈধ যখন বাতিল করা গুণনীয়ক শূন্য না হয়।  
4. লঘু করা ভগ্নাংশ লিখে নাও এবং প্রয়োজনে হরকে শূন্য করা মানগুলো (ডোমেইন অপরিবর্তিত রাখার জন্য) জানিয়ো — অর্থাৎ সেই মানগুলি ব্যতীত সমীকরণটি প্রযোজ্য থাকবে যেগুলো হরের মান শূন্য করে।

উদাহরণ (সংক্ষিপ্ত এবং দরকারী):
উদাহরণ হিসেবে ভগ্নাংশটি ধরা যাক [math]\dfrac{x^2-9}{x^2-6x+9}[/math]।  
- ফ্যাক্টরাইজ করলে: [math]x^2-9=(x-3)(x+3)[/math] এবং [math]x^2-6x+9=(x-3)(x-3)[/math].  
- সাধারণ গুণনীয়ক [math](x-3)[/math] বাতিল করলে পাওয়া যায় [math]\dfrac{x+3}{x-3}[/math].  
- তবে মনে রাখা যাবে যে মূল ভগ্নাংশে [math]x
eq 3[/math] (কারণ হর শূন্য হবে), তাই লঘুকৃত ফলেও [math]x
eq 3[/math] শর্ত বজায় থাকবে।

সংক্ষিপ্ত নোট: লঘুকরণে লক্ষ্য হল ভগ্নাংশকে সহজতর ফর্মে আনা—এবং সবসময় ডোমেইন বা শর্তগুলি চেক করা জরুরি যেন শূন্য দিয়ে ভাগ করা না হয়।

Explanation:
ব্যাখ্যা: বীজগণিতীয় ভগ্নাংশ লঘুকরণে লব ও হরকে তাদের সাধারণ গুণনীয়ক (সর্বোচ্চ সাধারণ গুণনীয়ক) দ্বারা ভাগ করা হয় যাতে একটি সমতুল্য কিন্তু সরল ভগ্নাংশ পাওয়া যায়। উদাহরণস্বরূপ, [math]\frac{6x}{9}=\frac{6x÷3}{9÷3}=\frac{2x}{3}[/math]।

Question:
লঘুকরণ বা সাধারণ হরবিশিষ্টকরণের গুরুত্ব কি এবং এটি কিভাবে সম্পন্ন হয়?

Options:


Correct Answer: লঘুকরণ বা সাধারণ হরবিশিষ্টকরণ ভগ্নাংশকে সহজে সমাধান করার জন্য ব্যবহৃত হয়, যেখানে লব এবং হরকে তাদের সাধারণ গুণনীয়ক দ্বারা ভাগ করা হয়।

Solution:
সমাধান:

লঘুকরণ বা সাধারণ হরবিশিষ্টকরণ (fraction reduction) একটি ভগ্নাংশকে সর্বনিম্ন গুণনীয়কে আনা—অর্থাৎ এমনভাবে লেখার প্রক্রিয়া যাতে লব (নিউমেরেটর) ও হর (ডিনোমিনেটর) আর কোন সাধারণ গুণনীয় না থাকে। এর গুরুত্ব এবং সম্পাদনের ধাপ নিচে দেয়া হলো।

গুরুত্ব:
- হিসাব সহজ হয়: যোগ, বিয়োগ, গুণ, ভাগ এবং তুলনা করা সুবিধাজনক হয়।
- ফলাফল সংকুচিত এবং সহজে ব্যাখ্যা যোগ্য হয়।
- ত্রুটি কমে এবং গণনা দ্রুত হয় (বিশেষত বড় সংখ্যার ক্ষেত্রে)।

কিভাবে সম্পন্ন হয় (ধাপগত পদ্ধতি):
1. সাধারণ গুণনীয় নির্ণয় করুন — অর্থাৎ লব ও হরের গREATEস্ট কমন ডিভাইসর (GCD) বা সর্বাধিক সাধারণ বিভাজক খুঁজে বের করুন। GCD পাওয়ার পদ্ধতি:
   - মৌলিক গুণকরণ (prime factorization) ব্যবহার: প্রতিটি সংখ্যার মৌলিক গুণনীয় লিখে সাধারণ গুণকগুলো নির্বাচন করুন।
   - অথবা ইউক্লিডীয় অ্যালগরিদম ব্যবহার: ধারাবাহিক ভাগশেষ নিয়ে GCD নির্ণয় করা যায়।
2. লব ও হরকে উভয়কেই GCD দ্বারা ভাগ করুন। ফলে পাওয়া অনুপাতটি হবে সরল রূপ।

সংক্ষিপ্ত উদাহরণ:
উদাহরণ 1: [math]\dfrac{18}{24}[/math] কে লঘুকরণ করুন।
- ধাপ 1: GCD(18,24) = 6 (কারণ [math]18=2\times3^2,\;24=2^3\times3[/math], সাধারণ গুণক [math]=2\times3=6[/math])।
- ধাপ 2: উভয়কে 6 দ্বারা ভাগ করলে [math]\dfrac{18}{24}=\dfrac{18\div6}{24\div6}=\dfrac{3}{4}[/math]।

চিহ্ন সম্পর্কিত নিয়ম:
- যদি লব বা হর নেগেটিভ হয়, সাধারণত ঋণাত্মক চিহ্নটিকে লব-এ রাখা হয়: উদাহরণ [math]\dfrac{-6}{8}=\dfrac{-3}{4}[/math]।
- যদি GCD = 1 হয়, ভগ্নাংশটি ইতোমধ্যে সূচকীয় (simplest) অবস্থায় আছে।

সংক্ষেপে: লঘুকরণটি ভগ্নাংশকে সহজ ও কার্যকরভাবে ব্যবহারের যোগ্য করে তুলতে লব ও হরকে তাদের সর্বাধিক সাধারণ বিভাজক দ্বারা ভাগ করাই।

Explanation:
ব্যাখ্যা: লঘুকরণ বা সাধারণ হরবিশিষ্টকরণ ভগ্নাংশকে সহজতম রূপে আনার পদ্ধতি, যেখানে লব (উপরের সংখ্যা) ও হর (নীচের সংখ্যা) তাদের সাধারণ গুণনীয়ক বা সর্বোচ্চ সাধারণ গুণনীয়ক দ্বারা ভাগ করে কাটা হয়। এতে ভগ্নাংশটি তুলনা, যোগ-বিয়োগ ও গুণায় সহজ হয় এবং হিসাব ভুলের সম্ভাবনা কমে যায়।

Question #59 (ID: 128862)

Question:
বীজগণিতীয় ভগ্নাংশের যোগ ও বিয়োগের নিয়ম কী?

Options:


Correct Answer: বীজগণিতীয় ভগ্নাংশ যোগফলের জন্য সাধারণ হরবিশিষ্ট করতে হয় এবং বিয়োগের জন্যও একই প্রক্রিয়া অনুসরণ করতে হয়।

Solution:
সমাধান:

উত্তর পদ্ধতি (সারাংশ)
1. যোগ বা বিয়োগ করার জন্য প্রথমে ভগ্নাংশগুলোর সাধারণ হর (common denominator) নির্ণয় করতে হবে — সাধারণত ডিনোমিনেটরগুলোর ল.সা.গু (LCM) নেওয়া হয়।  
2. প্রতিটি ভগ্নাংশকে ঐ সাধারণ হরে আনতে প্রয়োজনীয় মাধ্যমে গুণ করে নবীন অঙ্ক (numerator) সংশোধন করা।  
3. একই হরে আনা হলে যোগে সংখ্যাগুলো যোগ করতে হয় এবং বিয়োগে একটির থেকে অন্যটি বিয়োগ করতে হয়, ফলে একটি একক ভগ্নাংশ পাওয়া যায়।  
4. শেষে সাধারণ গুণক কেটে সরলীকরণ করতে হবে এবং ডিনোমিনেটর শূন্য হবার মান (domain restriction) উল্লেখ করতে হবে।

প্রধান নিয়ম (সংক্ষিপ্ত)
- যদি ভগ্নাংশগুলোর ডিনোমিনেটর একই হয়, তবে সরাসরি নিউমেরেটর যোগ/বিয়োগ করে ডিনোমিনেটর অপরিবর্তিত থাকে: [math]\frac{a}{d} \pm \frac{b}{d} = \frac{a \pm b}{d}[/math]।  
- যদি ডিনোমিনেটর ভিন্ন হয়, প্রথমে সাধারণ হর [math]D=\mathrm{LCM}(d_1,d_2,\dots)[/math] বের করে প্রতিটি ভগ্নাংশকে [math]D[/math] হারে আনুন, তারপর নিউমেরেটর যোগ/বিয়োগ করুন: [math]\frac{a}{d_1} \pm \frac{b}{d_2} = \frac{a\cdot (D/d_1) \pm b\cdot (D/d_2)}{D}[/math]।  
- শেষে সম্ভব হলে সংখ্যা এবং ভগ্নাংশকে সরল করুন এবং ডিনোমিনেটরের জন্য মান নিষিদ্ধ করুন (যেখানে ডিনোমিনেটর শূন্য হবে)।

উদাহরণ (সংক্ষিপ্ত, প্রাসঙ্গিক)
উদাহরণ ১ (ডিনোমিনেটর ভিন্ন): যোগ করুন [math]\frac{2x}{x^2-1} + \frac{3}{x+1}[/math].

ধাপ 1: ডিনোমিনেটর বিশ্লেষণ: [math]x^2-1=(x-1)(x+1)[/math], তাই LCM = [math](x-1)(x+1)[/math].  
ধাপ 2: দ্বিতীয় ভগ্নাংশকে LCM-এ আনতে গুণ করুন: [math]\frac{3}{x+1}=\frac{3(x-1)}{(x+1)(x-1)}[/math]. প্রথমটি ইতিমধ্যে সেই হরে আছে।  
ধাপ 3: নিউমেরেটর যোগ করুন:  
[math]\frac{2x}{(x-1)(x+1)}+\frac{3(x-1)}{(x-1)(x+1)}=\frac{2x+3(x-1)}{(x-1)(x+1)}[/math]  
ধাপ 4: সরলীকরণ: [math]\frac{2x+3x-3}{x^2-1}=\frac{5x-3}{x^2-1}[/math].  
ডোমেইন সীমাবদ্ধতা: [math]x
eq \pm 1[/math]।

উদাহরণ ২ (একই ডিনোমিনেটর, বিয়োগ): বিয়োগ করুন [math]\frac{x}{x+2}-\frac{2}{x+2}[/math].

একই ডিনোমিনেটর হলে সরাসরি: [math]\frac{x-2}{x+2}[/math], যেখানে [math]x
eq -2[/math]।

সংক্ষেপে: বীজগণিতীয় ভগ্নাংশের যোগ ও বিয়োগ করার মূল নিয়ম হল সাধারণ হর নির্ণয় করে প্রতিটি ভগ্নাংশকে সেই হরে আনা, তারপর নিউমেরেটরগুলো যোগ/বিয়োগ করে ফলকে সহজতর করা এবং ডিনোমিনেটরের শূন্য হওয়া মানগুলো বাদ দেওয়া।

Explanation:
ব্যাখ্যা: বীজগণিতীয় ভগ্নাংশ যোগ বা বিয়োগ করার আগে তাদের সাধারণ হর (একই হার) করতে হয় যাতে সরাসরি উপরের অংশগুলো যোগ‑বিয়োগ করা যায়। উদাহরণস্বরূপ, [math]\frac{a}{b}+\frac{c}{d}=\frac{ad+bc}{bd}[/math] এবং [math]\frac{a}{b}-\frac{c}{d}=\frac{ad-bc}{bd}[/math], পরে প্রয়োজন মতো সরলীকরণ করা হয়।

Question:
বীজগণিতীয় ভগ্নাংশের লঘুকরণের পদ্ধতি কী এবং এটি কেন গুরুত্বপূর্ণ?

Options:


Correct Answer: লব ও হরকে সাধারণ গুণনীয়ক দ্বারা ভাগ করা হয় এবং এটি জটিল সমস্যা সমাধানে সাহায্য করে।

Solution:
সমাধান:

সংজ্ঞা ও উদ্দেশ্য:
বীজগণিতীয় ভগ্নাংশ লঘুকরণ মানে হলো ভগ্নাংশটিকে এমন রুপে আনা যাতে উপরের অংশ (লব) ও নিচের অংশ (হরক)–এর মধ্যে আর সাধারণ গুণনীয়ক না থাকে (শুধুমাত্র ±1 বাকি থাকে)। এর উদ্দেশ্য হলো হিসাব সহজ করা, সমীকরণ ও অনুক্রম বিশ্লেষণ সহজতর করা এবং ত্রুটি কমানো।

পদ্ধতি (ধাপে ধাপে):
1. পূর্ণভাবে গুণনীয়ক বের করা: প্রথমে উপরের ও নিচের অংশকে পূর্ণরূপে গুণnij (factor) করা—যেমন সাধারণ গুণফল বের করা, পারফেক্ট স্কোয়্যার, পার্থক্য-স্কোয়ারের সূত্র, গোষ্ঠীকরণ ইত্যাদি ব্যবহার করা।
2. সাধারণ গুণনীয়ক খুঁজে বের করা: উভয় অংশে যে গুণনীয়কগুলো সাধারণ থাকলে সেগুলো শনাক্ত করা।
3. সাধারণ গুণনীয়ক বাতিল করা: সাধারণ গুণনীয়ক উভয় থেকে ভাগ করে ফেলা (ক্যান্সেল করা) যাতে ভগ্নাংশ সরল হয়ে যায়।
4. ডোমেইন-শর্ত রাখা: বাতিল করার পরে মনে রাখতে হবে যে যেসব মান হরককে শূন্য করত সেগুলো মুল ভগ্নাংশের জন্য বৈধ নয়—অর্থাৎ অনুচিত মানগুলো বাদ দিতে হবে।

গুরুত্বপূর্ণ নীতিমালা ও সতর্কতা:
- কেবল গুণনীয়ক (factor) ক্যান্সেল করা যায়; যোগ/বিয়োগের অংশ সরাসরি ক্যান্সেল করা যায় না।
- সংখ্যাগত সাধারণ গুণনীয়ক থাকলে প্রথমে তা ভাগ করা যেতে পারে।
- ক্যান্সেল করার পর মুল ভগ্নাংশের ডোমেইন শর্ত উল্লেখ করতে হয় (যেখানে হরক শূন্য হয় তা ব্যতীত)।

সংক্ষেপে গণিতীয় ব্যাখ্যা:
লব ও হরককে সাধারণ গুণনীয়ক দ্বারা ভাগ করা হয়—অর্থাৎ, যদি ভগ্নাংশটি [math]\dfrac{A\cdot C}{B\cdot C}[/math] রূপে লেখা যায়, তাহলে তা [math]\dfrac{A}{B}[/math] এ সরল করা যায় (শর্ত: [math]C
eq 0[/math] এবং মুল ভগ্নাংশের ডোমেইন ঠিক রাখতে হবে)।

উদাহরণ (সংক্ষিপ্ত ও বিদ্যমান মানে গুরুত্বপূর্ণ হলে):
ধরা যাক ভগ্নাংশ [math]\dfrac{x^{2}-9}{x^{2}-6x+9}[/math] লঘুকরণ করতে হবে।
- পার্থক্য-স্কোয়ার ও পূর্ণবর্গ চিহ্নিত করে ফ্যাক্টর করলে:
  [math]x^{2}-9=(x-3)(x+3),\quad x^{2}-6x+9=(x-3)(x-3)[/math]
- সাধারণ গুণনীয়ক [math](x-3)[/math] ক্যান্সেল করলে সরল রূপ হয়:
  [math]\dfrac{x+3}{x-3}[/math]
- কিন্তু মূল ভগ্নাংশে [math]x=3[/math] হলে হরক শূন্য হত; তাই সরল রূপ ব্যবহারের সময় শর্ত হিসেবে রাখতে হবে [math]x
eq 3[/math]।

গুরুত্ব:
এটি সমীকরণ সমাধান, বিভাজ্যতা বিশ্লেষণ, অন্তরক ও অনুক্রম বিশ্লেষণে সরলতা আনে, বাজে গাণিতিক কাজ কমায় এবং গণনাকে কার্যকর করে তোলে।

সংক্ষিপ্ত উত্তর:
লব ও হরককে সাধারণ গুণনীয়ক দ্বারা ভাগ করা হয় যাতে ভগ্নাংশটি সরল হয়; এটি গণিতীয় কাজগুলো সহজ ও নির্ভুল করতে গুরুত্বপূর্ণ।

Explanation:
লঘুকরণে সংখ্যার লব (উপরের অংশ) ও হর (নীচের অংশ) তাদের সর্বাধিক সাধারণ গুণনীয়ক দ্বারা ভাগ করা হয় যাতে ভগ্নাংশের মান অপরিবর্তিত রেখে একদম ছোট রূপ পাওয়া যায়। এটি ঠিক কারণ একই সংখ্যায় উপরের ও নিচের অংশ ভাগ করলে ভগ্নাংশের অনুপাত অপরিবর্তিত থাকে কিন্তু সংখ্যাগুলো ছোট হয়। ফলে গণনা, যোগ-বিয়োগ, তুলনা ও সমীকরণ সমাধান করা সহজ ও নির্ভুল হয়।

Question:
কিভাবে সাধারণ হরবিশিষ্ট ভগ্নাংশ তৈরি করা হয় এবং এর প্রয়োজনীয়তা কেন?

Options:


Correct Answer: দুটি ভগ্নাংশের হরের ল.সা.গু বের করে এবং উভয়ের লবকে সেই ভাগফল দ্বারা গুণ করা হয়, এটি যোগ-বিয়োগে সুবিধা দেয়।

Solution:
সমাধান:

প্রক্রিয়া (answer approach):
- সাধারণ হরবিশিষ্ট ভগ্নাংশ তৈরি করতে দুটি বা বেশি ভগ্নাংশের অভিন্ন হর (common denominator) করা হয়। এজন্য প্রত্যেক ভগ্নাংশের হরের ল.সা.গু (least common multiple) বের করা হয় এবং প্রতিটি ভগ্নাংশকে সেই ল.সা.গু দিয়ে সমান হর করা হয়।

প্রধান ধাপসমূহ (essential steps):
1. ধরা যাক ভগ্নাংশ দুটো [math]\frac{p}{q}[/math] এবং [math]\frac{r}{s}[/math]। প্রথমে তাদের হরের ল.সা.গু নির্ণয় করো: [math]m=\text{lcm}(q,s)[/math]।
2. প্রতিটি ভগ্নাংশকে [math]m[/math] হরে আনতে প্রথমটির উভয় সদস্যকে [math]\frac{m}{q}[/math] দ্বারা এবং দ্বিতীয়টির উভয় সদস্যকে [math]\frac{m}{s}[/math] দ্বারা গুণ করো:
   - প্রথমটি হয়ে যাবে [math]\frac{p\cdot(m/q)}{m}[/math]
   - দ্বিতীয়টি হয়ে যাবে [math]\frac{r\cdot(m/s)}{m}[/math]
3. এখন যোগ-বিয়োগ সহজ: একই হরের ফলে তাদের যোগ/বিয়োগ করা যায় যেমন
   [math]\frac{p\cdot(m/q)}{m}\pm\frac{r\cdot(m/s)}{m}=\frac{p\cdot(m/q)\pm r\cdot(m/s)}{m}[/math]।
4. প্রয়োজনে ফলটিকে সংক্ষেপ করো (গ.সা.গু দিয়ে ভাগ)।

প্রয়োজনীয়তা ও যুক্তি (why it is necessary):
- ভগ্নাংশ যোগ বা বিয়োগ করার সময় সরাসরি যোগ্য করার জন্য হর সমান হওয়া বাধ্যতামূলক; ভিন্ন হর হলে সংখ্যা সরাসরি যোগ করা যায় না।
- সাধারণ হর থাকার ফলে তুলনা, ক্রম নির্ণয় এবং সরলীকরণ সহজ হয়।
- বহু ভগ্নাংশের ক্ষেত্রে একক সাধারণ হর নিয়ে কাজ করলে গণনায় সুবিধা এবং সঠিকতা বাড়ে।

উদাহরণ (concise useful example):
- যোগ করো [math]\frac{1}{4}+\frac{1}{6}[/math]।
  ল.সা.গু([math]4,6[/math]) = [math]12[/math]।
  তাই [math]\frac{1}{4}=\frac{1\cdot3}{12}=\frac{3}{12}[/math], [math]\frac{1}{6}=\frac{1\cdot2}{12}=\frac{2}{12}[/math]।
  যোগ করলে [math]\frac{3}{12}+\frac{2}{12}=\frac{5}{12}[/math]।

এই পদ্ধতি সাধারণ ভগ্নাংশ তৈরি করে যোগ-বিয়োগ ও অন্যান্য কার্যে সুবিধা দেয় এবং ফলকে সহজে সংক্ষেপও করা যায়।

Explanation:
দুই ভগ্নাংশ যেমন [math]\frac{a}{b}[/math] ও [math]\frac{c}{d}[/math] এর হরের ল.সা.গু (LCM) নিলে, প্রতিটির লবকে অনুপাতে [math]\frac{\mathrm{lcm}(b,d)}{b}[/math] ও [math]\frac{\mathrm{lcm}(b,d)}{d}[/math] দ্বারা গুণ করে একই হর পাওয়া যায়। একই হরে ন্যুমেরেটরগুলো সরাসরি যোগ বা বিয়োগ করা যায়, তাই যোগ-বিয়োগ সহজ ও সঠিক হয়।

Question:
বীজগণিতীয় ভগ্নাংশের লঘুকরণের প্রক্রিয়ায় কীভাবে লব এবং হরকে সাধারণ গুণনীয়ক দ্বারা ভাগ করা হয়?

Options:


Correct Answer: লব এবং হরকে তাদের সাধারণ গুণনীয়ক দ্বারা ভাগ করে ভগ্নাংশটিকে লঘিষ্ঠ আকারে পরিণত করা হয়।

Solution:
সমাধান:
ভগ্নাংশকে লঘু (irreducible) করতে লব (উপরের সংখ্যা — numerator) এবং হর (নীচের সংখ্যা — denominator) উভয়কে তাদের সাধারণ গুণনীয়ক (greatest common divisor — GCD) দ্বারা ভাগ করা হয়। এই পদ্ধতির মূল ধারণা: একই অসমূহ দ্বারা লব ও হরকে ভাগ করলে ভগ্নাংশের মান অপরিবর্তিত থাকে, কিন্তু গুণনীয়কগুলো সরিয়ে ভগ্নাংশ সরল রূপে আসে।

পদ্ধতি (ধাপভিত্তিক):
1. লব ও হরের সাধারণ গুণনীয়ক নির্ণয় করো — সাধারণত গুণনীয়কগুলো তালিকা করে সর্বোচ্চ সাধারণ গুণনীয়ক (GCD) খুঁজে বা ইউক্লিডীয় অ্যালগরিদম ব্যবহার করে।  
2. লব ও হর উভয়কে ওই GCD দ্বারা ভাগ করো।  
3. প্রাপ্ত নতুন লব ও হর আর কোনো সাধারণ গুণনীয়ক না থাকলে ভগ্নাংশ লঘু হয়েছে।

যুক্তি:
- যদি লব ও হরকে একই সংখ্যায় ভাগ করা হয় তাহলে ভগ্নাংশের মান অপরিবর্তিত থাকে, কারণ [math]\dfrac{a}{b}=\dfrac{a/k}{b/k}[/math] (এখানে [math]k
eq0[/math])।
- সর্বশেষে যদি GCD = 1 হয়, ভগ্নাংশটি লঘিষ্ঠ আকারে আছে।

উদাহরণ:
1) [math]\dfrac{18}{24}[/math] — সাধারণ গুণনীয়ক GCD = [math]6[/math]। তাই [math]\dfrac{18}{24}=\dfrac{18/6}{24/6}=\dfrac{3}{4}[/math]।  
2) (ইউক্লিডীয় অ্যালগরিদম দেখিয়ে) [math]\dfrac{84}{126}[/math] — ইউক্লিড অনুযায়ী GCD = [math]42[/math] (কারণ [math]126=84\times1+42[/math], [math]84=42\times2+0[/math])। তাই [math]\dfrac{84}{126}=\dfrac{84/42}{126/42}=\dfrac{2}{3}[/math]।

বিশেষ নোট:
- যদি লব = 0 হয় এবং হর ≠ 0 হয়, ভগ্নাংশশ = 0 (আর এটির লঘু রূপ [math]0/1[/math] হিসেবে দেখা যায়)।  
- নেগেটিভ চিহ্ন থাকলে সাধারণত চিহ্নটি লব-এ রেখে হর ধনাত্মক করা হয়।

Explanation:
কারণ একই সাধারণ গুণনীয়ক দিয়ে লব ও হরকে ভাগ করলে ভগ্নাংশটির মান অপরিবর্তিত থাকে কিন্তু সংখ্যাগুলো ছোট হয়ে সহজ রূপে আসে। তাই ভগ্নাংশ লঘুকরণের জন্য লব ও হর-এর সর্বোচ্চ সাধারণ গুণনীয়ক (GCD) খুঁজে নিয়ে উভয়কে দিয়ে ভাগ করা হয়।

Question:
বীজগণিতীয় ভগ্নাংশের যোগফল নির্ণয় করার জন্য প্রথমে কোন ধাপগুলো অনুসরণ করতে হবে?

Options:


Correct Answer: প্রথমে ভগ্নাংশগুলোকে লঘিষ্ঠ সাধারণ হরবিশিষ্ট করতে হবে এবং তারপর লবের যোগফল বের করতে হবে।

Solution:
সমাধান:

বীজগণিতীয় ভগ্নাংশ যোগ করার সাধারণ পদ্ধতি সংক্ষেপে ও ধাপে ধাপে:

১) ভগ্নাংশগুলোর হার (নামমাত্র) যদি জটিল হয় তবে সেগুলোকে প্রথমে গুণনীয়ক (factor) করে নাও — কারণ লঘিষ্ঠ সাধারণ হার নির্ণয়ে এটা প্রয়োজন। উদাহরণ: [math]x^2-1=(x-1)(x+1)[/math]।

২) ভগ্নাংশগুলোর জন্য লঘিষ্ঠ সাধারণ হার (LCM) নির্ণয় করো — অর্থাৎ এমন একটি নামমাত্র যা সব নামমাত্রকে সম্পূর্ণভাবে বিভাজ্য করে। উদাহরণ: [math]\text{LCM}=(x-1)(x+1)=x^2-1[/math]।

৩) প্রতিটি ভগ্নাংশকে লঘিষ্ঠ সাধারণ হারের সঙ্গে সমরূপ করতে (equivalent fraction) উচিতীয় গুণ দ্বারা বড়াও, যাতে সব ভগ্নাংশের নামমাত্র সমান হয়। উদাহরণ:  
- [math]\frac{1}{x+1}=\frac{1\cdot(x-1)}{(x+1)(x-1)}=\frac{x-1}{x^2-1}[/math]  
- [math]\frac{2}{x-1}=\frac{2\cdot(x+1)}{(x-1)(x+1)}=\frac{2(x+1)}{x^2-1}[/math]

৪) নামমাত্র সমান হলে শুধু লব (নিউমেরেটর) গুলো যোগ করো এবং সাধারণ নামমাত্র রাখো:  
উপরে উদাহরণে, যোগফল হবে  
[math]\frac{x-1}{x^2-1}+\frac{2(x+1)}{x^2-1}=\frac{x-1+2x+2}{x^2-1}=\frac{3x+1}{x^2-1}[/math]।

৫) যদি সম্ভব হয় সংক্ষেপ (simplify) করো — অর্থাৎ লব ও হার common factor থাকলে ভাগ করে নাও। পরে ডোমেইন বা অনুপযুক্ত মানগুলো যাচাই করো: নামে থাকা মানগুলোকে ০ বানায় এমন x-গুলো গ্রহণ করা যাবে না (উদাহরণ: এখানে [math]x
eq\pm1[/math])।

সংক্ষেপে: প্রথমে নামমাত্রগুলোকে ফ্যাক্টর করে লঘিষ্ঠ সাধারণ হার নির্ণয় করতে হয়, তারপর প্রতিটি ভগ্নাংশকে সেই হারের সঙ্গে সমরূপ করে লবগুলো যোগ করে, শেষে প্রয়োজনমতো সরলীকরণ ও ডোমেইন পরীক্ষা করতে হবে।

Explanation:
ভাগ অংশের আকার (হার) একই না হলে সরাসরি যোগ করা যায় না, তাই প্রথমে হারগুলোর লঘিষ্ঠ সাধারণ গুণিতক (LCM) নিয়ে সাধারণ হার তৈরি করতে হয়। এরপর প্রতিটি ভগ্নাংশকে সেই সাধারণ হারে অনুপাতে বাড়িয়ে নিউমেরেটরগুলো যোগ করা হয় — উদাহরণ: [math]1/3 + 1/4 = 4/12 + 3/12 = 7/12[/math]।

Question:
যখন দুটি ভগ্নাংশ সমহরবিশিষ্ট করা হয়, তখন তাদের হরের ল.সা.গু বের করার প্রক্রিয়া কেমন?

Options:


Correct Answer: ভগ্নাংশগুলোর হরের ল.সা.গু বের করে তা ব্যবহার করে নতুন সমহরবিশিষ্ট ভগ্নাংশ তৈরি করা হয়।

Solution:
সমাধান:

পদ্ধতি ও মূল ধারণা:
1. সমহরবিশিষ্ট (equivalent) ভগ্নাংশ তৈরির মূল নীতি—প্রতিটি ভগ্নাংশের হিসেবে নূন্যতম সমান হরের (common denominator) থাকে; ভগ্নাংশের মান একই রাখতে প্রতিটি ভগ্নাংশের সংখ্যক (numerator) এবং হর (denominator) একই সংখ্যায় গুণ করতে হয়।
2. দুইটি ভগ্নাংশকে সমহরবিশিষ্ট করতে তাদের হরের সর্বনিম্ন সাধারণ গুণিতক (LCM) নিতে হয়—এটাই সবচেয়ে ছোট সাধারণ হর যা উভয় হরকে ভাজ্য করে।
3. প্রতিটি ভগ্নাংশের নতুন সংখ্যক হবে: পুরানো সংখ্যক × (LCM ÷ পুরানো হর) এবং নতুন হর হবে LCM।

চরণবিন্যাস:
Step 1: দুই হরের LCM নির্ণয় করুন।
Step 2: প্রতিটি ভগ্নাংশের জন্য গুণফল নির্ণয় করুন: গুণক = [math]\dfrac{\text{LCM}}{\text{হর}}[/math]।
Step 3: পুরানো সংখ্যককে সেই গুণকে দিয়ে গুণ করুন: নতুন সংখ্যক = [math]\text{পুরানো সংখ্যক} \times \dfrac{\text{LCM}}{\text{হর}}[/math]। নতুন হর = LCM।
Step 4: প্রয়োজনে ফলকে সরল করুন, কিন্তু সমান হরের ভগ্নাংশ হিসেবে LCM হরই ব্যবহার করা হবে।

উদাহরণ (সংক্ষেপে মূল্যবান প্রয়োগ):
উদাহরণ 1: ভগ্নাংশ [math]\dfrac{2}{3}[/math] এবং [math]\dfrac{5}{6}[/math] সমহরবিশিষ্ট করতে—
- LCM(3,6) = 6
- প্রথমটির গুণক = [math]\dfrac{6}{3}=2[/math], তাই [math]\dfrac{2}{3} = \dfrac{2\times2}{3\times2} = \dfrac{4}{6}[/math]
- দ্বিতীয়টি ইতিমধ্যেই হর 6: [math]\dfrac{5}{6}[/math]
ফল: সমহরবিশিষ্ট ভগ্নাংশ [math]\dfrac{4}{6}[/math] এবং [math]\dfrac{5}{6}[/math]।

উদাহরণ 2: [math]\dfrac{3}{4}[/math] এবং [math]\dfrac{5}{8}[/math]—
- LCM(4,8)=8
- [math]\dfrac{3}{4}=\dfrac{3\times2}{4\times2}=\dfrac{6}{8}[/math], এবং [math]\dfrac{5}{8}[/math] অপরিবর্তিত।
ফল: [math]\dfrac{6}{8}[/math] ও [math]\dfrac{5}{8}[/math]।

সংক্ষেপে: দুই ভগ্নাংশকে সমহরবিশিষ্ট করতে তাদের হরের LCM বের করে প্রতিটি ভগ্নাংশের সংখ্যক-হারকে সেই LCM অনুযায়ী সমান অনুপাতে গুণ করা হয়।

Explanation:
ব্যাখ্যা: সমহরবিশিষ্ট করতে ভগ্নাংশগুলোর হরের একই করে নিতে হয়; এজন্য তাদের হরের ল.সা.গু নেওয়া হয় যাতে ছোটতম সাধারণ হার পাওয়া যায়। এরপর প্রতিটি ভগ্নাংশ [math]\frac{a}{b}[/math] কে [math]\frac{k}{k}[/math] দিয়ে গুণ করা হয় যেখানে [math]k=\dfrac{\text{ল.সা.গু}}{b}[/math], ফলে সবগুলো ভগ্নাংশ একই হরে সমহরবিশিষ্ট হয়ে যায়।

Question:
বীজগণিতীয় ভগ্নাংশ সরলীকরণের প্রক্রিয়া কিভাবে সম্পন্ন হয় এবং এর উদ্দেশ্য কী?

Options:


Correct Answer: ভগ্নাংশগুলোকে লঘিষ্ঠ আকারে প্রকাশ করা এবং সহজতর রূপে রূপান্তরিত করার উদ্দেশ্যে সরলীকরণ করা হয়।

Solution:
সমাধান:

1. উদ্দেশ্য সংক্ষেপে:
   - ভগ্নাংশগুলোকে লঘিষ্ঠ বা সর্বনিম্ন রূপে প্রকাশ করা যাতে তুলনা, গণনা এবং আরও জটিল অপারেশন (যোগ, বিয়োগ, গুণ, ভাগ) সহজ হয়। পাশাপাশি মানদণ্ডগত ও একক উপস্থাপনার জন্যও সরলীকরণ করে রাখা হয়।  

2. সাধারিত পদ্ধতি (সংখ্যাত্মক ভগ্নাংশের জন্য):
   - ধাপ ১: সূচক (numerator) ও নামাঙ্ক (denominator)–এর সাধারণ গুণক (GCD) বা সাধারণ ভগ্নাংশীয় গুণক নির্ণয় করো।  
   - ধাপ ২: উভয় অংশকে সেই GCD দিয়ে ভাগ করো। ফলাফল হবে লঘিষ্ঠ রূপ।  
   - উদাহরণ: [math]\frac{18}{24}[/math]–এ GCD = [math]6[/math], তাই [math]\frac{18}{24}=\frac{18÷6}{24÷6}=\frac{3}{4}[/math]।

3. বিকল্প পদ্ধতি (গুণনীয়ক বিভাজন/প্রাইম ফ্যাক্টরাইজেশন):
   - অংকগুলোকে প্রাইম গুণফলের রূপে ভাঙো এবং সাধারণ গুণফলগুলো বাদ দাও।  
   - উদাহরণ: [math]18=2×3×3,\quad 24=2×2×2×3[/math]; সাধারণ অংশ = [math]2×3[/math] => বাদ দিলে [math]\frac{3}{4}[/math]।

4. বীজগণিতীয় (বৈচিত্র্যময়) ভগ্নাংশের ক্ষেত্রে:
   - ধাপ ১: সংখ্যক ও নামাঙ্ককে ফ্যাক্টর করে সাধারণ ফ্যাক্টরগুলো বের করো।  
   - ধাপ ২: সাধারণ ফ্যাক্টরগুলো বাতিল (cancel) করো, কিন্তু যা বাতিল করো সেগুলো শূন্য হতে পারবে না—অর্থাৎ বাতিল করার পূর্বে সেই মানগুলোর জন্য শর্ত (যেমন [math]x≠0[/math]) উল্লেখ করো।  
   - উদাহরণ: [math]\frac{2x^2+4x}{4x}=\frac{2x(x+2)}{4x}[/math]. এখানে সাধারণ [math]2x[/math] বাতিল করলে পাওয়া যায় [math]\frac{x+2}{2}[/math] অথবা যদি সরাসরি [math]4x=2x×2[/math] ধরা হয়, আর যে কোনো ক্ষেত্রে শর্ত [math]x≠0[/math] রাখতে হবে।

5. বিশেষ বিষয়াদি ও যুক্তি:
   - নামাঙ্ক কখনোই শূন্য থাকতে পারে না; সরলীকরণের সময় কোন অংশ বাতিল করলে সেই অংশ শূন্য হলে সমীকরণের ডোমেইন প্রভাবিত হবে—এই শর্তগুলো স্পষ্ট করতে হবে।  
   - পুরো ভগ্নাংশকে নেগেটিভ করা হলে সাধারণত মাইনাস সাইন numerator বা বহিরাগতভাবে নেওয়া হয়: [math]\frac{-a}{b}=-\frac{a}{b}[/math]।  
   - সরলীকরণ গণনা সহজ করে, তুলনা করা সহজ করে এবং সমাধান বা পরিচিত রূপে ফল উপস্থাপন করে।

সংক্ষেপে: ভগ্নাংশ সরলীকরণে সাধারণ গুণক বা সাধারণ ফ্যাক্টরগুলো বাদ দিয়ে সংখ্যা/অভিব্যক্তিকে লঘিষ্ঠ রূপে আনা হয় যাতে গণনা ও বিশ্লেষণ সহজ হয় এবং ফলটি মানদণ্ডে থাকে।

Explanation:
এই উত্তরটি সঠিক কারণ ভগ্নাংশের লেখক (উপরে) ও নাম (নিচে) তাদের সর্বাধিক সাধারণ বিভাজক দিয়ে ভাগ করে ছোট করা হয়, যাতে কোনো সাধারণ গুণক না থেকে ভগ্নাংশটি লঘিষ্ঠ রূপে আসে। এর উদ্দেশ্য হল গণনা ও তুলনা সহজ করা এবং সমীকরণে বা ব্যবহারিক সমস্যা সমাধানে ভগ্নাংশকে সরল ও সুবিধাজনক করে তোলা।

Question:
বীজগণিতীয় ভগ্নাংশের বিয়োগের ক্ষেত্রে কীভাবে হরের লঘিষ্ঠ সাধারণ হর নির্ধারণ করা হয়?

Options:


Correct Answer: বিভিন্ন ভগ্নাংশগুলোর হরের লঘিষ্ট সাধারণ হর বের করে তারপর লবের বিয়োগফল নির্ধারণ করতে হয়।

Solution:
সমাধান:

উত্তরগত পদ্ধতি (স্পষ্ট ধাপগুলো):
1. প্রতিটি ভগ্নাংশের হরকে পূর্ণভাবে গুণনীয় করে ফেলুন (প্রয়োজনমতো ভগ্নাংশের হরগুলোকে পলিনোমিয়ালগুলোর গুণনীয় রূপে ভাঙ্গুন) — উদাহরণ: গুণনীয়করণ যেমন [math]x^2-1=(x-1)(x+1)[math] ইত্যাদি।
2. সব হরের বিভিন্ন প্রাইম (বা অপরিবর্তনীয়) গুণনীয়কগুলোর সর্বোচ্চ ঘাত নিন; সেই গুণনীয়কগুলোর গুণফলকে লঘিষ্ঠ সাধারণ হর (LCM) বলা হয়। অর্থাৎ প্রতিটি ভগ্নাংশের হরে উপস্থিত প্রতিটি ফ্যাক্টর LCM-এ সর্বোচ্চ পাওয়া ঘাত অনুযায়ী থাকবে।
3. প্রতিটি ভগ্নাংশকে LCM দিয়ে সমান হর করার জন্য প্রয়োজনীয় গুণনীয়কে সংখ্যক অংশে গুণ করুন (উপরি ও অধোভাগ উভয়ই)।
4. এখন ভগ্নাংশগুলো একই হরে থাকলে অংকের অংশগুলোকে বিয়োগ করুন এবং ফলটি সাধারণীকরণ করুন (সম্ভব হলে ভগ্নাংশ সংকুচিত করুন)।
5. অবশেষে ডোমেইন সীমাবদ্ধতা লিখুন (যেসব মান হরকে শূন্য করে তাদের বাদ দিন)।

সংক্ষেপে: প্রথমে প্রত্যেক হরকে ফ্যাক্টর করে LCM নির্ণয় করতে হবে; তারপর প্রতিটি ভগ্নাংশকে LCM এ আনিয়ে বিয়োগ করুন এবং সরলীকরণ করুন।

সংক্ষিপ্ত উদাহরণ (প্রাসঙ্গিকতা উপলক্ষে):
প্রশ্ন: [math]\frac{2}{x^2-1}-\frac{3}{x^2+x}[/math]

ধাপ 1: হরগুলো ফ্যাক্টর করা:
[math]x^2-1=(x-1)(x+1),\quad x^2+x=x(x+1)[/math]

ধাপ 2: LCM = [math]x(x-1)(x+1)[/math] (প্রতিটি ফ্যাক্টর সর্বোচ্চ একবার)

ধাপ 3: ভগ্নাংশগুলোকে LCM-এ আনুন:
প্রথমটিকে [math]x[/math] দ্বারা গুণ করলে হবে [math]\frac{2x}{x(x-1)(x+1)}[/math]।
দ্বিতীয়টিকে [math](x-1)[/math] দ্বারা গুণ করলে হবে [math]\frac{3(x-1)}{x(x-1)(x+1)}[/math]।

ধাপ 4: বিয়োগ করে সরলীকরণ:
[math]\frac{2x}{\cdots}-\frac{3(x-1)}{\cdots}=\frac{2x-3(x-1)}{x(x-1)(x+1)}=\frac{2x-3x+3}{x(x-1)(x+1)}=\frac{3-x}{x(x-1)(x+1)}[/math]

ধাপ 5: ডোমেইন সীমাবদ্ধতা:
[math]x
eq 0,\;1,\;-1[/math] (কারণ এগুলো হরকে শূন্য করে)

এভাবেই বীজগণিতীয় ভগ্নাংশের বিয়োগে হরের লঘিষ্ঠ সাধারণ হর নির্ণয় ও ব্যবহার করে সমাধান করা হয়।

Explanation:
কারণ ভগ্নাংশগুলোকে একই হরে (লঘিষ্ঠ সাধারণ হর বা LCM) এনে সারাসরি সংখ্যকগুলো বিয়োগ করা যায়; তাই প্রথমে সকল হরের লঘিষ্ট সাধারণ হর নির্ণয় করতে হয়। প্রতিটি ভগ্নাংশকে সেই হরে বিস্তৃত করে সংখ্যকগুলো বিয়োগ করে ফলকে সরল করা হয় — যেমন [math]\frac{a}{b}-\frac{c}{d}=\frac{a\cdot (LCM/b)-c\cdot (LCM/d)}{LCM}[/math].

Question:
বীজগণিতীয় ভগ্নাংশের লঘুকরণের প্রক্রিয়া কী?

Options:


Correct Answer: বীজগণিতীয় ভগ্নাংশের লঘুকরণের জন্য লব ও হরকে তাদের সাধারণ গুণনীয়ক দ্বারা ভাগ করা হয়।

Solution:
সমাধান:

বীজগণিতীয় ভগ্নাংশের লঘুকরণে মূল ধারণা হলো লব (নিউমেরেটর) ও হর (ডিনোমিনেটর)‑কে তাদের সাধারণ গুণনীয়ক (common factor) দ্বারা ভাগ করা। ধাপে ধাপে পদ্ধতি এবং যুক্তি:

ধাপ 1 — উভয় অংশকে গুণনীয়কে ভেঙে নেওয়া:
- লব ও হর উভয়কে সাধারনভাবে ফ্যাক্টরাইজ (গুণনীয়কে ভাঙ্গা) করতে হবে। উদাহরণস্বরূপ পলিনোমিয়াল হলে সাধারণ ফ্যাক্টরগুলো বের করা।

ধাপ 2 — সাধারণ গুণনীয়ক বাতিল করা:
- লব ও হর‑এ উপস্থিত যে কোনো সাধারণ গুণনীয়ক (যা শূন্য নয়) তা numerator ও denominator থেকে কেটে ফেলতে পারি। এটি ভগ্নাংশকে সহজ করে দেয়।

ধাপ 3 — নির্দিষ্টতা (ডোমেইন) উল্লেখ করা:
- ভগ্নাংশ লঘুকরণের পরেও মূল ভগ্নাংশের হরে যে মানগুলোকে শূন্য করে (অর্থাৎ মূল ডিনোমিনেটরের মূলগুলো) সেগুলোকে নিষিদ্ধ মান হিসেবে রাখতে হবে। অর্থাৎ সরলীকৃত রূপে সেই মানগুলো থেকে বাদ দেয়া উচিত, কারণ আসল ভগ্নাংশ ঐ মানগুলোতে অপ্রযোজ্য।

সংক্ষিপ্ত উদাহরণ (প্রাসঙ্গিক হলে):
- ধরা যাক ভগ্নাংশ [math]\dfrac{x^{2}-9}{x^{2}-x-6}[/math]।
  Step 1: ফ্যাক্টরাইজ করুন:
  [math]x^{2}-9=(x-3)(x+3)[/math], [math]x^{2}-x-6=(x-3)(x+2)[/math].
  Step 2: সাধারণ গুণনীয়ক (x-3) বাতিল করে পাওয়া যায়:
  [math]\dfrac{x^{2}-9}{x^{2}-x-6}=\dfrac{(x-3)(x+3)}{(x-3)(x+2)}=\dfrac{x+3}{x+2}[/math].
  Step 3: কিন্তু মূল ডিনোমিনেটর থেকে নিষিদ্ধ মানগুলো অর্জিত হয়: [math]x
eq 3[/math] এবং [math]x
eq -2[/math]. সুতরাং সরলীকৃত রূপ [math]\dfrac{x+3}{x+2}[/math] হলেও [math]x=3[/math] এবং [math]x=-2[/math] নিষিদ্ধ।

প্রধান নীতি: সবসময় লব ও হর‑কে ফ্যাক্টর করে সাধারণ গুণনীয়কগুলো কেটে দিন, এবং মূল ডিনোমিনেটরের শূন্যস্থলগুলো উল্লেখ করতে ভুলবেন না।

Explanation:
কারণ লব ও হরকে তাদের সাধারণ গুণনীয়ক দ্বারা ভাগ করলে ভগ্নাংশের মান অপরিবর্তিত থেকে সংখ্যা ছোট হয়, ফলে ভগ্নাংশ সহজ রূপে প্রকাশ পায়। উদাহরণস্বরূপ, [math]\frac{6}{8}[/math]–এর সাধারণ গুণনীয়ক 2 দ্বারা ভাগ করলে পাওয়া যায় [math]\frac{3}{4}[/math]।

Question:
বীজগণিতীয় ভগ্নাংশের যোগ এবং বিয়োগ কিভাবে সম্পন্ন হয়?

Options:


Correct Answer: বীজগণিতীয় ভগ্নাংশের যোগ ও বিয়োগ সাধারণ হরবিশিষ্ট করার পর সম্পন্ন হয়।

Solution:
সমাধান:

সংক্ষিপ্ত পদ্ধতি:
1. প্রথমে প্রতিটি ভগ্নাংশের হরগুলোকে দেখে সাধারণ হর (LCD —least common denominator) নির্ণয় করুন (প্রয়োজন হলে হরগুলোকে গুণনীয়কের রূপে ভাঙুন এবং LCM নিন)।  
2. প্রতিটি ভগ্নাংশকে সেই সাধারণ হরে নিয়ে আসার জন্য উপরের এবং নীচের অংশ যথাযথভাবে গুণ করুন।  
3. একই হরের ভগ্নাংশগুলোর উপরা অংশ যোগ বা বিয়োগ করুন।  
4. ফলাফল সোজা করে লিখুন এবং সম্ভব হলে উপরা ও নীচে সাধারণ গুণক বাতিল করে সরলীকরণ করুন। ধ্যান রাখবেন—বিহিত মানগুলো (denominator) শূন্য হতে পারে না; সেই মানগুলো বাদ দেবেন।

প্রযোজ্য ধারণা ও নিয়ম:
- ভগ্নাংশ যোগ/বিয়োগ করার নিয়ম একই নগদ যেমন: [math]\dfrac{a}{d} \pm \dfrac{b}{d} = \dfrac{a \pm b}{d}[/math] কিন্তু ভিন্ন হর হলে প্রথমে সাধারণ হর করা প্রয়োজন।  
- সাধারণ হর পাওয়ার জন্য হরগুলোর LCM ব্যবহার করুন বা লক্ষ করুন যে কিছু ক্ষেত্রে হরগুলোকে গুণ করলে সরাসরি সাধারণ হর হতে পারে।  
- ফলাফল সর্বদা সহজীকরণ (ফ্যাক্টর করে সাধারণ গুণক বাতিল) করুন।  
- ডোমেইন শর্ত: যে কোন ধাপেই যেসব মান হরকে শূন্য করে তা প্রতিপাদ্য থেকে বাদ দিন।

উদাহরণ (যোগ):
ধরা যাক যোগ করুন [math]\dfrac{2}{x+1} + \dfrac{3}{x-1}[/math]  
Step 1: হরগুলো ভিন্ন; সাধারণ হর হবে [math](x+1)(x-1)=x^2-1[/math].  
Step 2: প্রতিটি ভগ্নাংশকে সাধারণ হরে আনুন:  
[math]\dfrac{2}{x+1}=\dfrac{2(x-1)}{(x+1)(x-1)}[/math],  [math]\dfrac{3}{x-1}=\dfrac{3(x+1)}{(x-1)(x+1)}[/math].  
Step 3: উপরা যোগ করুন:  
[math]\dfrac{2(x-1)+3(x+1)}{(x+1)(x-1)}=\dfrac{2x-2+3x+3}{x^2-1}=\dfrac{5x+1}{x^2-1}[/math].  
Step 4: সম্ভব হলে সরলীকরণ; এখানে আর সরলীকরণ নেই। শর্ত: [math]x
eq \pm 1[/math]।

উদাহরণ (বিয়োগ):
ধরা যাক [math]\dfrac{x}{x^2-1}-\dfrac{1}{x+1}[/math]। নোট করুন [math]x^2-1=(x-1)(x+1)[/math].  
Step 1: সাধারণ হর [math](x-1)(x+1)[/math].  
Step 2: প্রথম ভগ্নাংশই ইতিমধ্যেই সেই হরে আছে: [math]\dfrac{x}{(x-1)(x+1)}[/math]; দ্বিতীয়টিকে আনুন: [math]\dfrac{1}{x+1}=\dfrac{x-1}{(x+1)(x-1)}[/math].  
Step 3: বিয়োগ করুন:  
[math]\dfrac{x-(x-1)}{(x-1)(x+1)}=\dfrac{x-x+1}{(x-1)(x+1)}=\dfrac{1}{x^2-1}[/math].  
Step 4: সরলীকরণ সম্পন্ন; শর্ত: [math]x
eq \pm 1[/math]।

সংক্ষেপে: বীজগণিতীয় ভগ্নাংশের যোগ ও বিয়োগ সাধারণ হরবিশিষ্ট করে (LCD) উপরা যোগ/বিয়োগ করার মাধ্যমে করা হয়, পরে দরকারে সূচকীয় সরলীকরণ ও ডোমেইন শর্ত বিবেচনা করতে হয়।

Explanation:
বীজগণিতীয় ভগ্নাংশ সরাসরি যোগ বা বিয়োগ করা যায় না কারণ তাদের হর ভিন্ন হলে তারা ভিন্ন "কাঠামোর অংশ", তাই প্রথমে একটি সাধারণ হার (common denominator) করা লাগে। সাধারণত হরগুলোর সর্বনিম্ন সাধারণ গুণিতক নিয়ে প্রতিটি ভগ্নাংশকে উপযুক্তভাবে প্রসারিত করে একই হার বানিয়ে নেয়া হয়, তারপর উর্ধ্বাংশগুলো যোগ বা বিয়োগ করে ফলকে সরল করা হয়। উদাহরণ: [math]\frac{a}{b}+\frac{c}{d}=\frac{ad+bc}{bd}[/math]
