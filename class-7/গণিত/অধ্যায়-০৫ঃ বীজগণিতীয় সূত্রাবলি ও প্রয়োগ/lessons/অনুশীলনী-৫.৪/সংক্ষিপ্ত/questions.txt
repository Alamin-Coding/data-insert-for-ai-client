Question:
[math](x + 2y - 4z)[/math] এর বর্গ নির্ণয় কর ।

Options:


Correct Answer: 

Solution:
[math](x + 2y - 4z)^2[/math]
 [math]= x^2 + (2y)^2 + (-4z)^2 + 2 \cdot x \cdot 2y + 2 \cdot x \cdot (-4z) + 2 \cdot 2y \cdot (-4z)[/math]
 [math]= x^2 + 4y^2 + 16z^2 + 4xy - 8xz - 16yz[/math]
 [math]= x^2 + 4y^2 + 16z^2 + 4xy - 8xz - 16yz[/math]


Explanation:
ব্যাখ্যা: একটি তিন-ঋতু যোগের বর্গের সূত্র [math](a+b+c)^2 = a^2 + b^2 + c^2 + 2ab + 2ac + 2bc[/math] ব্যবহার করা যায়; এখানে [math]a=x,\; b=2y,\; c=-4z[/math]। ফলে [math](x+2y-4z)^2 = x^2 + 4y^2 + 16z^2 + 4xy - 8xz - 16yz[/math]।

Question:
[math]a + b = x[/math], [math]a - b = x - 3[/math] এবং [math]ab = \frac{45}{4}[/math] হলে, [math]x[/math] এর মান নির্ণয় কর ।

Options:


Correct Answer: 

Solution:
[math]4ab = (a + b)^2 - (a - b)^2[/math]
 [math]4 \cdot \frac{45}{4} = x^2 - (x - 3)^2[/math]
 [math]45 = x^2 - (x^2 - 6x + 9)[/math]
 [math]45 = x^2 - x^2 + 6x - 9 = 6x - 9[/math]
 [math]6x = 45 + 9 = 54[/math]
 [math]x = \frac{54}{6} = 9[/math]


Explanation:
দুটি সমীকরণ যোগ ও বিয়োগ করলে পাওয়া যায় [math]2a=2x-3[/math] এবং [math]2b=3[/math], তাই [math]a=x-\frac{3}{2}[/math], [math]b=\frac{3}{2}[/math].  
[math]ab=\left(x-\frac{3}{2}\right)\frac{3}{2}=\frac{45}{4}[/math] হলে [math]x-\frac{3}{2}=\frac{45}{4}\cdot\frac{2}{3}=\frac{15}{2}[/math].  
সুতরাং [math]x=\frac{15}{2}+\frac{3}{2}=9[/math]।

Question:

Options:


Correct Answer: 

Solution:
[math]m^2(m + 1) = 15[/math]
 [math]m^2(m - 1) = 9[/math]
 [math](m^2(m + 1))^2 - (m^2(m - 1))^2 = 15^2 - 9^2[/math]

 [math]4m^3m^2 = 144[/math]
 [math]4m^5 = 144[/math]

Explanation:
যোগ করলে: [math]m^3+m^3=15+9\Rightarrow 2m^3=24[/math], তাই [math]m^3=12[/math]। বিয়োগ করলে: [math]m^2+m^2=15-9\Rightarrow 2m^2=6[/math], তাই [math]m^2=3[/math]। অতএব [math]m^5=m^3\cdot m^2=12\cdot3=36[/math]।

Question:
[math]\frac{a}{b} + \frac{b}{a} = 2[/math] হলে, দেখাও যে [math]a = b[/math]

Options:


Correct Answer: 

Solution:
[math]\frac{a}{b} + \frac{b}{a} = 2[/math]
 [math]\Rightarrow \frac{a^2 + b^2}{ab} = 2[/math]
 [math]\Rightarrow a^2 + b^2 = 2ab[/math]
 [math]\Rightarrow a^2 - 2ab + b^2 = 0[/math]
 [math]\Rightarrow (a - b)^2 = 0[/math]
 [math]\Rightarrow a - b = 0[/math]
 [math]\therefore a = b[/math]
 (প্রমাণিত)

Explanation:
সমাধানে, [math]\frac{a}{b}+\frac{b}{a}=2[/math] থেকে (এখানে [math]ab\neq0[/math] ধরে) পাই [math]\frac{a^2+b^2}{ab}=2[/math], তখন [math]a^2+b^2=2ab[/math]. তাই [math](a-b)^2=a^2-2ab+b^2=0[/math], ফলে [math]a-b=0[/math] এবং [math]a=b[/math].

Question:
[math]3p + \frac{1}{2p} = ৭[/math] হলে, [math]9p^2 + \frac{1}{4p^2}[/math] এর মান নির্ণয় কর

Options:


Correct Answer: 

Solution:
[math]3p + \frac{1}{2p} = ৭[/math]
 [math]9p^2 + \frac{1}{4p^2} = (3p)^2 + \left(\frac{1}{2p}\right)^2[/math]
 [math]= \left(3p + \frac{1}{2p}\right)^2 - 2 \cdot 3p \cdot \frac{1}{2p}[/math]
 [math]= 7^2 - 3 = 49 - 3 = 46[/math]


Explanation:
ব্যাখ্যা: [math]3p+\frac{1}{2p}=7[/math] হলে উভয় পাশে বর্গ করলে [math](3p+\frac{1}{2p})^2=49[/math]. বর্গবিস্তারে [math]9p^2+2\cdot3p\cdot\frac{1}{2p}+\frac{1}{4p^2}=9p^2+3+\frac{1}{4p^2}[/math], তাই [math]9p^2+\frac{1}{4p^2}=49-3=46[/math].

 [math]\left(a + \frac{1}{2b}\right)^2 = ৩^2 = ৯[/math]

 [math]a^2 + \frac{a}{b} + \frac{1}{4b^2} = ৯[/math]
উভয় পক্ষকে [math]4b^2[/math]
 দ্বারা গুণ করি:[math]4a^2b^2 + 4ab + 1 = 36b^2[/math]
অতএব, [math]4a^2b^2 - 36b^2 + 4ab + 1 = 0[/math]

Question:
[math]x + \frac{1}{x} = 5[/math] হলে, [math]x^2 + \frac{1}{x^2}[/math] এর মান নির্ণয় কর ।

Options:


Correct Answer: 

Solution:
[math]x + \frac{1}{x} = 5[/math]
 [math]\left(x + \frac{1}{x}\right)^2 = 5^2 = 25[/math]
 [math]x^2 + 2 + \frac{1}{x^2} = 25[/math]
 [math]x^2 + \frac{1}{x^2} = 25 - 2 = 23[/math]


Explanation:
দুটি পদগুণী সূত্র ব্যবহার করে সহজে পাওয়া যায়: কিউব না করে বর্গ করলে [math](x+\frac{1}{x})^2=x^2+\frac{1}{x^2}+2[/math]। এখানে বামপাশের মান [math]5^2=25[/math] হওয়ায় [math]25=x^2+\frac{1}{x^2}+2[/math], ফলে [math]x^2+\frac{1}{x^2}=23[/math]।

Solution:

 [math]a^2 - 2 + \frac{1}{a^2} = 16[/math]

Explanation:

Question:

Options:


Correct Answer: 

Solution:
[math]a + b = 15[/math]
, [math]a^2 + b^2 = 113[/math]
 [math]a^2 + b^2 = (a + b)^2 - 2ab[/math]
 [math]113 = (15)^2 - 2ab[/math]
 [math]113 = 225 - 2ab[/math]
 [math]2ab = 225 - 113 = 112[/math]
 [math]\therefore ab = \frac{112}{2} = 56[/math]


Explanation:
ব্যাখ্যা: আমরা জানি [math](a+b)^2 = a^2 + b^2 + 2ab[/math]. এখানে [math](15)^2 = 225 = 113 + 2ab[/math] হওয়ায় [math]2ab = 112[/math] এবং ফলে [math]ab = 56[/math].

 [math]= (-1)^2 = 1[/math]


Explanation:

[math]9a^2+(3a+1)^2-6a(3a+1)=(3a)^2+(3a+1)^2-2(3a)(3a+1)=(3a-(3a+1))^2=(-1)^2=1[/math].
