Question:

Options:


Correct Answer: 

Solution:
[math]a + b = 15[/math]
, [math]a^2 + b^2 = 113[/math]
 [math]a^2 + b^2 = (a + b)^2 - 2ab[/math]
 [math]113 = (15)^2 - 2ab[/math]
 [math]113 = 225 - 2ab[/math]
 [math]2ab = 225 - 113 = 112[/math]
 [math]\therefore ab = \frac{112}{2} = 56[/math]


Explanation:
ব্যাখ্যা: [math](a+b)^2 = a^2 + 2ab + b^2[/math], তাই [math]15^2 = 113 + 2ab[/math]. সেখান থেকে [math]2ab = 225 - 113 = 112[/math] এবং ফলে [math]ab = 56[/math].

Solution:

 [math]\left(y + \frac{1}{2y}\right)^2 = 6^2 = 36[/math]

সমাধান: দুইপাশকে বর্গ করলে [math](y+\frac{1}{2y})^2 = y^2+\frac{1}{4y^2}+2\cdot y\cdot\frac{1}{2y}[/math] হয়, এবং [math]2\cdot y\cdot\frac{1}{2y}=1[/math]. অতএব [math]y^2+\frac{1}{4y^2}=6^2-1=35[/math].

 [math]\therefore a = b[/math]
 (প্রমাণিত)

Explanation:
প্রমাণ: [math]\frac{a}{b}+\frac{b}{a}=\frac{a^2+b^2}{ab}=2[/math] হওয়ায় [math]a^2+b^2=2ab[/math], যেটি থেকে [math]a^2-2ab+b^2=0\Rightarrow (a-b)^2=0[/math] হয়, ফলে [math]a=b[/math]. (এখানে ভাগ করার জন্য [math]a,b\neq0[/math] ধরতে হবে, কারণ ভাগ শূন্যে অযোগ্য।)

[math]x + \frac{1}{x} = 4[/math] হলে, [math]x^2 + \frac{1}{x^2}[/math] এর মান নির্ণয় কর

Options:


Correct Answer: 

Solution:
[math](x + \frac{1}{x})^2 = 4^2 = 16[/math]
 [math]\Rightarrow x^2 + \frac{1}{x^2} + 2 = 16[/math]
 [math]\Rightarrow x^2 + \frac{1}{x^2} = 16 - 2 = 14[/math]


Explanation:
সমাধান: দুই পাশে বর্গ করলে [math](x+\frac{1}{x})^2 = x^2 + 2 + \frac{1}{x^2} = 16[/math] হয়। তাই [math]x^2 + \frac{1}{x^2} = 16 - 2 = 14[/math]।

[math]a + b = 6[/math]
, [math]a - b = 4[/math]
 [math]3ab = 4ab \times \frac{3}{4} = \left((a + b)^2 - (a - b)^2\right) \times \frac{3}{4}[/math]

 [math]= (36 - 16) \times \frac{3}{4} = 20 \times \frac{3}{4} = 15[/math]

ব্যাখ্যা: সমীকরণগুলো যোগ করলে [math]2a=10\Rightarrow a=5[/math] পাওয়া যায়, এরপর [math]b=6-a=1[/math]. তাই [math]3ab=3\times5\times1=15[/math].
