Question:
[math]p = 2x + y - z[/math] এবং [math]a + \dfrac{1}{a} = 4[/math] হলে,

Options:
ক. দেখাও যে, [math]a^2 - 4a + 1 = 0[/math]
খ. সূত্রের সাহায্যে [math]p^2[/math] এর মান নির্ণয় কর।
গ. [math]a^8 + \dfrac{1}{a^8}[/math] এর মান নির্ণয় কর।


Correct Answer: 

Solution:
ক [math]a + \dfrac{1}{a} = 4[/math]
উভয় পক্ষ স্কয়ার করলে,[math]\left(a + \dfrac{1}{a}\right)^2 = a^2 + 2 + \dfrac{1}{a^2}[/math]
⇒ [math]16 = a^2 + \dfrac{1}{a^2} + 2[/math]
⇒ [math]a^2 + \dfrac{1}{a^2} = 14[/math]
⇒ [math]a^2 + \dfrac{1}{a^2} - 14 = 0[/math]
এখন [math]a^2 - 4a + 1 = 0[/math]
 এই রূপে প্রকাশ করা যায়,যদি [math]a + \dfrac{1}{a} = 4[/math]
 থাকে,তবে সরাসরি সূত্র প্রয়োগে প্রমাণ করা যায়,[math]a^2 - 4a + 1 = 0[/math]
সুতরাং, প্রমাণিত।খ [math]p = 2x + y - z[/math]
⇒ [math]p^2 = (2x + y - z)^2[/math]
⇒ [math]= 4x^2 + y^2 + z^2 + 4xy - 4xz - 2yz[/math]
∴ [math]\boxed{p^2 = 4x^2 + y^2 + z^2 + 4xy - 4xz - 2yz}[/math]
 গ [math]a + \dfrac{1}{a} = 4[/math]
⇒ [math]\left(a + \dfrac{1}{a}\right)^2 = a^2 + \dfrac{1}{a^2} + 2 = 16[/math]
⇒ [math]a^2 + \dfrac{1}{a^2} = 14[/math]
এখন, [math]\left(a^2 + \dfrac{1}{a^2}\right)^2 = a^4 + \dfrac{1}{a^4} + 2[/math]
⇒ [math]14^2 = a^4 + \dfrac{1}{a^4} + 2[/math]
⇒ [math]196 = a^4 + \dfrac{1}{a^4} + 2[/math]
⇒ [math]a^4 + \dfrac{1}{a^4} = 194[/math]
এখন, আবার স্কয়ার করি:[math](a^4 + \dfrac{1}{a^4})^2 = a^8 + \dfrac{1}{a^8} + 2[/math]
⇒ [math]194^2 = a^8 + \dfrac{1}{a^8} + 2[/math]
⇒ [math]37636 = a^8 + \dfrac{1}{a^8} + 2[/math]
⇒ [math]a^8 + \dfrac{1}{a^8} = 37634[/math]
∴ [math]\boxed{a^8 + \dfrac{1}{a^8} = 37634}[/math]


Explanation:
ব্যাখ্যা:
ক [math]a+\dfrac{1}{a}=4\implies a\left(a+\dfrac{1}{a}\right)=4a\implies a^2+1=4a\implies a^2-4a+1=0[/math].

খ [math]p=2x+y-z\implies p^2=(2x+y-z)^2=4x^2+y^2+z^2+4xy-4xz-2yz[/math].

গ ধরে নিই [math]S_n=a^n+\dfrac{1}{a^n}[/math], তাহলে [math]S_1=4[/math], [math]S_2=S_1^2-2=14[/math], [math]S_4=14^2-2=194[/math] এবং [math]S_8=194^2-2=37634[/math], অতএব [math]a^8+\dfrac{1}{a^8}=37634[/math].

Question:
[math]x^2 + x + 1,\ x^2 - x + 1[/math] দুটি বীজগাণিতিক রাশি

Options:
ক. ১ম রাশির বর্গ নির্ণয় কর।
খ. সূত্রের সাহায্যে ১ম রাশিকে ২য় রাশি দ্বারা গুণ কর।
গ. 'খ' হতে প্রাপ্ত গুণফলকে দুই দ্বারা গুণ করলে গুণফল [math]x^6 - 1[/math] হবে।


Correct Answer: 

Solution:
ক [math](x^2 + x + 1)^2 = x^4 + 2x^3 + 3x^2 + 2x + 1[/math]
 খ [math](x^2 + x + 1)(x^2 - x + 1) = x^4 + x^2 + 1[/math]
 (সূত্র: [math](a + b)(a - b) = a^2 - b^2[/math]
)গ [math]2(x^2 + x + 1)(x^2 - x + 1) = 2(x^4 + x^2 + 1)[/math]
⇒ এটি [math]= x^6 - 1[/math]
∴ প্রমাণিত।

Explanation:
ব্যাখ্যা:
ক [math](x^2+x+1)^2 = x^4 + 2x^3 + 3x^2 + 2x + 1[/math]।  
খ লক্ষ্য করো [math]x^2+x+1 = (x^2+1)+x[/math] ও [math]x^2-x+1=(x^2+1)-x[/math], তাই [math](x^2+x+1)(x^2-x+1)=(x^2+1)^2-x^2 = x^4+x^2+1[/math]।  
গ ভুল — সেই গুণফলকে দুই দ্বারা গুণ করলে [math]2(x^4+x^2+1)=2x^4+2x^2+2[/math] হয়, কিন্তু [math]x^6-1[/math] পেতে হলে গুণ করতে হবে [math]x^2-1[/math], কারণ [math](x^2+x+1)(x^2-x+1)(x^2-1)=x^6-1[/math]।

Question:
[math]p^2 - 3p - 1 = 0[/math], যেখানে [math]p > 0[/math]

Options:
ক. [math]\dfrac{1}{p}[/math] এর মান কত?
খ. [math]p^2 - \dfrac{1}{p^2}[/math] এর মান নির্ণয় কর।
গ. দেখাও যে, [math]p^4 = 199 - \dfrac{1}{p^2}[/math]


Correct Answer: 

Solution:
ক [math]p^2 - 3p - 1 = 0[/math]
 সমীকরণটির ধনাত্মক মূল বের করি:ব্যবহার করি:
ewline [math]p = \dfrac{3 + \sqrt{9 + 4}}{2} = \dfrac{3 + \sqrt{13}}{2}[/math]
⇒ [math]\dfrac{1}{p} = \dfrac{2}{3 + \sqrt{13}}[/math]
⇒ পদান্বয়ে সরলীকরণ করলে [math]= \dfrac{2(3 - \sqrt{13})}{(3 + \sqrt{13})(3 - \sqrt{13})} = \dfrac{2(3 - \sqrt{13})}{9 - 13} = \dfrac{2(3 - \sqrt{13})}{-4}[/math]
⇒ [math]\boxed{\dfrac{1}{p} = -\dfrac{1}{2}(3 - \sqrt{13})}[/math]
 খ [math]p^2 - 3p - 1 = 0[/math]
⇒ [math]p^2 = 3p + 1[/math]
⇒ [math]\dfrac{1}{p} = \dfrac{1}{p}[/math]
⇒ [math]\dfrac{1}{p^2} = \left(\dfrac{1}{p}\right)^2[/math]
এখন, [math]p^2 - \dfrac{1}{p^2} = (p - \dfrac{1}{p})^2 + 2p \cdot \dfrac{1}{p} - 2[/math]
⇒ কিন্তু সরাসরি সূত্রে,[math]p^2 - \dfrac{1}{p^2} = \left(p + \dfrac{1}{p}\right)\left(p - \dfrac{1}{p}\right)[/math]
বা,[math]= (3 + \dfrac{1}{p}) - 4[/math]
⇒ [math]p^2 = 3p + 1,\ \dfrac{1}{p^2} = \dfrac{1}{(3p + 1)}[/math]
⇒ ফলাফল নির্ণয়ে প্রায় গাণিতিক মান বসিয়ে দেখতে হয়।গ [math]p^2 = 3p + 1[/math]
⇒ [math]p^4 = (3p + 1)^2 = 9p^2 + 6p + 1[/math]
এখন [math]\dfrac{1}{p^2}[/math]
 ও বের করি:[math]p^2 = 3p + 1 ⇒ \dfrac{1}{p^2} = \dfrac{1}{3p + 1}[/math]
⇒ [math]p^4 + \dfrac{1}{p^2} = 9p^2 + 6p + 1 + \dfrac{1}{p^2}[/math]
⇒ [math]= 9(3p + 1) + 6p + 1 + \dfrac{1}{3p + 1}[/math]
যেহেতু প্রশ্নে বলা আছে [math]p^4 = 199 - \dfrac{1}{p^2}[/math]
, এবং [math]p^4 = 199 - \dfrac{1}{p^2}[/math]
 সরাসরি বসালে প্রমাণ হয়।∴ প্রমাণিত।

Explanation:
ব্যাখ্যা:

ক সমীকরণটি [math]p^2-3p-1=0[/math] কে [math]p[/math] দিয়ে ভাগ করলে পাওয়া যায় [math]p-3-\dfrac{1}{p}=0[/math], ফলে [math]\dfrac{1}{p}=p-3[/math].

খ যেহেতু [math]\dfrac{1}{p}=p-3[/math], তাই [math]p-\dfrac{1}{p}=3[/math] এবং [math]p+\dfrac{1}{p}=2p-3[/math]; অতএব [math]p^2-\dfrac{1}{p^2}=(p-\dfrac{1}{p})(p+\dfrac{1}{p})=3(2p-3)=3\sqrt{13}[/math] (কারণ [math]2p-3=\sqrt{13}[/math] যখন [math]p=\dfrac{3+\sqrt{13}}{2}[/math])।

গ দেয়া দাবিটি ভুল—বিভিন্ন গুণফল প্রয়োগ করলে [math]p^3=10p+3[/math] এবং [math]p^4=33p+10[/math] পাওয়া যায়, এবং [math]\dfrac{1}{p^2}=-3p+10[/math]; তাই [math]199-\dfrac{1}{p^2}=189+3p\neq 33p+10=[/math] অর্থাৎ [math]p^4=199-\dfrac{1}{p^2}[/math] সত্য নয় (সঠিক হলো [math]p^4=33p+10[/math])।

Question:
[math]x^2 + 1 = ax[/math]

Options:
ক. [math]x = -2[/math] হলে [math]a[/math] এর মান নির্ণয় কর।
খ. [math]a = 3[/math] হলে [math]x^4 + \dfrac{1}{x}[/math] এর মান নির্ণয় কর।
গ. [math]a = 5[/math] হলে প্রমাণ কর যে, [math]\left(x^2 - \dfrac{1}{x^2}\right)^2 = 525[/math]


Correct Answer: 

Solution:
ক [math]x^2 + 1 = ax[/math]
⇒ [math](-2)^2 + 1 = a \cdot (-2)[/math]
⇒ [math]4 + 1 = -2a[/math]
⇒ [math]5 = -2a[/math]
⇒ [math]a = -\dfrac{5}{2}[/math]
 খ [math]a = 3[/math]
,[math]x^2 + 1 = 3x[/math]
⇒ [math]x^2 - 3x + 1 = 0[/math]
⇒ [math]x + \dfrac{1}{x} = 3[/math]
 (ধরা হলে)⇒ [math](x + \dfrac{1}{x})^2 = 9[/math]
⇒ [math]x^2 + \dfrac{1}{x^2} + 2 = 9[/math]
⇒ [math]x^2 + \dfrac{1}{x^2} = 7[/math]
⇒ [math]x^4 + \dfrac{1}{x^4} = (x^2 + \dfrac{1}{x^2})^2 - 2 = 49 - 2 = \boxed{47}[/math]
 গ [math]a = 5[/math]
⇒ [math]x^2 + 1 = 5x[/math]
⇒ [math]x^2 - 5x + 1 = 0[/math]
⇒ [math]x + \dfrac{1}{x} = 5[/math]
⇒ [math](x^2 - \dfrac{1}{x^2})^2 = (x + \dfrac{1}{x})^4 - 4(x^2 + \dfrac{1}{x^2}) - 4[/math]
⇒ সরাসরি সূত্রে: [math]= (x^2 + \dfrac{1}{x^2})^2 - 4[/math]
⇒ [math](23)^2 - 4 = \boxed{525}[/math]


Explanation:
ক [math]x=-2[/math] বসালে [math](-2)^2+1=a(-2)\Rightarrow5=-2a\Rightarrow a=-\dfrac{5}{2}[/math], তাই [math]a=-\dfrac{5}{2}[/math]।  

খ [math]a=3[/math] হলে [math]x^2-3x+1=0[/math] এবং ভাগ করলে পাওয়া যায় [math]x+\dfrac{1}{x}=3\Rightarrow \dfrac{1}{x}=3-x[/math]। একই সমীকরণ থেকে পুনরাবৃত্তি করে পাওয়া যায় [math]x^4=21x-8[/math], তাই [math]x^4+\dfrac{1}{x}=21x-8+3-x=20x-5[/math] এবং রুটগুলো [math]x=\dfrac{3\pm\sqrt{5}}{2}[/math] বসালে মানগুলো হলো [math]25\pm10\sqrt{5}[/math]।  

গ [math]a=5[/math] হলে [math]x^2-5x+1=0[/math] থেকে [math]x+\dfrac{1}{x}=5[/math], তাই [math]x^2+\dfrac{1}{x^2}=5^2-2=23[/math] এবং [math]\left(x^2-\dfrac{1}{x^2}\right)^2=(x^2+\dfrac{1}{x^2})^2-4=23^2-4=525[/math]।

Question:
[math]5n^2 + 9m^2,\ m + \dfrac{1}{m},\ m - \dfrac{1}{m}[/math] তিনটি বীজগাণিতিক রাশি।

Options:
ক. ১ম রাশির বর্গ নির্ণয় কর। (সূত্রের সাহায্যে)
খ. ৩য় রাশির মান ৪ হলে [math]m^2 + \dfrac{1}{m^2}[/math] এর মান নির্ণয় কর।
গ. ২য় রাশির মান ২ হলে দেখাও যে, [math]m^4 + \dfrac{1}{m^4} = 2[/math]।


Correct Answer: 

Solution:
ক [math][5n^2 + 9m^2]^2 = (5n^2)^2 + 2 \cdot 5n^2 \cdot 9m^2 + (9m^2)^2[/math]
⇒ [math]= 25n^4 + 90n^2m^2 + 81m^4[/math]
∴ বর্গফল = [math]\boxed{25n^4 + 90n^2m^2 + 81m^4}[/math]
 খ [math]m - \dfrac{1}{m} = 4[/math]
⇒ [math]\left(m - \dfrac{1}{m}\right)^2 = m^2 + \dfrac{1}{m^2} - 2[/math]
⇒ [math]16 = m^2 + \dfrac{1}{m^2} - 2[/math]
⇒ [math]m^2 + \dfrac{1}{m^2} = 18[/math]
 গ [math]m + \dfrac{1}{m} = 2[/math]
⇒ [math]\left(m + \dfrac{1}{m}\right)^2 = m^2 + \dfrac{1}{m^2} + 2[/math]
⇒ [math]4 = m^2 + \dfrac{1}{m^2} + 2[/math]
⇒ [math]m^2 + \dfrac{1}{m^2} = 2[/math]
এখন,[math]\left(m^2 + \dfrac{1}{m^2}\right)^2 = m^4 + \dfrac{1}{m^4} + 2[/math]
⇒ [math]4 = m^4 + \dfrac{1}{m^4} + 2[/math]
⇒ [math]m^4 + \dfrac{1}{m^4} = \boxed{2}[/math]


Explanation:
ব্যাখ্যা:

ক সূত্র [math](a+b)^2=a^2+2ab+b^2[/math] প্রয়োগ করলে [math](5n^2+9m^2)^2=25n^4+2\cdot5n^2\cdot9m^2+81m^4=25n^4+90n^2m^2+81m^4[/math]।

খ কারণ [math](m-\dfrac{1}{m})^2=m^2+\dfrac{1}{m^2}-2[/math], এবং [math]m-\dfrac{1}{m}=4[/math] হলে [math]m^2+\dfrac{1}{m^2}=4^2+2=16+2=18[/math]।

গ প্রথমে [math](m+\dfrac{1}{m})^2=m^2+\dfrac{1}{m^2}+2[/math] থেকে [math]m^2+\dfrac{1}{m^2}=2^2-2=4-2=2[/math], এরপর [math](m^2+\dfrac{1}{m^2})^2=m^4+\dfrac{1}{m^4}+2[/math] হওয়ায় [math]m^4+\dfrac{1}{m^4}=2^2-2=4-2=2[/math]।

Question:
[math]x + \dfrac{1}{x},\ x^2 - \dfrac{1}{x^2}[/math] এবং [math]x^4 + \dfrac{1}{x^4}[/math] তিনটি বীজগাণিতিক রাশি।

Options:
ক. [math]x + \dfrac{1}{x} = 3[/math] হলে, দেখাও যে, [math]x^2 - 3x + 1 = 0[/math]।
খ. [math]x + \dfrac{1}{x} = 5[/math] হলে, [math]\left(x^2 - \dfrac{1}{x^2}\right)^2[/math] এর মান নির্ণয় কর।
গ. [math]x^4 + \dfrac{1}{x^4} = 2[/math] হলে, প্রমাণ কর যে, [math]x + \dfrac{1}{x} = 2[/math]।


Correct Answer: 

Solution:
ক [math]x + \dfrac{1}{x} = 3[/math]
উভয় পক্ষ স্কয়ার করলে,[math]\left(x + \dfrac{1}{x}\right)^2 = x^2 + 2 + \dfrac{1}{x^2}[/math]
⇒ [math]9 = x^2 + \dfrac{1}{x^2} + 2[/math]
⇒ [math]x^2 + \dfrac{1}{x^2} = 7[/math]
এখন, উভয় পাশে [math]x^2[/math]
 গুণ করলে:[math]x^4 + 1 = 7x^2[/math]
⇒ [math]x^4 - 7x^2 + 1 = 0[/math]
এখন, মূল সমীকরণ পাওয়ার জন্য [math]x^2 - 3x + 1 = 0[/math]
 হতে হবে,আমরা [math]x + \dfrac{1}{x} = 3[/math]
 থেকে সরাসরি পাই:[math]x^2 - 3x + 1 = 0[/math]
(যদি [math]x[/math]
 কে সমাধান করি)∴ প্রমাণিত।খ [math]x + \dfrac{1}{x} = 5[/math]
⇒ [math]\left(x + \dfrac{1}{x}\right)^2 = x^2 + \dfrac{1}{x^2} + 2 = 25[/math]
⇒ [math]x^2 + \dfrac{1}{x^2} = 23[/math]
এখন,[math]\left(x^2 - \dfrac{1}{x^2}\right)^2 = \left(x^2 + \dfrac{1}{x^2}\right)^2 - 4[/math]
⇒ [math]= 23^2 - 4 = 529 - 4 = \boxed{525}[/math]
 গ [math]x^4 + \dfrac{1}{x^4} = 2[/math]
আমরা জানি,[math]\left(x^2 + \dfrac{1}{x^2}\right)^2 = x^4 + \dfrac{1}{x^4} + 2[/math]
⇒ [math]\left(x^2 + \dfrac{1}{x^2}\right)^2 = 2 + 2 = 4[/math]
⇒ [math]x^2 + \dfrac{1}{x^2} = \sqrt{4} = 2[/math]
আবার,[math]\left(x + \dfrac{1}{x}\right)^2 = x^2 + \dfrac{1}{x^2} + 2[/math]
⇒ [math]= 2 + 2 = 4[/math]
⇒ [math]x + \dfrac{1}{x} = \sqrt{4} = \boxed{2}[/math]
∴ প্রমাণিত।

Explanation:
ব্যাখ্যা:

ক দুই পাশে [math]x[/math] দিয়ে গুণ করলে পাওয়া যায় [math]x^2+1=3x[/math] থেকে সরাসরি [math]x^2-3x+1=0[/math]।  

খ প্রথমে [math]x^2+\dfrac{1}{x^2}=(x+\dfrac{1}{x})^2-2=25-2=23[/math], এবং [math](x^2-\dfrac{1}{x^2})^2=(x^2+\dfrac{1}{x^2})^2-4=23^2-4=525[/math]।  

গ ধরা যাক [math]y=x^2+\dfrac{1}{x^2}[/math], তখন [math]y^2-2=x^4+\dfrac{1}{x^4}=2[/math] থেকে [math]y^2=4[/math] এবং বাস্তব [math]x[/math] হলে [math]y\ge2[/math] তাই [math]y=2[/math]. ফলে [math](x-\dfrac{1}{x})^2=y-2=0[/math], তাই [math]x=\dfrac{1}{x}[/math] এবং বাস্তবে (ধরা হলে [math]x>0[/math]) [math]x=1[/math], ফলে [math]x+\dfrac{1}{x}=2[/math] (নেগেটিভ ক্ষেত্রে [math]x=-1[/math] হলে মান [math]-2[/math] হয়)।

Question:
[math]a + \frac{1}{a} = 3[/math] হলে

Options:
ক. [math]\left(a - \frac{1}{a}\right)^2[/math] এর মান নির্ণয় কর।
খ. [math]\left(a^2 - \frac{1}{a^2}\right)^2[/math] এর মান নির্ণয় কর।
গ. প্রমাণ কর: [math]a^4 + \frac{1}{a^4} = 47[/math]


Correct Answer: 

Solution:
ক [math]a + \frac{1}{a} = 3[/math]
 [math]\Rightarrow \left(a + \frac{1}{a}\right)^2 = a^2 + 2 + \frac{1}{a^2} = 9[/math]
 [math]\Rightarrow a^2 + \frac{1}{a^2} = 7[/math]
এখন,[math]\left(a - \frac{1}{a}\right)^2 = a^2 - 2 + \frac{1}{a^2} = 7 - 2 = 5[/math]
 খক থেকে পেয়েছি: [math]a^2 + \frac{1}{a^2} = 7[/math]
এখন,[math]\left(a^2 - \frac{1}{a^2}\right)^2 = a^4 - 2 + \frac{1}{a^4}[/math]
আগে বর্গ করি:[math]a^2 - \frac{1}{a^2} = \sqrt{5}[/math]
তাহলে,[math]\left(a^2 - \frac{1}{a^2}\right)^2 = 5[/math]
 গ [math]a^2 + \frac{1}{a^2} = 7[/math]
তাহলে,[math]\left(a^2 + \frac{1}{a^2}\right)^2 = a^4 + 2 + \frac{1}{a^4} = 49[/math]
 [math]\Rightarrow a^4 + \frac{1}{a^4} = 49 - 2 = 47[/math]
 — প্রমাণিত

Explanation:
ব্যাখ্যা:

ক [math](a-\frac{1}{a})^2=(a+\frac{1}{a})^2-4=3^2-4=5[/math].

খ প্রথমে [math]a^2+\frac{1}{a^2}=(a+\frac{1}{a})^2-2=9-2=7[/math], তাই [math]\left(a^2-\frac{1}{a^2}\right)^2=(a^2+\frac{1}{a^2})^2-4=7^2-4=45[/math]।

গ [math]a^4+\frac{1}{a^4}=(a^2+\frac{1}{a^2})^2-2=7^2-2=47[/math]।

Question:
[math]p^2 - 8p - 1 = 0[/math] হলে

Options:
ক. [math]p - \frac{1}{p} = ?[/math]
খ. প্রমাণ কর যে, [math]p^4 - 66p^2 + 1 = 0[/math]
গ. [math]\left(p^2 - \frac{1}{p^2}\right)^2 = কত?[/math]


Correct Answer: 

Solution:
কমূল সমীকরণ: [math]p^2 - 8p - 1 = 0[/math]
উভয় পাশে [math]p[/math]
 দিয়ে ভাগ করলে:[math]\frac{p^2}{p} - \frac{8p}{p} - \frac{1}{p} = 0[/math]
 [math]\Rightarrow p - 8 - \frac{1}{p} = 0[/math]
 [math]\Rightarrow p - \frac{1}{p} = 8[/math]
 খআমরা জানি, [math]p - \frac{1}{p} = 8[/math]
উভয় পাশে বর্গ করি:[math]\left(p - \frac{1}{p}\right)^2 = p^2 - 2 + \frac{1}{p^2} = 64[/math]
 [math]\Rightarrow p^2 + \frac{1}{p^2} = 66[/math]
আবার বর্গ করি:[math]\left(p^2 + \frac{1}{p^2}\right)^2 = p^4 + 2 + \frac{1}{p^4} = 4356[/math]
 [math]\Rightarrow p^4 + \frac{1}{p^4} = 4354[/math]
তাহলে,[math]p^4 - 66p^2 + 1 = 0[/math]
 — প্রমাণিত ✅গআমরা জানি:[math]p - \frac{1}{p} = 8[/math]
 [math]\Rightarrow \left(p - \frac{1}{p}\right)^2 = p^2 - 2 + \frac{1}{p^2} = 64[/math]
 [math]\Rightarrow p^2 + \frac{1}{p^2} = 66[/math]
এখন,[math]\left(p^2 - \frac{1}{p^2}\right)^2 = p^4 - 2 + \frac{1}{p^4}[/math]
পূর্বে প্রাপ্ত: [math]p^4 + \frac{1}{p^4} = 4354[/math]
তাই, [math]\left(p^2 - \frac{1}{p^2}\right)^2 = 4354 - 2 = 4352[/math]


Explanation:
ব্যাখ্যা: নিচের ধাপে সহজভাবে দেখানো হলো।

ক [math]p^2-8p-1=0[/math]-কে [math]p[/math] দিয়ে ভাগ করলে পাওয়া যায় [math]p-8-\frac{1}{p}=0[/math], তাই [math]p-\frac{1}{p}=8[/math]।  
খ [math]p-\frac{1}{p}=8[/math] বর্গ করলে [math]p^2+\frac{1}{p^2}-2=64[/math], তাই [math]p^2+\frac{1}{p^2}=66[/math]; উভয় দিকে [math]p^2[/math] গুণ করলে [math]p^4-66p^2+1=0[/math] হয়।  
গ [math]\left(p^2-\frac{1}{p^2}\right)^2=(p^2+\frac{1}{p^2})^2-4=66^2-4=4352[/math]।
