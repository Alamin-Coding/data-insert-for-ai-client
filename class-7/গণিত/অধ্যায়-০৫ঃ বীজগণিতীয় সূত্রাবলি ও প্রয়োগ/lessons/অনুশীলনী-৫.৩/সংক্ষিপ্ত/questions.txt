Question:
[math]x + \frac{1}{x} = 5[/math] হলে, [math]x^2 + \frac{1}{x^2}[/math] এর মান নির্ণয় কর ।

Options:


Correct Answer: 

Solution:
[math]x + \frac{1}{x} = 5[/math]
 [math]\left(x + \frac{1}{x}\right)^2 = 5^2 = 25[/math]
 [math]x^2 + 2 + \frac{1}{x^2} = 25[/math]
 [math]x^2 + \frac{1}{x^2} = 25 - 2 = 23[/math]


Explanation:
সমাধান: উভয় পাশে বর্গ করলে সূত্র [math](a+b)^2=a^2+2ab+b^2[/math] প্রযোজ্য হয়, তাই [math](x+\frac{1}{x})^2=x^2+2+\frac{1}{x^2}=25[/math] (কারণ [math]2x\cdot\frac{1}{x}=2[/math]), ফলে [math]x^2+\frac{1}{x^2}=25-2=23[/math].

[math]\frac{a}{b} + \frac{b}{a} = 2[/math]

 [math]\Rightarrow a^2 + b^2 = 2ab[/math]
 [math]\Rightarrow a^2 - 2ab + b^2 = 0[/math]

 [math]\therefore a = b[/math]
 (প্রমাণিত)

Explanation:
সমাধান: দেওয়া আছে [math]\frac{a}{b}+\frac{b}{a}=2[/math]. উভয় পাশে [math]ab[/math] দিয়ে গুণ করলে পাই [math]a^2+b^2=2ab[/math], যা থেকে [math]a^2-2ab+b^2=(a-b)^2=0[/math] হয়, তাই [math]a=b[/math]।

[math]x + \frac{9}{x} = 6[/math] হলে, [math]\frac{1}{x}[/math] এর মান নির্ণয় কর ।

Options:


Correct Answer: 

Solution:
[math]x + \frac{9}{x} = 6[/math]
 [math]x^2 + 9 = 6x[/math]
 [math]x^2 - 6x + 9 = 0[/math]
 [math](x - 3)^2 = 0[/math]
 [math]x = 3[/math]
 [math]\frac{1}{x} = \frac{1}{3}[/math]


Explanation:
ব্যাখ্যা: [math]t=1/x[/math] ধরে লিখলে সমীকরণটি হয় [math]\frac{1}{t}+9t=6[/math], যা থেকে গুণ করলে পাওয়া যায় [math]9t^2-6t+1=0[/math]. এই কোয়াড্রাটিকের ডিসক্রিমিন্যান্ট শূন্য হওয়ায় মূল দুটির মান একই এবং [math]t=1/3[/math], ফলে [math]1/x=1/3[/math].

[math]m^2(m + 1) = 15[/math], [math]m^2(m - 1) = 9[/math] হলে, [math]m^5[/math] এর মান নির্ণয় কর ।

Options:


Correct Answer: 

Solution:
[math]m^2(m + 1) = 15[/math]
 [math]m^2(m - 1) = 9[/math]
 [math](m^2(m + 1))^2 - (m^2(m - 1))^2 = 15^2 - 9^2[/math]
 [math](m^3 + m^2)^2 - (m^3 - m^2)^2 = 225 - 81 = 144[/math]
 [math]4m^3m^2 = 144[/math]
 [math]4m^5 = 144[/math]
 [math]m^5 = \frac{144}{4} = 36[/math]


Explanation:
সমাধান: প্রথম সমীকরণগুলোকে লিখলে পাওয়া যায় [math]m^3+m^2=15[/math] এবং [math]m^3-m^2=9[/math]। যোগ করলে [math]2m^3=24\Rightarrow m^3=12[/math], বিয়োগ করলে [math]2m^2=6\Rightarrow m^2=3[/math]. তাই [math]m^5=m^3\cdot m^2=12\cdot3=36[/math].
