Question:
[math]x + \frac{1}{x} = 2[/math] হলে, [math]x^5 + \frac{1}{x^{10}}[/math] এর মান নির্ণয় কর

Options:


Correct Answer: 

Solution:
[math]x + \frac{1}{x} = 2 \Rightarrow x = 1[/math]
অতএব, [math]x^5 + \frac{1}{x^{10}} = 1^5 + \frac{1}{1^{10}} = 1 + 1 = 2[/math]
সঠিক উত্তর: [math]x^5 + \frac{1}{x^{10}} = 2[/math]


Explanation:
[math]x+\frac{1}{x}=2\Rightarrow x^2-2x+1=0\Rightarrow (x-1)^2=0[/math] তাই [math]x=1[/math].  
অতএব [math]x^5+\frac{1}{x^{10}}=1^5+\frac{1}{1^{10}}=1+1=2[/math].

Correct Answer: 

Solution:
[math]\frac{a}{b} + \frac{b}{a} = 2[/math]
 [math]\Rightarrow \frac{a^2 + b^2}{ab} = 2[/math]
 [math]\Rightarrow a^2 + b^2 = 2ab[/math]
 [math]\Rightarrow a^2 - 2ab + b^2 = 0[/math]
 [math]\Rightarrow (a - b)^2 = 0[/math]
 [math]\Rightarrow a - b = 0[/math]
 [math]\therefore a = b[/math]
 (প্রমাণিত)

Explanation:
যদি [math]a[/math] ও [math]b[/math] শূন্য না হন, তাহলে [math]\frac{a}{b}+\frac{b}{a}=\frac{a^2+b^2}{ab}=2[/math}, ফলে [math]a^2+b^2=2ab[/math], অর্থাৎ [math](a-b)^2=0[/math]. তাই [math]a=b[/math].
